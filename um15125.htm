15.1.2.5 Loop Searching                                                        Previous Next TOC

 

AI can perform a loop similar to that involved in creating a plot, described in the above sections, but where the object is to automatically select one of the points rather than to create a plot of the entire set.

 

One form takes the format

 

MINIMIZE (or MAXIMIZE) __ FOR __ EQUALS __ TO __

 

For example:

 

MINIMIZE SA3 FOR 2 TH = 1 TO 2

MAXIMIZE YA ON 11 AT HBAR = 1 AS 5 TH VARIES FROM 0 TO 20.

 

There is no optimization or searching involved with this feature; it simply selects the data point that best satisfies the condition.  Keep in mind that the MINIMIZE verb does not find the smallest absolute value, as does the optimization program: -10,000 is less than -1, and would be selected if those were the choices.

 

Another form can find the data point where a given noun comes closest to a desired value, as another noun is varied:

 

SEARCH FOR __ of VALUE for __ equals __ to __

 

Examples:

 

SEARCH FOR SA3 of 0 for bending on surface 1 = -2 to 2.

SEARCH FOR BACK OF 5.0 FOR 6 INDEX EQUALS 1.4 TO 1.8

 

The second example would select that index value for which the back focus distance comes closest to the value of 5.0.
