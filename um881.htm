8.8.1 Examples of MAP input                                Previous TOC

 

1. We want to examine the vignetted aperture shape ("cat's eye diagram") for a triplet across the field.  This lens is symmetrical about the Y-Z plane, so it is really not necessary to plot skew field points – but for this example we will anyway, to show what would be done for a lens without bilateral symmetry.  The input for this example is:

 

MAP PUPIL OVER FOV

FGRID RECT 5 5

RGRID CIRCLE 30

PLOT.

 

The results are shown below.

 

 

Note that we entered no scale factor; for PUPIL maps, the default scale gives a circle of one inch diameter for the unvignetted pupil.  The position of each diagram on the map (as given by the crosshairs within each set of dots) is proportional to the fractional field that it represents, scaled to a 5 by 5 inch square array in accordance with the (default) EXPLODED option.  (This is constant, unrelated to the “5 5" in the input, which specify the number of rows and columns in the grid of object points.)  If we had specified ACTUAL, the location of each plot would be proportional to the (X,Y) coordinates of that field point at the image, rather than at the object.  The default is EXPLODED, as in this case, because there is no ON SURFACE ... or ACTUAL included.

 

2. Here we wish to map the change of angle of incidence of the chief ray on a steeply curved surface as a function of field point. The input will be:

 

MAP UNI OVER FOV ON SURFACE 7

FGRID CREC 9 9

ANALOG

SCALE 4

PLOT.

 

In this case the default is ACTUAL coordinates, and we elect to see the data in analog fashion (as shown below) rather than digitally, which is the default.  The entered scale factor, 4 replaces the default of 1.  The analog scale factor usually equals the SCALE input, unless a value of ASCF is explicitly entered, but for these angle plots it is unused and the angles are shown actual size.

 

 

3. This example shows digital output.  The input is:

 

MAP HFREQ OVER PUPIL ON SURFACE 2

FGRID POINT 0 0

RGRID CREC 9 9

SCALE 600

PLOT.

 

The number at each location on the map is the hologram spatial frequency at that point, and the position of the point on the map gives the actual coordinates of the point on the surface, at 600 times scale.  This example is shown below.

 

 

4.  This example shows the spot diagram mapped over the field.  The input is:

 

ON 85

CHG

CFOV

END

MAP SPOT OVER FOV                                                              

FGRID CREC 7 7                                                                

RGRID CREC 15 15                                                              

COLOR M                                                                        

SCALE 1                                                                        

EXPLODED                                                                      

ANALOG 500                                                                    

PLOT

 

Before we can map over the field, this lens, which had no skew field, had to be assigned a circular field, with the CFOV directive.  Switch 85 is turned on here, so the spots show up in colors approximately the way they would appear to the eye.  Since a well-corrected image will usually show up as a white spot, which would be invisible on white paper, the program automatically adds the black area before it draws the spots, in this mode.
