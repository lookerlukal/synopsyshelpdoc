5.0 UTILITY FEATURES        

 

Many of the features described in the following sections can be accessed interactively in the dialog MUT.

 

 

5.1 Surface Shape Routines (SAG)

 

The entry

 

SAG SN [ HAAP ] NSTEPS

 

will print a sag table of surface SN for Y axis coordinates from 0 to HAAP in NSTEPS steps. Tilts and decenters are ignored.  An example of SAG output is shown below.  This feature may also be run interactively with the MUT dialog.

 

The special form

 

SAG SN NURBS [ FULL ]

 

is valid only if surface SN has been declared a NURBS surface.  In this case the program will list the Z-components of the control points, arranged in a matrix.  If the optional FULL is entered, the program instead lists (X,Y,Z) points on the surface, where the X and Y are taken from the control points and the Z give the sag at the surface itself (not the Z-coordinate of the control points).

 

Hyperspherical surfaces are treated somewhat differently; the analysis is done in spherical coordinates, and the program provides a list of angles and the (Y,Z) coordinates at each.
