MWL – Select From a List of Saved Lens Files

 

The MWL dialog shows a list of lens files that have been saved with the SAVE command.  This is more informative than given in the Windows File|Open dialog, because it also shows the ID of the lens along with a drawing.  Click on a lens file and then click OK, or just double-click the name.  That file will then be opened.

 

If you select one of the preview options (YZ profile or perspective) each lens is shown in a small box as you click on it.  The Auto option will select either the profile or the perspective drawing, depending on whether or not the lens has bilateral symmetry.  The SketchPAD option shows the lens in the PAD display in the currently selected format \-- which should be selected first since you cannot interact with PAD when the MWL dialog is open.

 

If you type into the Search window and then click the Search button, the program will only list those files that contain the entered string somewhere.  Thus you can select lens files that use the Gaussian object, for example, by selecting the word OBG.

 

Warning messages that would be listed when a saved lens files is opened are ordinarily not displayed at the time that the lens is shown in the preview pane.  If you want to monitor these warnings, turn on switch 54.

 

You can also use the command FETCH filename.
