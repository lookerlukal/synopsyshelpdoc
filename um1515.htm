15.1.5.1 Matching Vendor-Supplied Elements

 

SYNOPSYS contains a database with stock lenses from several vendors.  The current vendors supported by SYNOPSYS are: Melles Griot, Inc. (MELL); Spindler & Hoyer (SPIN); Edmund Scientific Company (EDMU); Newport Corporation (NEWP); JML Direct Optics (JML); CVI Laser Corporation (CVI); and Optics for Research, Inc (OFR).

 

DISCLAIMER:

OSD, Inc. makes no warranty regarding the completeness or accuracy of the data in the online vendor catalogs.  While every effort has been made to ensure that these data are accurate, we have no control over the listings provided by third-party vendors.  You are advised to contact the vendor to verify any and all data related to these stock lenses, including dimensions, glass types, and product availability.

 

The AI verb MATCH can find the stock lens element that most closely matches a given lens.  The format is

 

MATCH ELEMENT NB TO catalog

 

This match is based on a combination of the element EFL, bending parameter, clear aperture, and design wavelength.  The program will by default select the 10 best matches from the stated catalog, which are listed, but not automatically inserted into the lens (see INSERT CHOICE and REPLACE ELEMENT, below).  The number of choices listed may be controlled with the AI noun NMATCH, with a maximum allowed of 50.

 

NMATCH = NB

 

The program can also search the vendor catalogs for an element of stated properties:

 

MATCH { SINGLET / DOUBLET / TRIPLET / TORIC } WITH conditions [AND conditions] ...

 

where conditions may include

 

FOCL = NB

BENDING = NB

CAO = NB

WAVL = NB

 

For example "Match singlet with FOCL 100 AND BENDING =.5 TO MELLES'.  Once the set of matches is found, it may be redisplayed if necessary by the command

 

SHOW CHOICES

 

An entire vendor catalog may be displayed:

 

SHOW SPINDLER

 

A default thickness and clear aperture to be required of a match may be entered with

 

DEFAULT THICKNESS = NB

DEFAULT CAO = NB

 

or displayed with SHOW DEFAULTS.

 

The WorkSheet toolbar contains a button that will activate a Vendor Stock Lens dialog, where you can perform most of the above operations with mouse clicks.
