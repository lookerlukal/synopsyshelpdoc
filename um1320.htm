13.20 The TrayPrompt                                                        Previous Next TOC

 

To help you remember the formats of SYNOPSYS commands, the program monitors what you type into the Command Window and into the MACro editors, and whenever it detects that you have typed a valid single-line command, it displays the format of that command in the system tray, which is that strip at the bottom of your screen where Windows displays various messages.  Thus, when you type the command

 

PSPRD

 

The tray prompt shows this format:

 

 

and you can easily see what are the required arguments.

 

The TrayPrompt will also be displayed after you select some text in the MACro editor, provided that text has the format of a valid command.  This should help you verify that the arguments you may have typed are correct for that command.

 

The TrayPrompt display differs somewhat from the other prompt strings used in SYNOPSYS, because the tray cannot show underlined characters.  Instead of showing user-entered numeric words underlined (as in the Help files) we show them in lowercase letters in the tray.

 

Some character strings, such as “PLOT” are used in multiple ways in SYNOPSYS; in most cases, those strings are not prompted.

 

A few commands have a variety of forms, and in order to fit in the tray, we show only the most often used of those.

 

If the information displayed in the tray is not sufficient for your needs, press the F2 key.  This key immediately jumps to the Help index for the topic given by the first four characters in the tray.  There you will find a complete description of the feature you are after.
