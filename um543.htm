5.43 Duplicate and Mirror (DMIRROR, CMIRROR)        Previous Next TOC

 

The program can easily perform the common task where you want an exact copy of your lens, flipped so as to make a mirror image of the lens and put at the end.  There are two forms: DMIRROR makes a flipped copy, and CMIRROR creates a duplicate where the light returns through a copy that is not flipped after reflecting at a mirror;

 

Input for the first form is

 

 DMIRROR ISN

 

ISN is the surface number about which the system will be mirrored.

 

This feature may be run from the dialog MUT.

 

Here is an example.  This is a three-element lens that is approximately telecentric at the image.  We want to duplicate and mirror the system about the image plane, which is surface number 8.

 

 

After the command

 

DMIRROR 8

 

the system is duplicated as requested.

 

 

There are restrictions.  No HOEs, DOEs, or GRATINGs can appear in the affected lenses, and no tilts, decenters, or GLOBAL or LOCAL surfaces can be involved.  In other words, this is for very simple cases only.  The mirrored surfaces will all be assigned pickups of the original surfaces so mirror symmetry will be maintained if you change the first section.  If the object is type OBB (infinite object usually), the output from DMIRROR will be AFOCAL.  Otherwise the last thickness is taken from the value of TH0.  This thickness is not assigned a pickup, however, so if you want to maintain the paraxial focus you must add the YMT solve yourself.

 

In the more general case, where the above restrictions are not observed, you can still make a mirrored copy of the lens – but you have to use different features.  Here is one procedure:

 

1.  Load the starting lens into two ACONs, say 1 and 2.

2.  Switch to ACON 2 and execute the REVERSE command.  This is more general and allows the forbidden items.

3.  Use the LE command to put a copy of this reversed system in the editor.

4.  In the editor, drag-select and copy (Ctrl + C) all of the lines starting with surface 1, but not the object and system data.

5.  Switch to ACON 1, make a checkpoint, and open the WorkSheet.

6.  Note the surface number of the last surface, in the above case this would be number eight.

7.  In the Worksheet edit pane, below the text for surface 8, enter

 

ADD 8

 

and then paste the copied lines.  Click Update, and the reversed lenses are added to the current one.

 

8.  You are almost done.  This procedure has copied the data but has not added any pickups.  The only parameter that is not correct is thickness 8, which we did not specify yet.  So add

 

8 PTH 7,

 

click Update, and you are done.

 

Be sure to check your lens after these steps, or after the DMIRROR command, to be sure that the object and image are where you want them, and to check whether any GLOBAL surfaces were put where they should not be.  (They are located relative to surface 1, not the mirrored surface.)

 

 

Input for the second form is much the same:

 

 CMIRROR ISN

Some restrictions apply here as well: no HOEs, DOEs or GRATINGs.  The duplicated surfaces are all assigned pickups and are coincident with the originals, so one can tolerance the first half and expect the duplicate part will remain a duplicate.
