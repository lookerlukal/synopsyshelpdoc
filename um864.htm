8.6.4 Diffraction Image Model (DMODEL, WMODEL)                 Next Previous TOC

 

A mathematical description of the diffraction point-spread function may be calculated and stored for later evaluation by the general image dissection program FOR or the program DET (see sections 9.1 to 9.3).  The easiest way to create the model is with the dialog MMO; the command-mode input is as follows :

 

{D/W}MODEL { ICOL / M / P } HBAR NRYS WT GBAR

 

ICOL

is the color number, “M” requests a multicolor analysis, and “P” just the primary color.

 

HBAR

is the fractional Y-object height

 

NRYS

is the desired number of rays in the exit pupil, up to 20,000.

 

WT

is the relative weight of this image.  The weighting of multicolor (M) images is part of the lens data, and a WT of 1.0 should be entered for this case. See 8.2.)

 

GBAR

is the fractional X-object height

 

 

The image is sampled at 1/4 the cutoff spatial wavelength.  DMODEL samples the image to a distance of about 5 Airy disk radii, while the wide-angle version WMODEL samples to twice that area.  If you request that version, it is important that you specify a large enough number of rays to avoid aliasing in the image.  A good test is first to run the WPSPRD command, which will show the diffraction pattern and make aliasing perfectly obvious, if present.

 

In the event your image extends over an area larger than that sampled by WMODEL, which can happen, for example, if the image of a point is a long narrow slit that is more than 10 times the size of the Airy disk, then the diffraction image model programs described above are not suitable.  This should be obvious when you look at the WPSPRD plot and see that the image spills out of the picture.  In that case, you can use the Image Tools (MIT) instead, and request a coherent analysis.  (A point object is always coherent with itself, so this is an appropriate tool for that case.)  That program is not limited in sample area, and offers many of the same kind of image analysis that can be performed on the diffraction model.  The MIT coherent analysis feature is also slightly more precise for examining a poorly corrected image, since the image is more finely sampled.  It is also faster, since it uses a 2D FFT approach instead of the diffraction integral of the models.

 

See section 9.1 for a description of the use of the image model features.
