7.12 Grid distortion (GDIS)                        Next Previous TOC

 

There are several features in SYNOPSYS that will analyze and show the distortion properties of your lens.  The command DIS will plot the distortion in the Y direction, and MAP can show the distortion across the field in several ways.  Also, the field blur menu (MFB) can show what a selected target would look like when imaged by the lens.

 

This option can also be found in the menu MRR.

 

For the common case where you want to see how a grid of lines would be distorted, a simpler feature is available.  The command is

 

 GDIST [ NGRID [ G/0 SCF ] ].

 

The program traces the chief ray in the major color over a grid of field points, constrained within a circle if the lens has CFOV declared, and rectangular if the lens is declared RFOV, and plots the positions of each point.  If you include the optional [G], the program will superimpose a grid showing the paraxial locations of the same rays, allowing you to easily see the difference.  The grid number is entered in NGRID, and must be between 1 and 99 if entered; the default is 31.

 

If the lens does not have a skew field defined (which is a requirement for this feature), the program will temporarily change the field to circular.  See CFOV.  If this is not appropriate, the Pupil Wizard (MPW) is a good place to define the data as you wish.

 

If the lens is FOCAL, the plot shows the (X,Y) locations of each ray intercept at the image.  If it is AFOCAL, the plot shows the reduced angles Φ and Ψ (see Section 3.2).

 

the plot is scaled so that the real-ray grid fits on the page.  In the event that distortion is so large that the paraxial grid extends beyond the page, you can reduce the overall scale with the optional SCF in word 4, where 0.5 means half the usual size.
