LE – Lens Data Editor

 

The LE lens editor is a powerful tool for editing your lens, provided that you know the basic formats of an RLE file.  The entire lens file is formatted as it would be for a new lens, and you may change any data you want.  Then click the button and the changes are in effect.  If you started LE from a SketchPAD session with the button the display is automatically updated with the new lens.  If you started it with the command LE, just click in the SketchPAD window to update that display.
