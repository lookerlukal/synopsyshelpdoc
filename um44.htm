4.4 Print Options (POP)                                Previous Next TOC

 

The command POP (Print OPtions) gives a short table of all pickups and solves in effect, and also a table indicating the various flags that have been set for curvatures and indices, which is occasionally useful.  An example POP listing is shown below.  This feature is also found in the MLL dialog.

 

SYNOPSYS AI>POP
