File menu commands

The File menu offers the following commands:

New Lens

Deletes the current lens and starts the SYS editor.

Open a Lens File

Opens an existing RLE (lens) file.

Save Lens

Saves the current lens using the same file name.

Save As

Saves the current lens to a specified file.

Close the Current Window

Closes the current Command Window.

Open a New Command Window

Opens a new Command Window where you can type SYNOPSYS commands and where your output is listed.

Select Directory

Displays a dialog box where you can type the name of a new directory. This becomes your default directory

Print

Prints the contents of the active window..

Print Preview

Displays the window contents on the screen as it would appear printed.

Print Setup

Selects a printer and printer connection.

Exit        

 

Exits SYNOPSYS.
