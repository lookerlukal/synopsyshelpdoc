8.6.12 Variance-MTF relationship (VMR)                        Next Previous TOC

 

One sometimes would like to know the relationship between the wavefront variance and the diffraction MTF, especially when planning a BTOL analysis.  That program can accept a quality descriptor of either type, but a variance analysis executes more quickly.

 

The sequence

 

VMR SFREQ

HBAR HBAR HBAR HBAR

[ GBAR GBAR ]

ICOL { P / M / ICOL }

PLOT

 

Will alter the focus in steps over a reasonable range and plot the two analyses.  Here, SFREQ is the spatial frequency at which the analysis is performed.  The following picture shows an example of the output.

 

 

The value of variance at each field that corresponds to the desired MTF value can be read from the graph.  This is not often a simple single-valued function, however, so exercise discretion in selecting your targets.
