Select PAD display button

 

Clicking on one of these buttons restores the PAD display to the configuration that was assigned the last time the button was pressed.  You can define up to five different display configurations in this way, and switch from one to another with a single click.

 

Click one, and then set up the display the way you want, with the buttons .  Click another, and set up a different display, and so on.  Now you can restore any of the configurations you have selected just by clicking the same number button again.

 

If you are working on a periscope, for example, you might want the objective section in display 1, portions of the relay in displays 2 through 4, and the eyepiece in display 5 – along with image analysis displays or perspective views on the bottom of the screen.  The selections you have made are saved along with the other PAD parameters with the SSU (Save SetUp) command or the SPL (Save PAD Layout) command.  The former saves the layout plus all the lens files in every ACON, while the latter just saves the layout.

 

The layout can be restored at a later time with the RPL (Recall PAD Layout) command, and the whole setup with the RSU (Recall SetUp) command.
