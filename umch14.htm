14.0 ENVIRONMENTAL ANALYSIS PACKAGE

 

Several programs are available for evaluating the effects of changes in the thermal and mechanical environment on a lens system.  The system temperature and air pressure may be defined in the lens file itself, with the TEMPERATURE and PRESSURE parameters.  Those options only affect the indices of refraction of the lens elements, however, and the input values of other parameters still apply in that case.  If the system undergoes an environmental change that affects mechanical dimensions as well, then the programs described below are more appropriate.

 

Thermal soaks may be analyzed with the program THERM, described in section 5.9. That program will alter the radii and thicknesses of elements according to the change in temperature and the thermal coefficients appropriate to the element.

 

If your system undergoes an environmental change that results in a more complicated change of surface shape, you can calculate the new shape with a third-party structural or thermal analysis program, and then obtain the coefficients of the new shape from that program.  Those surface deformations may then be analyzed using the DC 1 ... input in the RLE file if the shape is described by that power series, and with the Zernike input if that polynomial is more appropriate.

 

If you have a set of position changes over the surface, the programs DEF and ZERNIKE can be used to derive the deformation coefficients.  These can be determined by digitizing an interferogram, thereby generating a set of Z-deformations at each of a number of (X,Y) positions. Section 5.7 describes this feature.

 

Another feature, TOSP, is more general but operates only on a perturbational basis; it is described below.
