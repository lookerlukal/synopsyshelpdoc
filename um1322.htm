13.22 Dual Monitor Support                                                Previous Next TOC

 

If your system has more than a single monitor, you may wonder why you can’t drag any of the windows in SYNOPSYS onto a different screen.  Actually, you can.

 

First, use the Restore Down feature after you start the program (on the extreme upper right of the outside frame).  The frame gets smaller.  Then drag the corners of the outer frame so that it expands and covers both of your monitors.  Now you can position any window opened by the program anywhere on either monitor.

 

If you spread your windows out just the way you want, but then drag the frame to make it smaller, it can happen that some of the windows can no longer be seen inside that smaller frame.  Should this happen, the PAD display can be retrieved with the commands

 

PAD

RESTORE.

 

You can also use the command DWL (default window location), or the menu item Options | Use Default PAD placement.  Those will close PAD, and the next time you open it, it will appear onscreen.  They will also restore the default location of the Display Scale popup.

 

The KAG command will close all open graphics windows, even if you can’t see them.
