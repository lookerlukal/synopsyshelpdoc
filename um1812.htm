18.1.2 CAPTURE surfaces                                                        Previous Next TOC

 

If a surface is declared CAPTURE, it can intercept any ray that has been set LOOSE, as well as the rays from either adjacent surface, which need not have a nonsequential flag set.  But the ray must fall within the active area of the surface (see section 18.2) or it will propagate elsewhere.  If nonsequential paths to the focal plane are of interest, for example, that surface should be declared CAPTURE (and a suitable CAO entered).

The CAPTURE flag may be removed with NOCAPTURE.

 

Note that in order for the program to determine whether a ray is captured by a given surface, it must have some kind of aperture data: a ray outside this aperture is not captured.  If there is no hard aperture assigned, the program will utilize the soft, or default, CAO instead.  This is the only case where the path of a ray can be affected by default CAOs.
