8.5.1.1 Spot Diagram (SPT)                                 Next Previous TOC

 

The command

 

SPT { ICOL / M / P } HBAR NRYS [ SCF GBAR [ YSCF ] ]

 

will generate a single spot diagram describing the image on the final surface.  The data are as follows:

 

ICOL

color number; the optional “M” calls for a multicolor analysis, and the “P” for just the primary color.

HBAR

fractional Y-object coordinate

NRYS        

number of rays in spot, up to 20,000

SCF

that image dimension which will be scaled to one inch on the plot.  If omitted or zero, the program will use a reasonable default.

GBAR

fractional X-object coordinate

YSCF

optional scaling for Y-direction

 

The origin of the plot is the chief ray position in the primary color (even if that color is not requested).  Spots may be drawn either as a single dot or with a symbol indicating the color, as determined by switch 27 (see the dialog MSS or App. B).  The default is ON 27, which gives just a dot.  The size of the symbols so requested may be controlled with the SSSIZE command. SPT may also be run interactively from the dialog MGI.  The geometry of the ray grid is set by switch 90: if on, the grid is hexapolar, while if off it is a square grid.

 

Multicolor SPT traces the same number of rays in each color, unlike TFS and MAP, which vary the ray number according to the spectral weights.  (The RMS and centroid calculations are more accurate if the ray counts are the same.)

 

SPT plots will be shown in color if switch 35 is ON.  The default displayed colors come from a built-in color table, and will not necessarily be the same color as the wavelengths analyzed.  To display the spots in approximately the visual colors of each wavelength, turn on switch 85 as well as 35 before requesting the plot.   In this mode, the program draws a black area before drawing the spots, since a good image could appear white, and would be invisible otherwise.  Wavelengths in the UV are shown in violet, and in the IR are drawn in red – even though in fact those wavelengths are invisible to the eye.

The program draws, for reference, the size of the Airy diffraction disk, calculated for the major color and the actual F/number in X and Y at the requested image point.  This may be defeated by turning on switch 80.

 

If switch 50 is ON, a listing will also be produced giving the coordinates of every ray in the spot diagram.
