3.1.5 Waveguide Object                                        Previous Next TOC

 

One occasionally needs to model a system with a wavelength much longer than is usually the case in optics; a mm-wave waveguide is a good example.  Although all of the features of SYNOPSYS work in such cases, one must be aware that diffraction effects are much more pronounced, since the wavelength is much larger than usual relative to the physical aperture of the system.  For this reason, the diffraction-propagation feature DPROP is perhaps the only analysis function that makes sense to use with this kind of system.

 

This kind of object is modeled with the input

 

 OBW SEMIAP MULTIPLIER

 

This object models the input wavefront as a Bessel function, where the amplitude is given by

 

 A = J0( MULTIPLIER*RHO/SEMIAP )

 

and RHO = SQRT( X**2 + Y**2).

 

The intensity of the beam is then the square of this quantity, and the phase is taken to be zero wherever the amplitude A is positive, and 180 degrees otherwise.

In this system, the beam has a radius of 65 mm, and the wavelength is 2 mm.  Let us first look at the beam profile on surface 1:

 

 DPROP P 0 0 1 SURF

 

 

This profile is truncated at the first zero of the Bessel function.  Now we get the profile on surface 2, which is 2.0e4 mm distant:

 

 

This closely resembles the beam at surface 1, except for the larger scale.  Here is the FRINGE analysis at this surface:

 

 

Where did the fringes come from?  This is supposed to be a plane wave, after all.  Well, it’s not plane anymore.  Like a Gaussian beam, this one expands due to diffraction, and at this distant plane it looks like a spherical wavefront centered near surface 1.  This analysis shows the fringes relative to the flat surface 2.  Make that radius –2.0e4, and the fringes mostly go away.

 

Like the Gaussian object, the OBW cannot take a field point other than zero.
