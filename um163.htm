16.3 Design Mode                                                        Next Previous TOC

SYNOPSYS NPASSES

 

 

The first four of these entries will create a set of targets automatically.  Thus, to minimize reflection over the range 0.4 to 0.8, one could enter GRW 0 0 50 .4 .8.  The total number of targets to be controlled must not exceed 50.

 

The method used to build the stack is from Dobrowolski (Appl. Opt., Aug. 1965, P. 937).  The program adds layers to the substrate or current stack two at a time, selecting the best pair from a net of four thicknesses and indices, giving 16 possibilities, and adding these to either the air-stack or stack-substrate interface.  After the best choice is added, the stack is optimized for a number of iterations, and then the whole process is repeated until the desired stack size is reached.  The program will delete layers that become too thin and consolidate adjacent layers whose indices are close.  When the design is finished, the program can optionally insert the closest matching actual material index at each layer, chosen from a stored table, and reoptimize the stack.  Input data are described below.

 

WAVL, ANGLE

are the same as in the RFILM file, for when there is no starting stack (BUILD).

 

DI

defines the position of the phase-angle discontinuity, for use when controlling phase changes.  Phase angles are defined between zero and THETA - 360), and the default THETA is 180 degrees.

 

KSD

can take values of 1 or 2.  1 adds layers only next to the substrate, while 2 adds layers next to the incident medium.  The default can add at either location, selecting the better place each time layers are added.  No KSD should be entered for this case.

 

TNET

gives the four values to the thickness net.  The default net is .1, .2, .3, .4 waves.  A value of zero may be entered for the first number; if entered elsewhere the net is terminated at that point.

 

NNET

gives four index values for the net.  The default values are 1.38, 1.62, 1.86, and 2.1.

 

TLIM

defines the thickness limits during optimization.  They must be outside of the thickness net.  Default limits are 0 to 1 wave.

 

NLIM

gives the limits to the index.  Defaults are 1.38 to 2.10.

 

PANT

is the file header command for the variable definition file.  The format is different from that in SYNOPSYS lens files: for every surface to be varied the surface number is entered along with two numbers.  The first applies to the thickness of that layer, which will be varied if the number is 1 and held fixed if it is 0.  The second number applies to the index.  Note that the PANT file is needed only if a starting stack is entered.  If the BUILD option is selected, no PANT file is needed since all parameters are assumed to be variables as they are added.  Layer number 1 is the incident medium, and variables begin with layer 2.

 

BUILD

causes the program to construct the entire stack given only the incident and substrate indices, and the number of layers.  If SN is negative, only the thicknesses will be varied as they are added. No is the refractive index of the incident medium; Nf and K are the real and (optional) imaginary parts of the substrate index.

 

If the word ALUMINUM is entered in word 4 of this command, the program will automatically update the index and K values at each wavelength analyzed.

 

The BUILD process will attempt to add layers up to the specified number – but only as long as doing so improves the performance.  If adding additional layers and optimizing results in no improvement, the process stops at that point.  (However, if the stack then has less than ½ of the requested number of layers, the process continues, but may fail to improve the stack – and you may have to abort the process with the stopsign button .)  Since layers are added two at a time and sometimes removed, the final number of layers will not necessarily equal the requested number.

 

ID ...

is an optional stack identification.

 

ADD

causes the requested number of layers to be added to a starting stack.  Negative SN varies only thicknesses.

 

FIX

causes the program to substitute the closest real material for each of the layers after optimization and reoptimize varying only the thicknesses.  The materials available are:

AANT

is the file header command defining the quantities to be controlled by the design program.

 

RAVE

is the average reflectance at the entered wavelength and angle. TAR is the desired value and WT the relative weight.

 

RSP

specifies the desired reflectance for the S- and P-polarized components, individually.

 

TAVE

controls the average transmittance.

 

TSP

controls the reflectance in S- and P-polarization.

 

RP

controls the phase change for S and P.

 

PDR

is the phase difference on reflection.

 

TP

controls the phase change on transmission.

 

PDT

is the phase difference on transmission.

 

END

ends the aberration definition file.

 

Up to 50 targets may be specified in the AANT file. SYNOPSYS initiates the optimization program.  When finished, the program automatically goes to common mode. Optimization may be terminated prior to completion of the requested number of iterations or accumulation of the requested number of layers, by clicking on after optimization has begun.
