13.12 Output Toggle (QUIET, LOUD)                                 Next Previous TOC

 

The command

 

 QUIET

 

Will suppress all output to the current Command Window.  This is intended primarily for long DO MACRO loops where you do not need to see all of the cases as they scroll by.  The loop will also execute more quickly in QUIET mode.

 

After the loop has finished, be sure to enter the command

 

 LOUD,

 

and then type AGAIN to see the summary of the loop results.  You may also toggle quiet mode by turning switch 72 ON and OFF.

 

A convenient way to manage this feature is to define two symbols:

 

DD: QUIET / DML / LOUD / AGAIN

DML: DO MACRO FOR …                (define your MACro loop variable here)

 

Then you simply type “DD” to run the loop in QUIET mode and display only the final table on the screen.  Of course, as with all DO MACRO loops, you must run the MACro once yourself (or click ) before starting the loop.  See section 15.1.2.1.

 

If the system is in printer-capture mode (following a PON command), the QUIET command suppresses the display on the screen, but the output data are still captured and may be printed or saved at a later time.
