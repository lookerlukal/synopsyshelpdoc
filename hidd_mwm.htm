MWM – Select a MACro File to Edit or Run

 

The MWM window lists the names of all saved MACros.  Click on a name, then click ‘Run selected’, ‘Edit selected’, or ‘Delete selected’ to run, edit, or delete that MACro.

 

If you type into the Search window and click the Search button, the program will only list those files that contain the entered string somewhere.  Thus you can select those files that perform a GHOST analysis, for example, by selecting that word.

 

You can also use the command-mode versions EM filename to execute a saved MACro, and LM filename to load that MACro into the editor, where it may be edited, saved, and run.
