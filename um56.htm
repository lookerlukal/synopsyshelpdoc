5.6 GLASS                                Next Previous TOC

 

Indices of refraction are specified to SYNOPSYS either as explicit values or in a form that permits recalculation as a function of wavelength.  This calculation may be based on either a glass model formula using the GLM input, or on a six-coefficient interpolation formula, which itself may be specified directly or indirectly in one of several ways.  Glass table requests (GTB ...) retrieve the coefficients from a stored table of data, while GLASS and GDF data enter the coefficients with the input stream (see 3.3.4).  The GFIT option will calculate the values of the six coefficients from index data input by the user.  Input for all of the above options is part of an RLE or CHG file, and the indices are computed automatically after the file is read, before the PXT is performed.

 

This section describes a rarely used feature that provides additional flexibility.  An optional way to input the coefficients is with a separate GLASS file (not part of an RLE file) of the form shown below.

 

 

GLASS

SN

NB NB NB        (six coefficients, taken from the glass catalog.)

NB NB NB

[ ALSO SN SN …]        (to enter other surfaces with the same glass type)

SN

…                (next surface data, etc.)

END

 

Here is how you would enter the coefficients for a glass like LAH55 on surface 1;

 

GLASS

1

3.27862859  -0.01642532  0.02910141

0.00126438  -0.00007089  5.42510179E-06

END

 

Note that the coefficients to be entered in this way are in the form used by the Schott Company for many years, and that newer catalogs generally use Sellmeier coefficients, which are not appropriate for this feature.  In such cases, the GFIT option is recommended instead.  (The glass-table feature (GTB) employs a variety of formulas, which are automatically selected according to the published data.)
