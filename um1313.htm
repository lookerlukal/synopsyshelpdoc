13.13 Scrolling to Recently-typed Commands                         Previous Next TOC

 

A recently-typed command may easily be run again without retyping it; just press the <up-arrow> or <down-arrow> keys to scroll through a buffer.  Then press <CR> to run the one you want.

 

You can also load the last 20 commands into a MACro editor with the command

 

LLB                (Load Line Buffer).
