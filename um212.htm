2.1.2 Command-level Input                                                        Previous Next TOC

 

All other input is termed command level input, and usually consists of a mnemonic followed by one or more numeric entries or other mnemonics, such as

 

RAY 2 1 0 1 SURF

 

Only the first three characters of command-level mnemonics are required.  (In AI mode, four characters are occasionally required, and input names defining glass types or file names must be spelled exactly.)

 

Input words, both alphabetic and numeric, must be separated from the next word by one or more blanks; except for this rule, the columns are generally not important.

 

Numeric input may be in any of the following formats:

 

6

integer number

-3.456

decimal number

1.0E-6

explicit exponent

0376.29843E04

 

-2.608E 3

 

+2.0E+ 1

leading plus signs are ignored

 

NOTE: numeric words containing E (denoting an exponent) must have a decimal point.
