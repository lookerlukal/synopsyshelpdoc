MSS – Mode Switch Dialog

 

The MSS dialog displays the current settings of the SYNOPSYS mode-control switches.  You may see the entire set or subsets related to graphics, image analysis, or optimization.

 

These switches control many of the optional features of SYNOPSYS, such as listing of all of the rays in a spot diagram or excluding dummy surfaces from lens drawings.  In addition, many of the other dialogs show the button which will bring up a display of only those switches that are relevant to that dialog and its features.

 

To restore the SYNOPSYS default switch settings, use the command RESET.  It is recommended that the user customize the switch settings according to his needs by including the preferred settings in a MACro named CUSTOM.MAC; this MACro is automatically called by the default startup MACro SYSTART.MAC, and should include the commands ON number number … and OFF …, as explained in section 2.9.

 

You can also get a listing of the current switches that are on with the command SSW, and more information in Appendix B.
