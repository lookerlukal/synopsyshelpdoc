15.1.4.4 Examples of Zn calculations                                        Previous Next TOC

 

1.

Suppose you want to evaluate the quality of a set of lenses, and define “quality" as the average of the RMS blur at three field angles.  Here is a MACro that will print this average:

 

RMS 2 0 400

Zl = FILE 1

RMS 2 .7 400

Z2 = FILE 1

RMS 2 1. 400

Z3 = FILE 1

= (Z1 + Z2 + Z3) / 3

 

2.

This example makes a plot of the absolute distortion over the field (the DIS command plots normalized distortion).

 

MACro:

Z1 = YA AT HBAR = AIP

Z2 = GIHT

Z3 = AIP

= ZI - Z2*Z3

ORDINATE = FILE 1

 

AI command:

DO MACRO FOR AIP = 0 TO 1

 

Note that we had to define parameters Z2 = GIHT and Z3 = AIP since the equation allows only Zn's and constants (plus math symbols).  After the distortion is calculated by the equation, the results are automatically placed into the file buffer, where the ordinate can pick up the result, in file location 1.

 

3.

Here is how you could use the Zn parameters to set up the targets for optimizing a zoom lens.  First we set the value of Z1 to the GIHT we want in zoom 1, and then generate the desired target values for nine more zooms.

 

Z1 = 4

= 1.44444*Z1

Z2 = FILE 1

 

= 1.888889*Z1

Z3 = FILE 1

 

= 2.333333*Z1

Z4 = FILE 1

 

= 2.7777778*Z1

Z5 = FILE 1

 

= 3.222222*Z1

Z6 = FILE 1

 

= 3.6666667*Z1

Z7 = FILE 1

 

= 4.1111111*Z1

Z8 = FILE 1

 

= 4.5555555*Z1

Z9 = FILE 1

 

= 5.0*Z1

Z10 = FILE 1

 

In the merit function we can utilize these targets with input such as

 

AANT

ZOOM 1

M Z1 10 A GIHT

…

ZOOM 2

M Z2 10 A GIHT

…

END

 

Why would one want to do things this way?  Well, suppose the customer specified a magnification range but did not give exact values at any point.  You can run this optimization and evaluate the quality – and then change Z1 to a value of 5, for example, and repeat the exercise.  Then compare them to see at what values the performance is best.  Parametric studies like this are easy to do, since you do not have to work out all of the targets yourself by hand.
