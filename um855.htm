8.5.5 RMS Focusing (FOCUS)                                 Next Previous TOC

 

To find the focal position that minimizes the RMS spot size, the input is

 

       [X/Y]FOCUS { ICOL / P / M } HBAR NRYS 0 GBAR

or        FOCUS DF

 

(Another feature, VFOCUS, will find the focus position that gives the minimum wavefront variance.  See that link for more information.)

 

The FOCUS program calculates the best focus in the color given by ICOL, the primary color (“P”), or all colors (“M”) at the entered field point, and alters the previous value of BACK accordingly.  If a number is entered for DF and the rest of the line is blank, the value of DF will be added to the previous value of BACK and no other refocusing will be done.

 

If the lens has a YMT solve on the final airspace, the solve remains in effect, but a value is inserted for the quantity BTH (see section 3.2).  Note that the BTH option remains in effect after the FOCUS command is processed, and if you alter the lens so that the YMT solve gives a different result, that result is again offset by the same BTH parameter.  If you want to remove the BTH, enter BTH 0 in a CHG file (or remove the YMT solve).

 

If the lens is in AFOCAL mode, the accommodation is varied, in diopters.

 

The program creates an optimization file with the appropriate variable parameter and creates a GNR rayset at the requested field and color(s) according to the current color weights.  It adjusts the grid spacing for the rayset to approximate the requested number of rays, limiting the latter to 20 x 20 to avoid overflowing the aberration array, and fewer if focusing in more than a single color.  When the process is finished, the program automatically runs the CHANGES command, which lists the changes to the variable parameter.

 

If XFOCUS or YFOCUS are entered, only the X or Y coordinate of the spot is considered.

 

FOCUS may be run interactively from the dialog MGI.

 

You may run your FOCUS command file via the MACro editor, but do not run it as the alternate MACro.  FOCUS creates its own alternate MACro to do the optimization, so this must be vacant.
