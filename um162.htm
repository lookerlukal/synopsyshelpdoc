16.2 Analysis Mode                                                        Next Previous TOC

 

Following the command ANALYSIS, the program will accept the input:

 

[ RFILM ... ]

[ PHASE ]

[ TRANS ]

[ LAM LLOW LHI NSTEPS ANGLE ]

[ PHI PLOW PHI NSTEPS WAVEL ]

[ PLOT [ SCF ORIGIN ] ]

[ EXIT ]

[ RETURN ]

[ DESIGN ]

 

any other command will return to common mode.

 

PHASE

requests phase output in the analysis.

 

TRANS

requests that the next plot show transmission instead of reflectance.

 

LAM ...

requests an evaluation over wavelength, between LLOW and LHI in NSTEPS steps at a fixed angle of ANGLE. Up to 200 steps may be requested.

 

PHI ...

requests an evaluation over angle, between PLOW and PHI in NSTEPS steps at a fixed wavelength of WAVL.

 

PLOT

causes the previous analysis to be converted into a plot, with a scale factor on the Y-axis (ordinate) of SCF per inch, if entered, or automatically scaled if not.  The desired ordinate ORIGIN value may be entered in word three.
