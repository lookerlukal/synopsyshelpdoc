3.6.3 Lens Initialization (NLENS, SLENS)                        Previous Next TOC

 

The command NLENS (New LENS) will erase the current lens and replace it with a dummy system with two surfaces and a default object description, ready to be altered by the user.  This command will prompt you whether you really want to erase your current lens and start over.  (The file header command RLE will also erase the lens and permit immediate input of user data.)

 

A similar command, SLENS (Starting LENS) works the same way but does not prompt you first.  This is intended to go into your CUSTOM MACro, (see section 13.2.1), and the result will be that you start the program with a new, blank lens.  But the program does that anyway, so what’s the deal?  Simple.  The CUSTOM MACro is executed after the other startup procedures are completed, and it is here that you will set your favorite switches on or off. Then the program will execute the SLENS command, if it is in the MACro, and you get a starting lens that honors those switches.  Users in countries with the metric system will appreciate this.  Just turn on switch 24 in that MACro.
