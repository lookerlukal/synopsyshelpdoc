Show Icons

 

Clicking on brings up the MACro editor with the current version of the file ICONS.MAC.  This file is used to maintain your choice of symbols to be substituted by the AI program in response to clicks on the SideBar as well as user input in the Command Window.  This file is usually called in the startup MACro SYSTART.MAC with the entry EM ICONS, and it installs the current symbol substitution strings automatically each time you start SYNOPSYS.  You can also edit and run this MACro any time you want to change the current set of definitions, and, of course, define additional strings by typing them into the Command Window yourself (although the latter will remain in effect only until you terminate the current SYNOPSYS session).

 

See AI symbols for a complete description of the AI feature.  Once the symbols have been defined and loaded, you can control which of the SideBar icons will actually generate the icon strings, and which will go to a menu instead, by clicking on the button .
