8.5.8 Examples of Geometric Image Evaluation Input        Next Previous TOC

 

1. Spot Diagrams

 

Assume that a lens description has been entered with RLE, and we wish to obtain spot diagrams at three wavelengths for an object at the 75% field point.  The image diameter is about 0.002 in., as determined previously by FANS output.  We would input

 

SPT 2 .75 150 .002

SPT 1 .75 150 .002

SPT 3 .75 150 .002

 

The above input would produce three separate spot diagrams.  If a composite spot diagram of all three colors were required, we would use the multicolor option, as follows:

 

 SPT M .75 150 .002

 

Another way is to use the through-focus program:

 

TFS 150 .002

HBAR .75

ICOL M

PLOT

 

where no defocus is specified, resulting in evaluation at the nominal focus position only.

 

2. Composite Knife Edge Trace

 

We want a five-wavelength composite knife-edge trace.

 

KNI M 1 200 .01 0 Y P

 

A listing and plot of the composite knife-edge trace will be produced.  The plot will have a horizontal scale of 0.01 inches per inch, and the evaluation will be performed by a vertical motion of the knife (Y) at the full field image (HBAR = 1).  (Here, we assume that the lens has five wavelengths defined.)

 

3. MTF

 

A lens is to be refocused at the 75% field point, and monochromatic MTF's obtained at four field angles:

 

FOCUS 2 .75 200

FCO 200

GMTF 2 0 200 0 0 P

GMTF 2 .5 200 0 0 P

GMTF 2 .75 200 0 0 P

GMTF 2 1 200 0 0 P

 

Each MTF will be printed and plotted, with a cutoff frequency of 200 lines/mm. Section 8.6.3.3 shows how to get a multifield diffraction MTF plot, showing all fields on the same picture.
