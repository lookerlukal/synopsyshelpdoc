7.4  Field Curvature (FCV)                        Next Previous TOC

 

A plot of the tangential and sagittal field curves may be obtained with the input

 

FCV SCF [ { P / 0 } [ GBAR [ ICOL / P / M ] ] ]

 

where SCF is the longitudinal defocus per inch on the abscissa of the plot.  The calculation is performed in the meridional plane (unless the optional word "GBAR" (not a number!) is entered in word 4 of the command, indicating the X-Z plane) by tracing the chief ray and a differential ray at 25 field angles.  The separations of these rays, together with the known (local) F/number give the defocus position at which they would coincide in either the X or Y direction.  These are output in a table and will also be output in a plot, if the optional [ P ] is entered.  A sample plot is shown below.  For AFOCAL systems the results are in units of semi-angular spread at the full aperture, or diopters if the ACCOM option is in effect.  A SCF of zero gives default scaling.  This command can also be run from the MRR dialog.

 

 

A color number may be entered in word 5, "M" plots the field curves in all defined colors, and "P" is the default primary color.   Plots are shown in color if switch 35 is on, displayed with a stored color table.  If switch 85 is also on, the colors will approximate the visual appearance of the wavelength analyzed.

 

A sample multicolor plot is given below, produced by the command

 

FCV .05 P 0 M
