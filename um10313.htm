10.3.1.3 Variance-based Aberrations                                                Next Previous TOC

 

The wavefront variance is a useful measure of image quality, and the VARIANCE command can calculate its value along with the standard deviation and the image point in X and Y.  The latter two quantities give the location of the center of the reference sphere at which the variance is minimized, measured from the chief ray point in the requested color or the primary color if the analysis is multicolor.  This is a useful approximation to the location of the peak of the diffraction pattern as long as the OPD errors are less than one wave or so.  For lenses where the distortion is extremely important, such as for microlithography objectives, one wants to control that location and not just the location of the chief-ray intercept -- which is a more common target for correcting distortion.  Aberrations described in this section allow one to do just that.  The formats are:

 

M TAR WT A { ICOL /P / M } WVAR HBAR GBAR NRYS

 

M TAR WT A { ICOL /P / M } WSTD HBAR GBAR NRYS

 

M TAR WT A { ICOL /P / M } YIP HBAR GBAR NRYS

 

M TAR WT A { ICOL /P / M }  XIP HBAR GBAR NRYS

 

M TAR WT A { ICOL /P / M }  IPX HBAR GBAR NRYS

 

M TAR WT A { ICOL /P / M }  IPY HBAR GBAR NRYS

 

These go into the AANT file.  The values of XIP and YIP give the distance from the reference point to the image point, which is near the peak of the diffraction pattern.  The reference point is the chief-ray point at the requested field angle in the requested color, or in the primary color if the multicolor "M" is requested.  WVAR returns the wavefront variance, and WSTD returns the square root of the variance, which is the standard deviation of the wavefront.

 

The quantities IPX and IPY are related but different.  Here, the program finds the image point, as above, and then also finds the corresponding point for the on-axis object.  The returned values are then the X and Y differences, in lens units, between the two.  Thus, if the on-axis image is perfect the calculation returns the location of the image point at the requested field.  If the on-axis image point has nonzero coordinates (because of axial coma, for example), the returned value is measured from that point instead of from the vertex.  To control distortion, therefore, one can simply target the difference between a desired image scale and the value returned by IPX or IPY.

 

The calculation is performed by the VARIANCE routine, which traces approximately NRYS rays and finds the desired quantity.
