Window menu commands

The Window menu offers the following commands, which enable you to arrange your windows:

New Command Window

Creates a new Command window.

Clear Command Window

Erases everything in the active Command Window.

New Graphics Window

Creates a new, empty window where the next graphical output will be displayed.

Cascade

Arranges windows in an overlapped fashion.

Tile

Arranges windows in non-overlapped tiles.  The TILE command puts new windows in a tile grid.

Arrange Icons

Arranges icons of closed windows.

 

Window 1, 2, ...

Goes to specified window.
