12.1.3.5 Additional quality descriptors                                        Next Previous TOC

 

In addition to the image-quality descriptors in the above sections, BTOL can examine the change in three other properties of the lens.  These give the location of a reference point found by taking the average of the intercept points of two rays at a given field and at aperture points YEN = +/-  0.473.  Those rays closely approximate the location of the diffraction image point when the lens has a small amount of coma.  For a well-corrected lens they are essentially equal to the chief-ray point, while for tolerancing, when the lens may display axial coma, the result shows how the image location changes when that aberration is accounted for.  Input is:

 

 [ TOL MAGNIFICATION MTOL ]

 [ TOL DISTORTION DTOL ]

 [ TOL BORESIGHT BTOL ]

 

TOL MAG

controls magnification change by specifying a maximum change in the reference point, minus the boresight shift.  The default value is 100.0, which is intended to disable this feature until it is needed.  Note that the magnification change in a lens without focus compensation is different from that in a lens with refocusing.  This should be taken into account in planning the adjustments to be specified.

 

TOL DIST

gives a maximum change in distortion, defined by taking the change in the reference point of the chief ray at the .5 field point and subtracting one-half of the change of MAG.  For lenses with a substantial amount of actual distortion, this is not necessarily appropriate and should not be requested. The default value is 100.0, which is intended to disable this feature until it is needed.

 

TOL BORE

controls boresight error by giving a maximum change in the reference coordinate at the on-axis point. The default value is 100.0, which is intended to disable this feature until it is needed.

 

To enter values for these three boundary conditions, the tolerance should be in linear lens units, except for AFOCAL lenses, for which it is in radians.

 

Note that you may include any or all of the boundary conditions BORE, MAG, and DIS, but only a single TOL or DEGRADE command that references a quality descriptor.
