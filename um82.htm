8.2 Multicolor Analysis                                Next Previous TOC

 

The wavelengths and weights entered in the RLE or CHG file define the colors for multicolor analysis.  The dialog MMC provides an interactive way to change the wavelengths and weights, and to assign specific wavelengths to the primary, long, and short colors (see CORDER).  They may also be edited in the SYS dialog.  The wavelengths of a multicolor analysis are examined in the CORDER order.  The Spectrum Wizard (dialog: MSW) can automatically select wavelengths and color weights according to the spectral output and sensitivities of a variety of sources and detectors.

 

To use the multicolor data entered in the above fashion, the entry M should be used instead of the normal ICOL data in those image analysis commands that can accept it, as described in the following sections.

 

If an explicit color number is entered on an image analysis command, a single-color analysis will be performed.  A color entry of “P” requests a single-color analysis in the primary color.
