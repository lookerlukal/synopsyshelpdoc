3.6.1.1 Inserting and deleting surfaces                        Previous Next TOC

 

Surfaces may be inserted or deleted in a variety of ways. The commands

 

INSERT { SN / JSSS JSPS }

 

DELETE { SN / JSSS JSPS }

 

operate in command mode; the AI program can insert or delete elements (see 15.1.5.2), and the PAD|Worksheet feature can insert or delete surfaces or elements and update the lens drawing immediately.  You can also do this within a CHG file, as shown below, or with additional commands described in section 5.27.

 

To insert a surface within a lens, enter (in the CHG file)

 

SN SIN

(Surface INsert).

 

This instruction automatically causes the surface data for the specified surface and all higher-numbered surfaces to be transferred to the next-higher numbered surface.  The specified surface is left relatively clean: flat, zero thickness, immersed in AIR (unless the surface with that number was previously buried); pickups, solves, tilts, decenters, deformations, TAGs, and CAOs are removed for safety.

 

No additional surface data may appear on the SIN input, and the surface renumbering is performed immediately.  This means that to change or insert additional surfaces you must refer to the renumbered surfaces rather than to the old numbers.

 

To delete a surface, enter

 

 SN SOUT                (Surface OUT).

 

The data applying to all higher-numbered surfaces are transferred to the next lower one.  Renumbering is performed immediately, and subsequent changes should refer to the new numbers.  If the deleted surface is the object of a pickup elsewhere in the lens, results maybe unexpected – so it is wise to delete such options first.  If the SOUT instruction causes the stop surface to become surface number one, no pupil solve is performed and the current value of YP1 is retained.  If a value of zero is desired, you must enter a new object or APS yourself.

 

The SOUT option deletes the requested surface but does not change any thicknesses or airspaces before or after.  If you wish to remove an element or surface and have the neighboring elements remain where they were, the WorkSheet method should be used instead.
