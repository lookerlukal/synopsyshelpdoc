4.1 Lens Specifications (SPEC)                Previous Next TOC

 

The command

 

 

produces a listing of the lens data and a table of first-order quantities.  The surface data are in fixed-point notation and most easily communicated to persons not familiar with SYNOPSYS printouts. Section 6.1 defines the paraxial terms used in SPEC listings.  A portion of a lens may be listed if the surface numbers are given with the SPEC command.  The default is the entire lens.  If the SYS option is entered, SPEC prints only the first order table.  The SURF option gives only surface data, and FULL adds a CAP listing at the end.  The RTG option prints only the symmetric data.  This feature is also found in the MLL dialog.

 

Each surface has a medium identification code that describes the material or index of refraction.  For air-spaced surfaces this is AIR or VACUUM, and for other surfaces the identification will be one of the following:

 

glass type name        for GTB glass table assignments

 

TYPE        for GID / TYPE assignments

 

GLM-NDVD        for GLM glass model assignments

 

PICKUP        for index pickup options

 

GLASS        for GLASS data entries

 

GRIN name        for gradient-index materials

 

For each non-AIR surface the program also prints out the value of the index at the primary color  and the quantity ( primary -1) / ( short - long ), which is the effective V-number for the wavelengths in effect – except for surfaces designated model glasses (GLM).  For these the program prints out the values of Nd and Vd of the model, as indicated by the notation "GLM-NDVD" under the heading of medium for such surfaces. If another surface picks up the index of the glass model, that surface will get the actual index and effective V-number and not the model parameters – and will therefore you will see different values from the GLM surface if the wavelengths do not happen to be the CDF lines.

 

An “A” following the surface number indicates that the surface has an asymmetry (tilt or decenter), while a “T” indicates that the surface has a TAG in effect (see 3.3.12).  If a quantity is influenced by a pickup or solve, the number so affected will be followed by a “P” or an “S”, as a guide to keeping track of numbers that are likely to change as a result of changes elsewhere in the lens.  A surface that has been assigned MELT data is indicated with an “M”.  To see the details of the pickup or solve, you should request a POP listing (see below).  The details of tilts and decenters will automatically be printed following a SPEC listing (see ASY, below), if any are in effect.  If the optional GLOBAL is entered, the ASY output will be an ASY GLOBAL.  If the LOCAL option is entered, the ASY output will be an ASY LOCAL, and if the lens is in EXTERNAL mode and EXTERNAL is entered, the output will include an ASY EXTERNAL.

 

Surfaces followed by negative space (following an odd number of reflections if the object is declared RRAYS, the default) are indicated on the SPEC listing with the symbol “<-“.  If the object is declared LRAYS, the lens starts out in negative space and the symbol is not shown following an odd number of reflections.

 

You can defeat the listing of the surface flags (“S”, “P”, etc.) by turning on switch 74.

 

There is also a plotted version of SPEC, which shows a lens drawing as well as the numeric tables.  This is the DSPEC command.
