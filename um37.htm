3.7 Storing and Retrieving Lens Data                Previous Next TOC

 

After a lens has been input to SYNOPSYS, the data may be stored in the lens library or on a separate disk file, described below.  If not stored in one of these ways, the data are lost when the program terminates

 

The dialog MOS provides a link to the dialogs MLB for library functions, and MWL for accessing saved lenses.
