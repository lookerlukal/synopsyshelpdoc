External Coordinate System Dialog

 

This dialog is for editing the EXTERNAL coordinates of the system. This feature is used when you want to see the positions of surfaces or the path of a ray in the coordinate system of the laboratory or the telescope mount,.  The data give the position of surface 1 in the external system, and the Euler angles that define the orientation of the initial Z axis. You can select the Euler-angle order from a list of six possibilities.

 

When these data have been entered, you can utilize the commands

 

ASY EXTERNAL

 

and

 

ERAY {ICOL / P } HBAR XEN YEN [ SURF / 0 ] GBAR

 

In addition, the AI nouns XE, YE, and ZE can inquire about ray or surface positions, and AE, BE, and GE can reference the Euler angles themselves.  The same mnemonics can appear in the AANT file to target these coordinates or the PANT file to vary them.

 

These data may also be entered into the RLE file.  See Section 2.4.1.4.

 

This dialog is reached via the SPS dialog.
