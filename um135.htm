13.5 Graphics tools                                                        Previous Next TOC

 

Several commands are available to help manage your graphics windows.

 

As you make new graphics displays, SYNOPSYS can either show them in the same window as the previous display, or it can create a new window for each one (up to a total of ten Graphics Windows).

 

You can select either of these options with menu items explained in the above links, or you can make your selection with the commands

 

GAW                Graphics Add Window

GRW                Graphics Reuse Window

KAG                Kill All Graphics windows

BLANK        makes a blank Graphics Window

GFRONT        brings all graphics windows to the top of the Z-order.

 

You can also make a new, blank graphics window by clicking on the toolbar button .  Any new picture you request will go into this new window, leaving any existing pictures intact on the screen.  This button (and the BLANK command) also activate GRW mode.

 

These commands can be used in a MACro in a sequence such as

 

KAG

GAW

(commands to make a series of pictures)

BLANK

 

When this MACro is run, all of the previous windows will be closed, and then the MACro makes a series of new pictures, each in its own window.  Finally it makes a blank window and changes the mode so that as you continue to work, making new pictures in real time, each one goes into the last window that was created.  This way you do not constantly make new windows that clutter the screen, but your old windows are still displayed.

 

Some users like to define the exact color used as the background for PAD and graphics windows.  This can be done with a menu item found at Options | Graphics | Define Background Color.  You adjust the color by moving three sliders, and the color so defined will be displayed when you select the “Black Background” option in those windows.

 

In the event that you closed the program but forgot to save an important picture, SYNOPSYS can recover it for you.  Just restart the program and type

 

RLP                (Recover Last Plot)

 

The picture will be redrawn in a graphics window.  This works for all normal graphics, but not for RPER or RSOLID displays—and it does not recover any annotation you may have added yourself.

 

For convenience, here is a link that explains the variety of tools that are provided in a Graphics Window.  These allow you to zoom and pan the display, select between white or dark background colors, add your own annotation, and make hardcopy printout at various scales.

 

Tiling your pictures

 

Another handy feature lets you create up to 10 graphics windows and display them tiled over the monitor.  The procedure, which you will create as a MACro, is:

 

TILE NCOLS TSIZE TSPACE XLOC YLOC

(Create a picture)

(Create a picture)

(Create a picture)

...

ETILE

 

When the TILE command is read, the program erases all open graphics windows; then it processes the remaining lines, displaying each picture in turn, drawing each one in the next tile location.  When the ETILE line is read, the tiling stops.

 

This input gives the number of columns (NCOLS), the size of each picture, relative to the size of the screen (TSIZE), the spacing as a fraction of the size (TSPACE), and the location of the upper-left picture in X and Y, as a fraction of the screen size (XLOC, YLOC).  Monitors have a variety of screen sizes and ratios, and there are many resolution settings that can affect the location of individual pixels.  If your tiles come out overlapped or too widely spaced, you can adjust things with the TSPACE parameter.

 

There are limits on XLOC and YLOC.  The valid range of these parameters is 0.01 to 1.0, and if you enter a value outside this range, a default value of 0.6 in X and 0.0 in Y will be applied.

 

Defaults are TILE 4 0.125 1.0 0.3 0.01.

 

If you make additional pictures after all the tiles are drawn, they will be drawn in the last usable window in the tiles.  If you want the tiles to remain unchanged, just click the New graphics window button on the top toolbar or type the command BLANK.  That will open yet another graphics window where more pictures may be displayed.

 

This is a great way to see several analysis pictures at once, whenever you want.  Just put the commands in a MACro and run it.  You can even program one of the sidebar buttons to run that MACro or define an AI command to do the same.  Then, with the click of that button you see all your plots.  Here is an example:
