9.2.1 Find Detector Size (FDS)                Next Previous TOC

 

Another use for the image model described in Section 9.1 is to find the radius of the circle that just encloses a given percentage of the image energy.  You can read the same information off of the encircled energy plot, described in the next section – but here is a way to get exactly the answer you need when you don’t want to see the whole curve.

 

After the model is created, enter the command

 

 FDS PERCENT XOS YOS

 

PERCENT is a number between 10 and 100, and XOS and YOS give the position, at the image, of the center of the circular area to be evaluated.  If the requested amount of energy is found within the image model, the program prints the radius of the circle that encloses it.  If not, an informative error message is produced.  (If PERCENT is a value between 0 and 1, the program will interpret it as a fraction rather than a percent.)

 

For example, here is how to find the 80-percent circle for a telescope:

 

GET 4

DMOD M 1 500        ; make a multicolor diffraction image model

FDS 80 0 0                ; find the 80% energy circle

 

 

THE RADIUS OF THE CIRCLE CENTERED AT    0.000       0.000

CONTAINING  80.00 PERCENT OF THE ENERGY IS 0.60136E-03
