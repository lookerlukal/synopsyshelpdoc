10.3.1.4 Wavefront coefficient aberration                                        Next Previous TOC

 

Individual terms of the wavefront expansion at a specified field and color may be controlled with the aberrations

 

{ A / S / MUL / DIV } ICOL COE HBAR GBAR nb

{ A / S / MUL / DIV } ICOL ZCOE HBAR GBAR nb

 

where nb is the desired coefficient.  COE refers to the terms of the power-series expansion of the COE feature, while ZCOE is the Zernike term.  Thus to obtain a value of 5.4 for the 3rd  Zernike coefficient at full field in color 2, one would input

 

M 5.4 1 A 2 ZCOE 1 0 3

 

in the AANT file.

 

Note that the results from this aberration might differ somewhat from the output from ZCOE.  This is because, to execute as quickly as possible, the aberration calculation ignores vignetting (as do all rays in the AANT file).  If vignetting is an issue, you might either remove the offending apertures or switch to WAP 3 (which is very slow).
