SPS Thickness Solve Dialog

 

Thicknesses and airspaces are all called "thicknesses" in SYNOPSYS.  Usually you will enter a specific number for the value, but you can also let the program calculate the value that yields a desired ray path thereafter.  If you select a "YMT" solve, the thickness will be adjusted so that the paraxial value of the Y-coordinate of the axial marginal ray will be the target value.  A "YPT" solve, on the other hand, will adjust the thickness so that the full-field chief ray will hit the next surface at the desired location.  A similar set of options operates in the X-Z plane.  You can only select a single option at a time.  For more information, see Section 3.3.3.

 

A solve may be deleted by clicking the button "Delete Solve".
