2.3 Log Number                                                        Previous Next TOC

 

Each SYNOPSYS run is assigned a log number, which is automatically incremented and displayed at the start of each run.  The current value of the log number may be incremented at any time by the user with the command

 

LOG

 

or initialized to a desired value with

 

 LOG SET NB.

 

You can see the current log number with

 

 LOG SHOW.

 

The current log number is automatically assigned to a lens when it is changed, and it appears on all printed and plotted output on the same line as the ID information to help you keep track of which run produced which lens.

 

You can also set the log number assigned to a lens to a desired value with the command

 

 LOG LENS NB.

 

In addition to these tools, the AI program can access the current log number with sentences like

 

LOG?

LOG = 331

change log to 56 .

 

 

It is also possible to reassign the log number associated with a particular lens with AI.  This is normally set to the current log number whenever the lens is changed, but sometimes you may want the number displayed on plots to be that of the saved lens, not the current log number, after you have changed it in some way.  Here is how to use AI to accomplish this:

 

 

Z1 = LENS LOG

 

(Change lens in some way)

 

LENS LOG = Z1

 

(Analysis that shows the lens log number, such as drawings, etc.)

 

You can also use the special noun "LLOG" to refer to the lens log number.
