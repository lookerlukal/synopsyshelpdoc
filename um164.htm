16.4 Example FILM Input                                                Next Previous TOC

 

The following input file will create a band-pass filter on a substrate of BK7, fix the indices, and analyze the resulting stack over wavelength at an angle of 30 degrees.  The resulting design is saved with a name of "EXAMPLE.FIL".

 

FILM

DESIGN

BUILD 20 1 1.517

 

AANT

GRW 0 30 25 .4 .55

GRW 1 30 25 .57 .85

END

 

FIX

SYNO 5

ANALY

LAM .4 .85 100 30

PLOT

SAVE

EXAMPLE

 

RETURN
