5.44 Slope Analysis (SLOPE)                Previous Next TOC

 

One often would like to avoid surfaces with very steep slopes, since they are more expensive to make and harder to coat than are shallower surfaces.  The slope can be controlled in the AANT file with the DSLOPE aberration, but to use that tool effectively one must know what the slope on a problem lens is currently.  Then one can nudge it back gently, hoping that the image quality remains high.  The slope control monitor ASC can also be used to control slopes; it will keep track of the slope of all surfaces whose curvature is a variable.

 

The command

 

SLOPE [ JSSS JSPS ]

 

will calculate the slope of every surface in the range from JSSS to JSPS.  The default range includes all surfaces except the image surface.  It takes the current CAO value as the Y-coordinate to be evaluated, or the current EFILE coordinate “A” or “E” if one has been assigned.  The calculation is performed in the Y-Z plane, and the result is the angle in degrees.  If you also want the slope at the image surface, you must enter an explicit range yourself.  An example follows.
