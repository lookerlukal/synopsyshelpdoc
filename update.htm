How to update your support contract:

 

SYNOPSYS requires a security key (a “dongle”) to run all of its features, and the date of expiry of your support contract is hard-coded into that key.  When your support period has run out, you can still continue to use all versions of SYNOPSYS that were released before the expiry, but you cannot then run newer versions.

 

To renew your support, simply follow these steps:

 

1.  Go to our website www.osdoptics.com, and click SYNOPSYS Software > Licensing

 

There you will find a link to

 

Contact us at info@osdoptics.com for more details.

 

Your license update purchase will be handled by the support personnel at that address.

 

When you installed the key drivers initially, you were instructed to download the file KeyFiles.zip, unzip it, and save the contents in a safe place.  Now you must use one of those files.  Double-click on the file SecureUpdateUtility.exe, make sure your key is plugged in, and click Generate Request Code.  This will create a small file, which you should save with a name that tells us the key number, such as KEYnn.req..  Then email this.req file to us at info@osdoptics.com.

 

When we receive this file, we will send you a .upw file, which you should save somewhere.  Then open the utility again, browse to the file you just saved, and click Apply Code.

 

That’s it!  Your key is now programmed for the new support period.
