12.1.3 BTOL Quality descriptors                                                Next Previous TOC

 

The kind of image quality criterion to be examined by BTOL is selected from four categories: wavefront variance, spot size variance, Strehl ratio, and the convolution MTF.  These are discussed in the following sections, and all begin with the mnemonic TOL or DEGRADE. Note that you may include any or all of the boundary conditions BORE, MAG, and DIS, but only a single TOL or DEGRADE command that references a quality descriptor.

The tolerance value that is input for any of these quality descriptors must exceed the initial value (except for the MTF and Strehl ratio, where it must be lower).  The greater the difference, the looser the tolerances.  A default tolerance (1.5 times nominal) will be substituted if this is not the case, and a warning message printed.

 

In addition to the above, BTOL can examine the change in three system properties: the magnification, boresight, and distortion.
