5.65_SWAP

Sometimes one wonders if a lens could be improved by swapping one element for another.  This command will interchange two lens elements so you can test that idea.  The format is

 

SWAP sna snb

 

where sna and snb are the surface numbers on the first side of the elements you wish to swap.

 

Thus for example, in the lens below, we would like to see what happens if the last two elements are swapped.

 

 

First, we have to delete any solves or pickups on the affected elements, since that would alter the revised data.  Then we can use the SWAP command.

 

CHG

NOP

END

 

SWAP 5 7

 

This command swaps the elements, and now we can edit the lens, add new solves, reoptimize, and so on.

 

The SWAP command does not work if the lens has any pickups or solves or any advanced index options, such as GRINs, BIREF, etc.  Tilt, decenter, and global data are left unchanged, so element locations should remain unaffected.  But check carefully afterward, to verify that things are where you want them to be.  Edge data entered with the EFILE editor MEW are removed by this command, since ray paths will likely be substantially altered and the previous apertures may no longer be appropriate.
