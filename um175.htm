17.5 Birefringent Materials                                                Next Previous TOC

 

When a ray passes through certain crystals, such as calcite, the index of refraction depends on the direction of travel and polarization of the ray.  A common application of such materials is the waveplate, discussed in Section 17.5.2.  An individual element can also be modeled in SYNOPSYS with the input

 

SN BIREFRINGENT PHI CHI PSI [ SPECIAL ]

name

 

[ N1 N2 N3 … N10 ]               ! ordinary indices

[ N1 N2 N3 … N10 ]               ! extraordinary indices

 

in the RLE or CHG file.  Here, name is CALCITE, CRQUARTZ, SAPBIREF, or some other name.  The three angle cosines PHI, CHI, and PSI give the crystal axis directions in (X,Y,Z); If the axis points in the +Y direction, these would be (0,1,0).  If the name is not CALCITE, CRQUARTZ, or SAPBIREF, the SPECIAL option and the refractive indices must be entered.

 

Each such surface in the lens can in addition be declared

 

 

SN EXTRAORDINARY

(also turns off switch 57)

or

SN ORDINARY

(the default)

 

and the appropriate index will be chosen for raytracing.  (Paraxial raytracing always uses the ordinary ray.)

 

The index may also be assigned one of these values by input in the AANT file.  See Section 10.3.1.

 

A birefringent material will give rise, in general, to two images, formed in different polarizations.  A given ray is split into two rays at the interface, and the different propagation paths should be traced separately.  Since SYNOPSYS cannot literally split a ray, the program must avoid the need to find two different paths for the same ray.  The simplest procedure is for the user to analyze the image twice, specifying the ordinary ray in one case and the extraordinary in the other.  Only a single image is calculated at a time in this case.

 

Another method is available, however, when you want to see both images in the same analysis: if switch 57 is on, those analyses that generate a grid of rays will cycle through all permutations of ordinary and extraordinary indices for all of the birefringent materials in the lens, when tracing successive rays.  If the image analysis traces 9000 rays, for example, there would be about 4500 traced with the ordinary index, and another 4500 with the extraordinary.  If the number available for each of the permutations is less than about 400, there is the possibility of aliasing due to the equivalence of the raygrid to a coarse grating, and the requested ray number should be large enough to avoid this situation.  The system must be in one of the polarization modes.  If this switch is on, any user-entered EXTRAORDINARY attributes are lost after the cycles are finished.  Note that this switch only operates when a program-generated raygrid is involved, and with PRAY.  A sample system containing a single calcite crystal is shown below.

 

RLE

ID CALCITE CRYSTAL

WAVL CDF

POLAR CIRCULAR RIGHT

UNITS INCH

OBA 1 .1 .1

1 TH .003

1 BIREFRINGENT .707 0 .707

CALCITE

2 YMT 0

3

END

 

Here we have specified a crystal whose axis is in the X-Z plane, 45 degrees to the optical axis.  To see the double image that is produced, we enter

 

ON 57

PSPRD 2 0 400 0 0 3.5 L

 

and the result is shown below.

 

 

Note that the crystal quartz selection in the U glass catalog (CRQUARTZ) ignores the fact that this is an optically active material.  Also, be aware that this feature is not useful for analyzing a quarter- or half-wave plate, even though the latter are usually made of a birefringent material. Section 17.5.2 describes how to model such elements.

 

You cannot pickup a BIREF material in an ACON pickup file.
