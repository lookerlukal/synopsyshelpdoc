3.3.4.1 Gradient-Index (GRIN)                                Previous Next TOC

 

(These options go into an RLE or CHG file, and may be entered interactively with the dialog SPS.)

 

Several kinds of gradient-index materials may be specified to SYNOPSYS with input of the form

 

 SN GRIN name DT N1 N2 N3

or        SN GRIN name DT N2

 

followed by coefficients entered with

 

GC1 ICOL C1 C2 C3 C4

GC2 ICOL C5 C6 C7 C8

 

where name is one of the following:

 

RADIAL, AXIAL, URN, SPHERICAL, SELFOC, LUNEBERG, MAXWELL, USR, EYE, LOCAL.

 

A special version of a GRIN is used by ENM, which can model the change in index or surface sag as a result of environmental changes, such as temperature gradients, with a special input format, described in Section 3.3.4.2..

 

GRIN materials can only be used in systems with exactly three wavelengths in which the primary color is color 2.  The entries N1, N2 and N3 are the base indices.  If only N2 is entered, all three indices will have the same value.  Coefficients may be entered for three wavelengths, with the color number (1 to 3) given by ICOL.

 

The quantity DT is the integration increment.  A large value will permit faster raytracing, but a small value gives more accurate results; a good compromise for some systems is 0.05.  The program uses the fast raytrace algorithm of Sharma (Sharma, Kumar, and Ghatak, Appl Opt. 21, 984 (1982).)  The path of a ray within a GRIN is curved, and will appear as such on lens drawings.  Because raytracing involves numerical integration of a second-order differential equation, it takes many times longer to trace a GRIN than a normal material.  OPD's are computed, also using the techniques of Sharma (A. Sharma, Appl. Opt. 24, 4367 (1985)).  The paraxial programs and third-order program automatically take into account the effect of a GRIN, using the method of Sands (P.J. Sands, JOSA 60,1436 (1970)), with the exception of third-order chromatic aberration, which is not at present treated.

 

Coefficients are entered with GC1 and GC2 input.  These data apply to the several types of GRIN as follows:

 

RADIAL:

N = No + C1P**2 + C2P**4 + C3P**6 + C4P**8

(P**2=X**2+Y**2)

 

AXIAL:

N = No + C1Z + C2Z**2 + C3Z**3 + C4Z**4

 

 

URN:

N = No +C1Z + C2Z**2 + C3Z**3 + C4Z**4 + C5P**2 + C6P**4+C7P**6+C8P**8

 

 

SPHERICAL:

N = No + C1P**2 + C2P**4

 

(P**2 = X**2 + Y**2 + (Z - C3) **2)

SELFOC:

N = No (1- C1P**2/2)

(P**2 = X**2 +Y**2)

 

LUNEBERG:

MAXWELL

N**2 = No**2 (2 - P**2/C2**2)

(P**2 = X**2 + Y**2 + (Z - C1) **2)

EYE

see below

 

LOCAL

see below

 

 

The axial gradient is always in the direction of propagation.

 

The aperture of the ray within a GRIN is compared at each increment with the clear aperture, if any, that has been entered on the GRIN surface.  If this aperture is exceeded, and the lens is in VIG mode (see 3.2), the ray is vignetted.  The GRS command will show statistics of the index within a GRIN.

 

The lens of the human eye can be modeled as a special type of GRIN.  This type does not allow user-entered coefficients, since eyes are all more or less the same.  The formula and coefficients were provided by Dr. Jose' Antonio Diaz Navas in the form below:

 

A complete model of the eye must also include the cornea and other structures; these can be modeled with data from the U catalog; an example is given below.  The GRIN portion is defined by

 

SN GRIN EYE DT AGE

 

The base index and other data are coded into the definition and cannot be entered by the user. AGE in this case gives the age in years of the subject eye.

 

It is useful to model the effect of a local index inhomogeneity in a lens element.  This uses a special form of GRIN input, as follows:

 

SN GRIN LOCAL DT N1 N2 N3

GC1 ICOL X0 Y0 Z0 DN

GC2 ICOL R

GRIN model of an inhomogeneous lens

 

The element at surface 13 of a microlithography lens has a small local index irregularity, and we wish to model the effect on the image.

 

CHG

13 GRIN LOCAL 1 1.476134 1.476137 1.47614

GC1 2 6 11 8 1.0E-5

GC2 2 5.9

END

 

Here, the grin data are entered with

 

GC1 ICOL X0 Y0 Z0 DN

GC2 ICOL RADIUS

 

In this input, we give the integration step (1 mm), and then the three nominal index values.  The GC1 line gives the (X, Y, Z) location of the center of the irregularity, which is modeled as a spherical volume with cosine profile.  The index change in this case is 1.0e-5 at the center of the volume, which is 5.9 mm in radius as specified on the GC2 line.  This type of GRIN input can only be assigned to color 2 in a 3-color lens, or color 1 in a 1-color lens, but the effects are applied to all colors.

 

Then we create a picture of the fringe pattern in the output beam for an object point at half field.  The effect of this small index variation is clearly visible.  The PSPRD plot is also affected.

 

FRINGES                                                                                   

PUPIL P .5 0 5 0 3 0 

 

 

PSPRD P .5 600 0 0 3.5 R
