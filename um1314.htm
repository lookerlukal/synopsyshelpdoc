13.14 Displaying several plots on a single page                 Next Previous TOC

 

There are five methods of creating a page containing more than a single graphics picture.

 

A.  Positioned by the Graphics Window

 

The first is to make the plot in the usual fashion, and then click the button .  You will be prompted to select which quarter of the page you want to send the hardcopy plot to.  To add another picture to the same page, you must feed the paper back into the printer again and repeat the process, selecting a different quarter.

 

B.  Positioned by Q1 through Q4

 

A second method is preferable when you want to prepare the layout of up to four pictures in advance, or create the entire set in a MACro.  This does not require feeding the paper in a second time, and it uses the sequence

 

Q1

(make a picture)

Q2

(make a picture)

Q3

(make a picture)

Q4

(make a picture)

QPLOT

 

The composite picture is shown on the screen, and you may annotate, save, or print it as you would any other picture.  This method works with all of the graphics features except RPER and RSOLID, which require their own drivers to display properly.

 

C.  Superimposed via CPLOT

 

 

The third method will superimpose separate pictures on the same page.  Annotation is suppressed for all pictures except the first.  Of course it is not meaningful to combine just any two pictures, since the formats, dimensions, and axis labels will be correct for the first but will not necessarily be correct for the others.  It is up to the user to insure that the annotation applied to the first picture is correct for all of them.  The sequence for this method is

 

CPLOT

(make a picture)

(make a picture)

(make a picture)

…

QPLOT

 

Here is how you might superimpose three encircled-energy plots on a single page.  The MACro contains

 

GET 4

CPLOT

COLOR RED

DMOD M 0 600

FOR CIRCLE

VARY Y SIZE FROM 0 TO .002

PLOT

 

COLOR GREEN

DMOD M .7 600

FOR CIRCLE

VARY Y SIZE FROM 0 TO .002

PLOT

 

COLOR BLUE

DMOD M 1 600

FOR CIRCLE

VARY Y SIZE FROM 0 TO .002

PLOT

 

QPLOT

 

The results are shown below.  Here, the axis labels are correct for all cases, but we have added labels to the individual curves with the annotation editor.  Note that in this example the default scale and origin are the same for all three cases, so the results are correct even though no scale was entered.  In general, you are not always so lucky, and we recommend declaring those parameters yourself.  See Section 9.3 for the formats.

 

 

 

The above MACro shows the plots in different colors.  This is easily done with the command

 

 COLOR color

 

before the command that makes the plot.  Here, the argument color must be one of

 

RED

BLUE

MAGENTA

CYAN

GREEN or LIME

BLACK

DEFAULT

LTMAGENTA

LTCYAN

 

The default color is yellow if the background is black, and black if it is white.  The option BLACK is similar except that the lines are white on a black background.  The requested color is in effect only for the next picture, and will be shown only if switch 35 is ON.  Many of the plots made by SYNOPSYS are themselves created in various colors, and in these cases those colors will take precedence, ignoring the color requested on a previous COLOR command.

 

D.  Superimposed by the Graphics Window

 

The Graphics Window provides a fourth way to combine pictures on a single page.  To use this feature, first prepare all of the pictures you want to superimpose, as usual, and save each of them with a unique name (as an SSP file).  Then erase the graphics window (or create a new, blank window) and click one of the toolbar buttons .  If you click the left one, the saved picture comes back in its entirety; the right one will return all of the lines but not the text.  Here is an example:

 

 

To prepare this picture, we first execute a MACro that sets up the desired case, with wavelengths, weights, and focus position.  This MACro ends with the lines

 

SCALE = .2 .2

ORIGIN = 0, 0

LTM STREHL_PLOT

 

The last line loads another MACro into memory.  This was prepared earlier, and contains the two lines

 

VAR M AIP 600

ORD = FILE 3                ; FILE location 3 is the Strehl Ratio for the VAR command

 

Now we analyze this case with

 

DO MACRO FOR AIP = 0 TO 1

 

(This runs the MACro that we loaded above, changing the value of AIP each time.  That value gets used in the VAR command as the HBAR parameter.)

 

The resulting plot shows the Strehl ratio plotted over the field.  Suppose the first case was at the nominal focus.  We save that plot with the name A.

 

Now we change the first MACro to change the focus for the second case, and repeat the procedure.  This time, however, we want the plot in blue; after the picture is shown, we enter

 

COLOR BLUE

AGAIN

 

The picture instantly turns blue, and we save it with the name B.  Repeat for the third case, this time requesting the color red, and save it with the name C.

 

Okay, all three plots have been saved.  To combine them, we first erase the graphics window, with the button , and then click the right button of the pair .  A file dialog opens, and we select the file B.  Click the same button and select file C.  Now there are two plots on the screen, one red and one blue – and no text.  Click the left button of the pair, and select file A.  The picture comes back with all of its annotation.  (It is wise not to superimpose the annotation from all three, since the time stamp will show all three times on top of each other.)  The colored pictures were added first so that the axes and annotation would be from file A, in black, plotted on top of the others.  If you like red or blue annotation, change the order.

 

To complete this drawing, we go to the Annotation Editor to add the extra text and the wavelength table.

 

E.  Combining plots with AI

 

Yet a fifth way to combine plots uses the ADD PLOT or ADD DO MACRO features, described in Section 15.1.2.

 

The quarter-size option (Q1, etc.) and CPLOT are mutually exclusive.  If you ask for one of these, you must not request the other for the same picture(s).

 

Some of the above options disable normal display of graphics until the sequence is completed.  In the event that you wish to abort the sequence, use the EPL command.
