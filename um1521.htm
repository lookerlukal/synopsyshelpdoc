15.2.1 Basics                                                                Previous Next TOC

 

There are five basic operations that can be done by AI:

 

1.

Evaluate a parameter (question verb)

 

2.

Change a parameter (change verb)

 

3.

Execute a loop in which one or both of the above are done, and prepare a plot of the results.

 

4.

Define a symbol as a string of characters.

 

5.

Evaluate an equation.

 

The next sections describe these operations in detail.
