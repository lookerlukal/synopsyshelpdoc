Object Wizard (MOW)

The Object Wizard can help you set up the object specifications for your lens.  The dialog functions much like the Pupil Wizard (MPW) but is organized differently.  You are encouraged to try both and select the one that best suits your needs.

 

You can select any allowed combination of object declarations, pupil definitions, and so on.  The program will disable any options that are incompatible with others that you have already selected.  The lens is not actually changed until you click the OK button*.

 

The Object Wizard can be found in the top menubar under “Wizard”, and may be accessed via the command MOW or with a toolbar button in the WorkSheet.  This Wizard will help you to set up the object, the stop surface and aperture, and control the paraxial and real entrance pupil definitions.

 

The Wizard works best if your lens already has all of the surface data.  Then it will let you select from the more exotic options that you may need.

 

This dialog is intended to be self-explanatory.  It differs from the SYS dialog by having a “Cancel” button.  This button will discard any changes you make while in the dialog (except for the VFIELD option, as noted below).  The SYS dialog, on the other hand, is more like a spreadsheet, and any changes you type are immediately entered into the lens and may in some cases alter the data elsewhere in the dialog.

 

The Object Wizard will echo the lens changes in the Command Window.

 

For a general discussion of stops and pupils, see Section 2.6.

 

* an exception is the VFIELD editor: changes are made to the lens immediately after you click the OK button in that dialog.
