Appendix G

 

GLASS TABLES                                                        Next TOC

 

When a material is designated in the RLE file as a glass-table glass with the GTB mnemonic, the program reads a set of interpolation coefficients from a stored table, and subsequently calculates the indices of refraction of the material when the RLE or CHG file is ENDed.  The coefficients exist in four forms: six Sellmeier coefficients used by most catalogs, a different set of six that applies to the polynomial used in some other catalogs, a set of twelve, which applies only to certain materials with very broad transmission bands which are found in the "U" (Unusual) catalog.  These go into a polynomial that is valid over the entire range of the material.  The last format uses 30 coefficients to describe Germanium as a function of wavelength and temperature; this applies to glass GE_NIR, found in the U catalog.

 

The polynomials are as follows:

 

1.  Sellmeier:

 

N**2 = 1 + ( A1λ**2 ) / ( λ**2 - A4 )  +  ( A2 λ**2 ) / ( λ**2 - A5 )  +  ( A3 λ**2 ) / ( λ**2 - A6 )

 

2.  Other glass catalogs:

 

N**2 = A0 + A1 λ**2 + A2 λ**-2 + A3 λ**-4 + A4 λ**-6 + A5 λ**-8

 

3.  12-Coefficient wide-band materials:

 

N**2 = A0 + A1 λ**2 + A2 λ**-2 + A3 λ**-4 + A4 λ**-6 + A5 λ**-

 

+ A6λ**4 + A7λ**6 + A8λ**8 + A9λ**10 + A10λ**12 + A11λ**3.

 

Some of the less-used of the unusual materials have but six coefficients, as mentioned in the section on the unusual catalog, below – and for these materials the valid wavelength range is narrower than for the 12-coefficient materials, and is given in the table.  For the material KRS-5, for example, you would specify the material as KRS5A if used in the range of 0.5 to 16.9 μM, and KRS5B if used from 17 to 40 μM.

 

The PAD toolbar button will display the several glass tables on the monitor screen.  There you can graph any of a number of properties of each of the glasses or select which ones to insert into your lens.  See section 13.3.5. You can also get to the table display with the command MGT.
