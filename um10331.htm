The mnemonic P1X specifies the X coordinate of point P1, and so on. SN is the surface number of the HOE.  Note that the above aberrations refer to the HOE itself, not the construction system.

 

It is also possible to specify targets for the HOE points and aberration coefficients coming from the construction optics itself, without necessarily considering the HOE system that will use these data.  The construction optics may in this case be either the main configuration or an ACON whose aberration coefficients are transferred to a HOE in the main configuration.  Either way, the following mnemonics apply:

 

 

HCOE gives aberration coefficient number NB while HPX, HPY, and HPZ give the location of the HOE point.  Since these mnemonics refer to the construction optics, and not the HOE system, it is not necessary to specify whether the point is P1 or P2 in this case.  The above mnemonics may not be specified for the system actually containing the HOE.

 

These options allow you to design a construction system so as to achieve a desired set of coefficients and image point.  If done as a single-configuration optimization, this is faster than designing both construction optics and playback optics simultaneously (although that procedure is more powerful and general).
