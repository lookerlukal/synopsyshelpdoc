16.1 Common Mode                                                        Next Previous TOC

 

The default mode is common, giving the prompt FILM>. In this mode the permitted entries are:

 

DESIGN

to change to design mode

ANALYSIS

to change to analysis mode

SAVE [ name ]

to save a multilayer stack.  To see a listing of all films that have been saved, use the command DIR FILM at the Command prompt.

FETCH [ name ]

to fetch a saved stack

RFILM

to enter new stack data

SPEC

to print the current stack

RETURN

to return to SYNOPSYS

PON

start printer capture

POF

end printer capture

EXIT

exit from SYNOPSYS

 

Certain other commands will automatically return to the top level SYNOPSYS command processor:  GDS, GADS, EM, EAM, LM, LAM, CCW, GNW, GRW, MWM, and PAD.  To run any other command, use the RETURN option first.

 

To enter a new stack file, the format is

 

RFILM

(Read FILM)

 

[ ID identification ]

 

[WAVL LAMBD0 ]

 

[ANGLE ANGLE ]

 

[MICRONS]

Specifies that the thicknesses are in microns, not waves

TH { IND / name }

See the list below.

TH { IND / name }

 

…

 

TH IND [ K name ]

 

END

 

 

Some common coating materials may be declared by name.  These include the following:

 

MGF2        (index 1.38)

SIO2        (index 1.47)

THF4        (index 1.52)

CEF3        (index 1.64)

CEO2        (index 1.74)

HFO2        (index 2.0)

ZRO2        (index 2.1)

TAO5        (index 2.154)

 

Material THF4 can be used over two regions: 0.27 to 0.7 um and 4.0 to 12 um.  The program selects the proper index polynomial for the region selected.

 

The name ALUMINUM may be entered in word four of the substrate line.  The program will then update the index and K value for this material at each wavelength analyzed.

 

Coatings defined with any of these materials will be modeled with index data calculated for the wavelength that is in effect when the coating is used.  Thus, the dispersion of the layers and the substrate is taken into account when analyzing the performance when the coating is inserted into a lens.

 

Note, however, that the actual index of these materials when deposited as a thin film usually differs from that of the bulk material, and the design and analysis in SYNOPSYS are therefore not to be taken too literally.  It is a good indication of the type and performance one can expect from a film stack, but the final details should be worked out by the company that will apply the coatings.  The index of a given layer can be different from one vendor to another, and the vendors we have contacted are not willing to disclose the values produced by their equipment.  Nonetheless, for our purposes, the film design and analysis use useful since it gives a good picture of what will eventually be achieved by a competent vendor.

 

If you have measured index values for a material of interest, or can get the Sellmeier or interpolation coefficients from the vendor, you can enter those data into a custom library and then use that material in your coating stack.  That should permit a more accurate evaluation.

 

LAMBDA0 defines the control wavelength.  Optical thicknesses are expressed as fractions of a wave, or in microns if they follow a MICRONS entry. ANGLE is the control angle.  Thicknesses and indices are entered without a surface number, starting with the incident medium and ending with the substrate, which may optionally be given an imaginary index component, K. Up to 50 layers total may be input.  To enter a quarter-wave coating on a substrate of BK7, for example, the input would be:

 

RFILM

0 1

.25 MGF2

0 1.517

END

 

Here we entered no control quantities, so the default wavelength of .5876 mm and an angle of zero degrees apply.  The coating has an index of 1.38, and a thickness value of zero is entered for the incident medium and the substrate.  When a new stack has been entered with RFILM data, the program automatically goes to ANALYSIS mode.

 

The RFILM file may optionally be input in design or analysis mode as well as in common mode.

 

A variety of commonly used film stacks are described in Section 17.2.  These may be placed on lens surfaces without using the features of this chapter or defining the film stack explicitly.

 

Note that each layer must be specified in a film stack, including the incident medium and the substrate.  For representative complex indices of common mirror materials, see section 17.2.5.  At least three layers must be present.
