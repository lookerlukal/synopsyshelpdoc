Top Toolbar

A toolbar is displayed across the top of the main SYNOPSYS window, below the menu bar.  This toolbar provides quick mouse access to many tools used in SYNOPSYS,

 

A SideBar is displayed on the left side of the window.

To hide or display the top toolbar, choose Toolbar from the View menu (ALT, V, T).

 

Click        To

Open the SPS spreadsheet lens editor. This is an intuitive feature intended primarily for new users.

 

Open the Sketchpad and the WorkSheet, where you can edit lens data with sliders or by typing into an edit pane.

 

Open an existing lens file.  SYNOPSYS displays the Open dialog box, in which you can locate and open the desired lens. The command MWL, which provides the ID of each lens listed, can also be used to open a lens file.

 

Runs the MWL dialog, where you can select from a saved lens file.

 

Save the active lens with its current name.  If you have not named the lens, SYNOPSYS displays the Save As dialog box.

 

Print the contents of the active window.

 

Open a new Command Window.

 

Erase the contents of the current Command Window.

 

Freeze this instance of the Command Window.  The program will make a new one, which you will then use for input and output, while the old one remains on the screen.  You may activate it, scroll it, copy from it, and print it – but you cannot enter input there or receive output.  This is intended to preserve a listing that you have made that you want to refer to at a later time, without any danger that you will accidentally send your new output there and change what you see.  If you close an unfrozen window while a frozen one is on the screen, any attempt to read from or write to the frozen one will generate an error message, and you must create a new Command Window to continue with your work.

 

Open a new window for editing a MACro.

 

Open a MACro file and start the editor.

 

Open the MWM MACro selection dialog.

 

Cycle through a range of text font sizes. This affects onscreen display only. Hardcopy text size may be adjusted via Options|Set Printer Offset, Margins, Font.

 

Freeze the current Command Window.  You cannot type into it, and the program will not write anything into it.  This is how you can preserve a listing on the screen to compare with a later one, with no danger of changing it by accident.  The program automatically opens (or finds) an unfrozen window, so you can work there.

 

Close all graphics windows. You can have up to 10 windows open at once.  After that you must close some, or all, of them to make new pictures.

 

When selected, causes new graphics to replace previous picture in the active window.

 

When selected, causes new graphics to open a new window. You can have up to 10 graphics windows open at a time.

 

Click here to open a new, blank graphics window.  The next picture you make will be displayed here, leaving any previous picture intact.  The command BLANK does the same thing.

 

This button brings all graphics windows to the top of the Z-order.  If your window is covered up by the Command Window or something else, clicking here will show it.  The command GFRONT does the same thing.

 

Turns on the AI (artificial intelligence) feature. You can do many tasks by typing an English sentence in the Command Window. In AI mode, type SUBJECT? or VERB? to see the vocabulary of the AI feature.

 

Turns the AI mode off. You can type SYNOPSYS commands into the Command Window, but not AI sentences.

 

Activates the SketchPad window, where you can monitor both the lens and the image while optimizing and during many other tasks.

 

Activates one of the six alternate configurations. All work from this point will refer to this configuration, leaving any others previously defined unchanged.

 

This button provides instant access to the SYNOPSYS help file. The question-mark button brings up the Help | About box showing the current version number.

 

Click here to stop a lengthy process that you do not want to complete, such as optimization or tolerancing.

 

Click here to rerun the last optimization after perturbing all variables.  See KICK.

 

Click here to run the simulated-annealing optimization feature.  See ANNEAL.

 

This button is normally gray, but turns red when certain errors are encountered.  Clicking here will display the error messages.

 

Click here to defeat the beeper that sounds when an error occurs.

 

Toggles the QUIET status.  When clicked, output is not sent to the Command Window.  Click it again, and the status returns to LOUD.

 

If you encounter ray failures on the first pass of your optimization, click this button.  It will create and run a quick optimization with a single aberration, the ray error value of the first ray in the AANT file that failed to trace.  The current set of variables will be employed to attempt to resolve the ray failure.  When it is finished, that ray should trace through the lens.  Other rays may still fail, of course, and you may have to run this feature more than once.  To speed things up, it is wise to put your most demanding rays at the start of the AANT file, so those rays will be fixed first.  Then the others are likely to trace properly also.  See Section 10.3.1.2 for more information.
