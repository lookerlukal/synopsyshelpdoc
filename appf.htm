SYNOPSYS User’s Manua

Appendix F

 

GLASS MODEL CALCULATIONS                                TOC

 

When varying the index of refraction, one would ideally wish the indices at all wavelengths to vary in such a way that the relative partial dispersions agree with those found in real glasses of similar index and V-number.  Since real glasses show a variety of wavelength dependencies, however, one can at best model this behavior only approximately.  SYNOPSYS uses an interpolation formula that gives the index at any wavelength as a function of Nd and Vd; although these index values do not necessarily correspond exactly to any cataloged glass, the model is nonetheless useful as a design tool since you can predict approximately the amount of secondary spectrum you will obtain when real glasses from the same region of the glass map are substituted into a design.

 

The interpolation formula in SYNOPSYS is of the following form:

Given a desired Nd, Vd, and wavelength, λ, let

 

L1 = λ**2 - λd**2

L2 = 1 / λ**2 - 1 / λd**2

L3 = 1 / λ**4 - 1 / λd**4

L4 = 1 / λ**6 - 1 / λd**6

 

where λd = 0.58756 μM.  Then

 

N = Nd + ( C1 + C2 Nd + C3 Nd**2) L1

+ ( D1 + D2 Nd + D3 Vd**Nd/Nd**2 ) L2

+ ( E1 + E2/(Nd**3 Vd) + E3/(Nd Vd ) L3

+ ( F1/Vd**2 + F2/(Nd Vd**2)) L3

+ ( F3*L4**2*Vd**4 + G1/Vd**4 + G2*Nd**Vd + G3*Nd**2/Vd**3) L4

 

 

The values of the coefficients in the above formula have been determined to be

 

     (C1 = -0.659750E-01, C2 =   0.823701E-01, C3 =  -0.266608E-01)

     (D1 = -0.176259E-01, D2 =   0.157046E-01, D3 =  -0.952593E-05)

     (E1 =  0.431113E-03, E2 =   0.615268E-01, E3 =  -0.117961E+00)

     (F1 =  0.717936E+00, F2 =   0.315466E+01, F3 =   0.239483E-19)

     (G1 = -0.478519E+02, G2 =   0.151743E-20, G3 =   0.507817E+00)

 

This formula gives representative index values over the spectral range of λ =.35 to .9μM.

 

The values returned model the catalog indices with reasonable but not perfect accuracy.  This is to be expected, since many glasses show dispersions that depart from the normal line, and the model represents an average value.  Because the fit is not perfect for all glasses, a surface that picks up a glass model will, in general, exhibit Nd and Vd that are slightly different from the values called for in the model.  The SPEC listing will show the desired parameters for the model glass (the input to the model), while the calculated properties (the output results) are shown for a surface that picks up the index – and the two therefore look slightly different in the listing.

 

The coefficients above were found with a least-squares fit to selected glasses from the Schott catalog, with errors minimized over the spectral range 0.35 to 0.9 um.  Although the model can be used outside this range, accuracy degrades as one goes beyond those values.

 

A second set of coefficients is available, for use when all wavelengths are in the UV, between 0.35 and 0.45 uM.  This set is more precise over just this region, and can help when you design microlithography lenses, which require the shorter wavelengths.  To activate this option, include the system parameter

 

ILINE

 

in the RLE or CHG file.  To remove it, use NILINE.
