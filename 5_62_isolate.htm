5.62_Isolate

The command

 

ISOLATE { FIRST / LAST / BOTH }

 

will insert a flat dummy surface before every lens element (FIRST), after every lens element (LAST) or in both places (BOTH).  This can be useful, for example, when creating a black-box lens to be sent to another party.
