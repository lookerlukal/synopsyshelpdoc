WorkSheet: Delete Element

 

The simplest way to delete a lens element is with the Worksheet dialog.  First, open the SketchPAD display with the command PAD or by clicking .  Then open the Worksheet dialog by clicking .  Make a checkpoint.  Now click on toolbar button .  The cursor changes to look like the button.  Now click on either vertex of the element you want to remove.  If you click on the Undo button , the system is restored.

 

You can also delete an element with the command DELETE or using an AI sentence.
