4.6 Print Asymmetries (ASY)                Previous Next TOC

 

Asymmetries, including tilts, decenters, and special surface data, may be output in several formats, using the commands:

 

ASY [ SN ]

ASY GLOBAL [ SN ]

ASY LOCAL

ASY JSSS JSPS

ASY EXTERNAL

 

The basic ASY command produces a listing of tilts and decenters, showing the rule by which they were input (RELATIVE, GLOBAL, LOCAL).  If a surface is deformed, coefficients G1 through G22 are printed.  Surfaces that undo a previous tilt or decenter are indicated.  This listing is automatically produced following SPEC and PRT if the lens has asymmetries.  An example ASY listing is given below.  This feature is also found in the MLL dialog.

If ASY LOCAL is entered, the table will be in the coordinates of the previous surface.  If two surface numbers are entered, only that portion of the lens is listed.

 

If the lens is in EXTERNAL mode, the ASY EXTERNAL command produces a listing similar to ASY GLOBAL, but in the external coordinate system rather than that of surface 1.
