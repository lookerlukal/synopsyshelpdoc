7.17 Coefficients of distortion (CDIST)

 

Modern technology has made it possible to project a "virtual reality" picture to someone wearing a headset consisting of a display panel and optics that project that display to the eye.  The optical designs of these displays tend to be complex, with multiple aspheric surfaces or mirrors, often decentered, and designing them presents challenges not ordinarily encountered in axially-symmetric designs.

 

There are also unique opportunities, however.  If the display has pixels in three colors, one need only correct the image at those wavelengths.  And if the tilts and decenters involved create a highly distorted field, one can correct the location of each image point with a suitable adjustment in the location of each pixel on the display.  If one compensates for distortion in each of three colors independently, then one does not have to correct lateral color either.  These possibilities can simplify the design or improve the resolution of a design that otherwise would require such correction.

 

Then the challenge is to characterize the required pixel distortion in a way that lends itself to this kind of correction.  That is the function of CDIST, which finds the coefficients that describe the distortion map.

 

Input is

 

CDIST { icol / P } [ FULL ]

 

Here is an example.  A VR headset creates an image that is well-corrected in each color, but with substantial distortion and lateral color, as shown on the MAP analysis below.

 

 

To project an undistorted image to the user, a rectangular visual field has to be mapped onto the display panel as shown above.  The engineers designing the display have to know how to distort the picture to mimic this mapping.

 

The command CDIST P FULL prints a table of coefficients and lists the polynomials.  This output tells you, for example, that to find the location on the display that will project at field point (1,1), at the corner of the visual field, you take Y = 1,  X = 1, find R**2 = (X**2 + Y**2), and evaluate the equations for the location in X and Y according to the coefficients listed. The answers are relative to the central ray point in the requested color, whose location is also listed.  Similarly, running CDIST in colors 1 and 3 yields the coefficients that apply to the red and blue pixels.  With these data, one can control the pixel mapping and project an undistorted and lateral-color-free image to the user.

 

If word three is not "FULL", the program prints a short table of coefficients, without showing the polynomials themselves.
