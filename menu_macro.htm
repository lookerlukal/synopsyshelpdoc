MACro menu commands

MACros are files containing a sequence of valid SYNOSYS commands or AI sentences.  See MACros for a general discussion.

The MACro menu offers the following commands, which enable you to manage and run MACro files:

Create a New Editor Window

Opens a new, empty window for the MACro editor. The SYNOPSYS command EE will also create a new window if none is currently open.

Open a New Editor with a MAC File

Displays the File Open Dialog where you can select a MACro file. Then opens a new MACro editor with the file displayed. The SYNOPSYS command LM filename will also load that file into a new MACro editor window.

Clear MACro

Displayed only when a MACro editor window is active, this will clear the window.
