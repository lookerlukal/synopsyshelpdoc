2.6.6 Summary of Pupil Options                                                Previous Next TOC

 

The many options available for defining the entrance pupil can be understood as a hierarchical structure where you select the degree of complexity you need. The Object Wizard (MOW) and Pupil Wizard (MPW) dialogs explain how they are used, and can automatically put them into your lens.

 

The levels can be described as follows:

 

1.

The paraxial pupil has a fixed size and location.  Use this for lenses with a fixed entrance pupil.  The size is foreshortened at off-axis field points.  If the stop is internal and the real chief ray does not go through the center because of aberrations, use the real pupil.  If you do not know the size of the paraxial pupil but want to fill the clear aperture of the stop surface, use the FILLSTOP option.

 

2.

The pupil size grows at off-axis field points so that the diameter appears constant (i.e. is not foreshortened).  Use WAP 1

 

3.

The pupil size must change so as to fill the clear aperture of the stop at all field points.  Use WAP 2.

 

4.

The pupil size must change so as to fill the stop clear aperture, and it must also be adjusted in size so that clear apertures at other surfaces are not exceeded, or because rays at the edge of the pupil would otherwise fail to trace.  Use WAP 3.

 

5.

You need the power of the WAP 3 option, but want a faster way to run it.  When all of the clear apertures are input, run the FVF feature to set up the VFIELD data.  Now the program traces rays as quickly as with the paraxial pupil, but the entrance pupil is the same as that given by  the WAP 3 search.
