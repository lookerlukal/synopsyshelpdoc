THIRD, FIFTH-ORDER ABERRATIONS

 

THIRD ORDER ABERRATIONS

 

Third-order aberrations are described by a polynomial in fractional field and aperture:

 

Y = GIHT*HBAR + SA3*YEN3 + CO3*HBAR*YEN2 +TI3*HBAR2*YEN

+ DI3*GIHT*HBAR3

 

X = SA3*XEN3 + SI3*RBAR2*XEN

 

where

 

Y =

Y-coordinate of ray at paraxial image plane

X =

X-coordinate of ray at paraxial image plane

 

GIHT =

Gaussian image height

 

HBAR =

 

fractional object height

YEN =

 

fractional Y-coordinate of ray at entrance pupil

XEN =

 

fractional X-coordinate of ray at entrance pupil

SA3 =

 

transverse third-order spherical aberration

CO3=

 

transverse third-order coma

T13 =

 

transverse third-order tangential astigmatism

D13 =

 

fractional third-order distortion (note: SYNOPSYS prints the fractional value: DI3 / GIHT)

 

SI3 =

transverse third-order sagittal astigmatism

 

The total amount of these aberrations is given by the sum of the contributions from each surface.  For AFOCAL afocal systems, the output is in radians; for FOCAL systems it is in lens units.

 

To calculate the above aberrations, the following formulas are used:

 

 

Nj =

Index of refraction following surface number j

 

Cj =

curvature of surface number j

 

CCj =

conic constant of surface j

 

DD j =

fourth-order aspheric coefficient (G3) of surface j

 

UA j =

tangent of slope angle of paraxial marginal ray, at surface j

 

YA j =

height of paraxial marginal ray at surface j

 

UB j =

tangent of slope angle of paraxial chief ray at surface j

 

YB j =

height of paraxial chief ray at surface j

 

j  = 0

denotes the object surface

 

j = f

denotes the image plane

 

 

MU =

-1 / ( 2Nf UAf )

(for FOCAL systems)

 

MU =

1 / ( 2Nf Yf - 1 )

(for AFOCAL systems)

 

I =

N0 YA1 UB0

(the paraxial invariant) At surface j:

 

 

At surface j:

 

IA = YAj cj + UAj-1

 

IB = YBjcj +UBj-1

 

SA = YA ( N j-1 / Nj ) ( N j-1 - Nj ) ( UA + IA )

 

SB = YB ( N j-1 / Nj ) ( N j-1 - Nj ) ( UB + lB )

 

T = ( N j-1 - Nj ) ( CCj cj + 8DDj ) YA4

 

and the third-order contributions are:

 

 B = SA IA2 +T                                                (SA3)

 F = SA IA lB + ( YBj / YAj ) T                                (CO3)

 C = SA IB2 + ( YBj / YAj) 2 T                                (astigmatism)

 E = SB IA lB + I ( UB2j - UB2j-1 ) + ( YBj / YAj )3 T        (distortion)

 

P1 = I2 ( N j-1 - Nj ) c j / ( N j-1 Nj )                                (Petzval contribution)

 

and these are summed as follows:

 

SA3 = MU ΣB

CO3 = 3MU ΣF

TI3 = MU Σ ( PI + 3C )

DI3 = MU Σ E

SI3 = MU Σ ( PI + C )

PETZ = MU ΣPI

 

The chromatic aberrations are given by:

 

NPj = ( N1j - Nj ) / N2j

 

NPPj = ( N1j - N2j ) / N2j

 

APj = ( NPj-1 - NPj ) N2j-1 YAj IA

 

BPj = ( NPj-1 - NPj) N2j-1 YAj lB

 

APPj = ( NPPj-1 - NPPj ) N2j-1 YAj IA

 

BPPj = ( NPPj-1 - NPPj ) N2j-1 YAj lB

 

where N1, N2, and N3 are the indices of refraction of the medium at the three wavelengths entered in the lens data.  N2 is the second (major) index.  The above quantities are summed to give:

 

PAC = 2MU ΣAP                        (primary axial color)

 

PLC = 2MU ΣBP                        (primary lateral color)

 

SAC = 2MU ΣAPP                        (secondary axial color)

 

SLC = 2MU ΣBPP                        (secondary lateral color)

 

FIFTH-ORDER ABERRATIONS

 

Fifth-order aberration theory approximates the height YA of a meridional ray at the image surface in a system with rotational symmetry as a function of the relative field HBAR and fractional aperture YEN as a power-series polynomial:

 

YA =

GIHT HBAR + SA3 YEN3 + COM3 HBAR YEN2 + TI3 HBAR2 YEN

+ DI3 HBAR3 + SA5 YEN5 + CO5 HBAR YEN4 + TOBSA HBAR2 YEN3

+ ECOMA HBAR3 YEN2 + TI5 HBAR4 YEN + DI5 HBAR5

 

 

and the X-coordinate XA of a sagittal ray by the polynomial

 

XA =

SA3 XEN3 +SI3 HBAR2 XEN + SA5 XEN5 + SOBSA HBAR2 XEN3

+ SI5 HBAR4 XEN

 

 

where XEN is the fractional entrance pupil coordinate of the skew ray.  The coefficients of these polynomials are the third and fifth-order monochromatic aberrations, given the following names:

 

SA3

 

third-order spherical aberration

CO3

third-order coma

 

TI3

third-order tangential astigmatism

 

SI3

 

third-order sagittal astigmatism

DI3

 

third-order distortion (note: SYNOPSYS prints the fractional value, DI3/GIHT

 

SA5

 

fifth-order spherical aberration

CO5

 

fifth-order coma

TI5

 

fifth-order tangential astigmatism

SI5

fifth-order sagittal astigmatism

ECOMA

 

fifth-order elliptical coma

TOBSA

 

fifth-order tangential oblique spherical aberration

SOBSA

fifth-order sagittal oblique spherical aberration

 

DI5

fifth-order distortion (note: SYNOPSYS prints the fractional value.)

 

 

Another useful polynomial gives the height YB of the chief ray at the exit pupil:

 

YB = SAP HBAR3

 

The coefficient SAP is the third-order spherical aberration of the pupil.

 

These quantities are calculated by summing the surface contributions and scaling the results by a constant, which for focal systems is given by

 

RMU = -1 / (2 NF UAF)

 

where NF is the index of refraction in image space, and UAF is the slope angle of the marginal ray in image space.  For spherical aberration of the pupil, the constant is

 

RMUB = -1 / (2 NF UBF)

 

where UB is the slope angle of the paraxial chief ray in image space. For afocal systems,

 

RMU = 1 / (2 NF YAF)

 

where YA is the linear coordinate of the marginal ray at the last surface.  The aberrations are found by summing the contributions over all surfaces:

 

SAP = RMUB ΣBB

SA3 = RMU ΣB

CO3 = 3 RMU ΣF

TI3 = RMU Σ( PI + 3C )

SI3 = RMU Σ( PI + C )

DI3 = RMU ΣE

SA5 = RMU ΣB5

CO5 = RMU ΣF5

TOBSA = RMU ΣM5

SOBSA = RMU ΣM2

ECOMA = RMU ΣN5

TI5 = RMU Σ( PI5 +5 C5 )

SI5 = RMU Σ( PI5 + C5 )

DIS = RMU ΣE5

 

The quantities to the right of the summation are surface contributions, defined as follows:

 

BB

third-order spherical aberration of the pupil

B

third-order spherical aberration

F

third-order coma

PI

third-order Petzval contribution

C

third-order astigmatism

E

third-order distortion

B5

fIfth-order spherical aberration

F5

fifth-order circular coma

M5

TOBSA contribution

N2

SOBSA contribution

N5

elliptical coma contribution

PI5

fifth-order Petzval contribution

C5

fifth-order astigmatism contribution

ES

fifth-order distortion contribution

 

The third-order Seidel aberrations are given by B, F, PI, C, and E.  The corresponding fifth-order quantities are B5, F5, PI5, C5, and E5.  The other fifth-order quantities have no third-order counterpart.

 

The calculation of these quantities is explained in the sections below.  In these equations, the paraxial quantities of interest are:

 

YA

marginal ray height on this surface

YAN

marginal ray height on next surface

YB

chief ray height on this surface

YBN

chief ray height on next surface

UA

marginal ray slope angle before surface

UAN

marginal ray slope angle after surface

UB

chief ray slope angle before surface

UBN

chief ray slope angle after surface

RIA

angle of incidence of marginal ray

RIAP

angle of refraction of marginal ray

RIB

angle of incidence of chief ray

RIBP

angle of refraction of chief ray

 

Two paraxial rays are traced through the system. The quantifies YA, YB, UA, and UB take starting values at the first surface, and then propagate as follows:

 

RIA = YA CV + UA

RIB = YB CV+UB

RIAP = V RIA

RIBP = V RIB

UAN = UA + RIA ( V - 1 )

UBN = UB +RIB ( V - 1 )

YAN = YA + T UAN

YBN = YB + T UBN

 

At the next surface, RIAP becomes RIA, and YAN becomes YA, etc.  These formulas involve the following quantifies:

 

RAD

radius of curvature of surface

CV

curvature of surface

RON

index before this surface

RNN

index after this surface

V

RON / RNN

T

axial thickness to next surface

RI

the optical invariant, equal to RNN ( YA UBN - YB UAN )

 

 

First, the third-order surface contributions are derived.

 

SA = RON ( V - 1 ) YA ( RIA + UAN )

 

SB = RON ( V - 1 ) YB ( RIB + UBN )

 

B = S RIA2

 

F = S RIA RIB

 

C = S RIB2

 

E = SB RIA RIB + RI ( V - 1 ) RIB (UBN + UB )

 

PI = ( V - 1 ) RI2 CV / RON

 

BB = SB*RIB2

 

FB = SB RIB RIA

 

CB = SB RIA2

 

EB = SA RIA RIB - RI ( V - 1 ) RIA ( UAN + UA )

 

The last three quantities are third-order aberrations of the pupil, used in calculating the fifth-order aberrations:

 

FB     third-order coma of the pupil

CB     third-order astigmatism of the pupil

EB     third-order distortion of the pupil

 

To get the fifth-order aberrations, a number of intermediate quantities are useful:

 

W = 1/8 ( RIA2 +RIAP2 +UAN2  3 UA2 )

 

X73 = 3 RIA RIAP + 2 UAN2 - 3 UA2

 

X74 = 3 RIA RIAP + 2 UAN UBN - 3 UA UB

 

X75 = 3 RIB RIBP + 2 UBN2 - 3 UB2

 

X76 = RIA ( 3 UA - UAN )

 

X77 = RIB ( 2 UA - UAN ) + RIA UB

 

X78 = RIB ( 3 UB - UBN )

 

X42 = YB2 CV RIA + YA RIB ( UBN + UB )

 

X82 = YB2 CV UA - YA RIBP ( UBN + UB )

 

The so-called S-factors are defined as follows:

 

DS1 = 3 W SA RIA

 

S2 = ¼ SA (RIB X73 + RIA X74 - UBN X76 - UAN X77 )

 

DS3 = ¼ RON ( V -1 ) [ X42 X73 + X76 X82 + YA ( RIA + UAN )*( RIA X75 - UAN X78 ) ]

 

DS4 = SA ( RIB X74 - UBN X77 )

 

S5 = ¼ RON (V -1) [ X42X74 + X77X82 + YA ( RIA + UAN )*( RIB X75 - UBN X78)]

 

DS6 = ¼ RON (V - 1) ( X42X75 + X78 X82 )

 

The following are the intrinsic fifth-order contributions.  These are added to the transferred part to get the total fifth-order surface contribution.

 

BI5 = RIA DS1

FI5 = RIB DS1 +2 RIA S2

RMI5 = 2 RIB S2+ RIA ( DS3 + DS4 )

RMI2 = RIA DS3

RNI5 = RIB ( DS3 + DS4 ) +2 RIA S5

CI5 = ½ RIB S5

PII5 = RIA DS6 - ½ RIB S5

EI5 = RIB DS6

 

The contributions from an aspheric surface may be included as well.  The sag of a rotationally symmetric conic section with even-order aspheric deformations is given by

 

Z = ( CV Y2 ) / ( 1 + ( (1 - CC + 1 ) CV2 Y2 ) ½ )+ G3 Y4 + G6 Y6 +...

 

where G3 and G6 are terms in the power-series expression of an asphere given in section 3.3.2.2.2 of the User’s Manual, and CC is the conic constant.

 

We define the following quantities:

 

C1 = 8G3 +CC CV3

 

C2 = 3 [ 4 G6 + ¼ CV2 ( CV3 CC ( CC + 2 ) - 2 C1 ) ]

 

C1B = ( RON - RNN ) C1

 

C2B = ( RON - RNN ) C2

 

The aspheric contributions to the intrinsic aberrations are then

 

BA = C1B YA4

 

FA = C1B YA3 YB

 

CA = C1B YA2 YB2

 

EA = C1B YA YB3

 

BB = C1B YB4

 

FB = C1B YB3 YA

 

CB = C1B YB2 YA2

 

EB = C1B YB YA3

 

Several quantities are needed to compute the fifth-order intrinsic aberrations:

 

ELL = ¼ [ 3 RIAP + 2 ( 2 V - 1 ) UAN ]

 

EMM = V RI / RNN

 

J0 = C2B YA - ¼ CV C1B ( 3 RIAP - 5 UAN )

 

ALP = ½ ( UAN2 + 3 RIA RIAP - 2 RIA UAN )

 

BET = UAN UBN + 3 RIA RIBP - RIA UBN - RIB UAN

 

GAM = ½ (UBN2 + 3 RIB RIBP - 2 RIB UBN )

 

LAM = ALP C1B + J0 YA

 

MU = BET C1B +2 J0 YB

 

OM = YA GAM C1B + J0 YB2

 

The first aspheric S-factors are:

 

DS1C = C1B YA4 ELL

 

S2C = ½ C1B YA3 ( 4 YB ELL + EMM )

 

DS3C = C1B YA2 YB ( 2 YB ELL + EMM )

 

D54C = 2 DS3C

 

S5C = ½ C1B YA YB2 ( 4 YB ELL +3 EMM )

 

S6C = C1B YB3 ( YB ELL + EMM )

 

The second aspheric S-factors are:

 

DS1G = YA3 LAM

 

S2G = ½ YA2 ( 2 YB LAM + YA MU )

 

DS3G = YA ( YB2 LAM + YA OM )

 

DS4G = 2 YA2 YB MU

 

S5G = ½ YA YB ( YB MU +2 OM )

 

DS6 = YB2 OM

 

The fifth-order intrinsic contributions are then given by adding the aspheric part to the spherical part:

 

BI5 = BI5 + RIA DS1C + YA DS1G

FI5 = FI5 + RIB DS1C + YB DS1G +2 (IRIA S2C + YA S2G )

MI5 = MI5 +2 (IRIB S2C + YB S2G ) + RIA DS3C + YA DS3G + RIA DS4C + YA DS4G

MI2 = MI2 + RIA DS3C + YA DS3G

NI5 = NI5 + RIB DS3C + YB DS3G + RIB DS4C + YB DS4G +2 ( RIA S5C + YA S5G )

CI5 = CIS + ½ ( RIB S5C + YB S5G)

PII5 = PII5 + RIA DS6C + YA DS6G - ½ (RIB SC + YB SG)

EI5 = EI5+ RIB DS6C + YB DS6G

 

Now the intrinsic and transferred parts must be combined. In the following section, a primed symbol indicates a summation up to the previous surface.

 

B5 = BI5 + (3 / ( 2RI ))  ( BA' FA - EB' BA )

 

F5 = FI5 + ½ RI [ 2BA' ( PI + 3CA ) + ( 3FA' -2 EB' ) FA - 3( PI'+ 3 CB' ) BA ]

 

M5 = MI5 + ½ RI [ 3BA' EA + 6( FA'- EB' ) ( PI +3CA ) + 3(3CA'+ P1' - 2(3 CB' + PI' ) ) FA - 9FB' BA ]

 

M2 = MI2 + ½ RI [ BA' EA + ( 2FA' - EB' ) ( PI + CA ) + ( PI' + 3CA' -2CB' ) FA - 3FB' BA ]

 

N5 = NI5 + ½ RI [ 9FA' EA + (2( PI' + 3CA' ) - ( PI' + 3CB’ ) ) ( PI +3CA ) + 3( EA' - 6FB' ) FA -3BB' BA ]

 

C5 = CI5 + ¼ RI [ 4(CA' + PI' ) EA - FB' PI + 2( EA' -2FB' ) CA - 2BB' FA ]

 

PI5 = PII5 + ¼ RI [ ( P1' - 2CA' ) EA + ( 4EA' - FB' ) P1 + 2 ( EA' + FB' ) CA -2 BB' FA ]

 

E5 = EI5 + ½ RI [ 3EA' EA - BB' ( PI +3 CA ) ]

 

When normalized with the constant RMU, defined above, these quantities give the total contribution of the surface.
