15.1.2.3 Automatic Output File (FILE)                                        Previous Next TOC

 

It is sometimes necessary to plot a quantity which is not in the AI vocabulary, and which therefore cannot be given in a sentence as in the above examples.  Many of the analysis features of SYNOPSYS automatically save the printed output data in a special file for use by AI for this purpose.

 

To use these data, first run the feature that gives the desired quantity, and then ask AI to show you the buffer:

 

Show buffer.

 

There you will see the data that have been saved.  Locate the number that you want to see on the plot, and set the ordinate equal to that file location.

 

For example, suppose you want the narcissus of surface 4 as a function of thickness 1. Type the NAR command; the output is:

Now ask to see the buffer:

 

SYNOPSYS AI>BUF?

 

The current FILE BUFFER contains

 1       0.00991197 NARCISSUS

 2       0.03010778 NARCISSUS

 3       0.16110719 NARCISSUS

 6       0.03810517 NARCISSUS

 7       0.05449725 NARCISSUS

 10       0.02730198 NARCISSUS

 11       0.01541717 NARCISSUS

 12       0.02796404 NARCISSUS

 13       0.00000334 NARCISSUS

 

The desired narcissus is in file location number 4.  Then type your MACro:

 

NAR

ORD = FILE 4

 

(This must be typed into a MACro editor.)  Run the MACro once, and then enter the Al sentence

 

Do macro for 1 TH = 1 to 5.

 

The abscissa of the plot will be thickness number 1, and the ordinate will be the narcissus of surface 4, as desired.  Each time the MACro is executed (by the AI loop “Do macro for..."), the NAR command calculates the narcissus of the lens, and the results are placed into the FILE buffer.  FILE number 4 is then selected as the ordinate for that data point.
