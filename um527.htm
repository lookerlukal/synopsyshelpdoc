5.27 Inserting, deleting surfaces and elements        Next Previous TOC

 

Surfaces and elements may be inserted and deleted in the WorkSheet dialog with immediate graphical feedback.  You can also use the command forms

 

INSERT { ISN / JSSS JSPS }

 

DELETE { ISN / JSSS JSPS }

DELETE { ELEMENT NB }

DELETE SURFACE ISN

DELETE ELEMENTS NB NB

 

where the data items specify the number of the surface or element(s) that you wish to insert or delete.  These commands do not automatically correct the spacing on either side of the affected surface(s), unlike the WorkSheet feature, which maintains the separation between enclosing elements as elements or surfaces are inserted or deleted.

 

The AI program can also insert elements that have desired optical characteristics, or that are selected from a table of stock vendor elements.  The Worksheet Stock lenses feature can insert selected elements drawn from the table as well.

 

Individual surfaces may also be inserted by CHG file entries

 

SN SIN

 

and deleted with

 

SN SOUT.

 

These entries can also be entered in the WorkSheet edit box, as can all RLE-file input.
