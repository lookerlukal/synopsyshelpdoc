SPS – Edit Relative Tilts and Decenters

 

This dialog will help you set up relative tilts and decenters.  With this option you can specify a single tilt direction plus a decenter in three directions for one or a group of surfaces, and the program will automatically perform a reverse decenter and tilt at the surface following the group.

 

This is the basic tilt and decenter mechanism in SYNOPSYS, although the GLOBAL and LOCAL options are sometimes more appropriate, since they can tilt in all three axes at the same time.  Although you can only specify a single tilt direction, you can stack any number of undos on a given surface, making this feature especially useful.

 

This dialog is reached via the SPS dialog.
