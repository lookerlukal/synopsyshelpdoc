MDI Diffraction-Based Image Analysis Dialog

 

The MDI dialog provides simple access to many of the most often used diffraction analysis features of SYNOPSYS.

 

Click                To

 

PUPIL

Show the shape of the wavefront exiting the system.  Select ‘Show as surface’ to see an oblique-perspective view of the wavefront, and ‘Show as contour map’ to see contour lines instead.  The command-mode version allows you to specify the exact contour interval as well.

 

PSPRD

Make a diffraction point-spread intensity plot.  You can select the orientation of the diagram with the radio buttons ‘Straight’, ‘Right view’ and ‘Left view’, and enter a desired height of the plot in the ‘Height’ box.  If you enter the number ‘0’ in this box, the program will assign a default height that depends on the peak intensity of the image, which is useful for comparing images with different aberration content – but which will draw very low peaks for a poor image.  If the multicolor option is selected, the default height is always 3.25 inches.

 

You can also create a contour plot or show the visual appearance of the diffraction pattern in colors that approximate the wavelengths analyzed.

 

VAR

Calculate the wavefront variance, standard deviation, and Strehl ratio, as well as the location relative to the chief ray in the primary color of the center of the reference sphere giving the smallest variance (and the peak of the intensity distribution).  You may enter an optional position yourself in the command mode version.

 

VFOCUS

Adjusts focus position so that the wavefront variance is minimized.

Select SYNOPSYS mode switches that apply to these features.
