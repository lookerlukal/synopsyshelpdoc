Image Tools MTF Dialog

 

Command shortcut: MIF

 

The image tools MTF dialog displays the diffraction point-spread function along with a 3-D Fourier-transform MTF.  Both displays can be rotated via a slider so that you can view them from any angle.  When the PSF has been rotated to the desired orientation, the MTF then shows the modulus of the transfer function calculated for that orientation.

 

To use this feature, enter the desired field points in Y and X (HBAR and GBAR), the color number (or select the “M” option), and the desired number of rays to be traced.  Then click “Process”.

 

This feature utilizes the wide-sample option of the PSPRD function, which requires a larger number of rays than the narrow option, in order to prevent aliasing over the larger area.  A well corrected image may need only 600 rays, while a poorer image may require more than 2000.  In any case, you can easily see if the number is too small: in that case there will be signs of the next diffraction order at the edges of the display, as shown below.

 

 Aliasing due to requesting too few rays.

 

The “Perfect” option causes all OPDs to be set to zero before the image is calculated.  The image therefore shows only the effect of apodization, vignetting, obliquity, polarization and coatings (if polarization mode is in effect).

 

The “Convert to Graphics” button creates a plot that can be saved, annotated, and printed.

 

You can also get a plot of the 2-D MTF contours with the command M2F.
