13.17 Lens Cell Editor                                                        Next Previous TOC

 

When you have finished optimizing your lens, it is time to think about the kind of lens cell you want to put it in.  SYNOPSYS has a lens-cell editor with which you can design many kinds of cells and prepare drawings of the piece parts.  This feature is not a replacement for the much larger CAD programs used by mechanical engineers, but it will produce a pretty good lens cell and a set of drawings for those without those tools.  Also, it will examine the lens geometry to determine the length of spacers and cell diameters.

 

There are restrictions on this feature.

 

1. All elements in a lens group that is to fit into a single cell must be rotationally symmetric and aligned on the same optical axis as the first element in that group.  If your system has fold mirrors between groups, that is okay–as long as the individual groups meet this requirement.

 

2. For best results, all elements in the group should have EFILE edges defined.  This is because the cell editor will let you tie the ID (inside diameter) of a cell to the outside diameter of a lens element, with optional clearance allowance.  If the element has an EFILE edge, the program simply picks up the quantity "C".  If the lens has no EFILE edges, then the program can pick up the CAO, which may be either soft or user-defined.  The problem then is that if you later make element drawings with the ELD feature, the OD (outside diameter) of the element will be larger than the CAO, because ELD in that case adds 0.05 inches to the aperture radius.  But the lens cell will not reflect this, and you must be very careful to fill in the ID diameter yourself in the cell editor.  Otherwise the element will not fit into the cell.  With EFILE edges, this problem does not come up.

 

3. You must make a lens drawing (the "D" variety, not paraxial or perspective) on the top of the PAD display, and you must assign a non-zero scale factor at that time.  The display of the cell is coupled intimately to the PAD display setup, and that is the scale that will apply to any finished drawings that you may make.  The first surface in the drawing range must be the first element that will go into the cell.

Making mechanical drawings of your cell

 

Save this cell with the name "example".  If you want to open a cell file at a later time, it is important that you first get the same lens back, and show it in PAD in the same way, before you go to the cell editor.  Otherwise you may get strange results.  The PAD display is intimately connected with the cell editor, and must be initialized the same way before you can proceed.

 

Click on the "Drawing editor …" button.  A new dialog opens up.  Initially you see the same PAD display.  Designing a cell requires two separate steps: the first you have already done; which is designing the cell itself.  The second is to design the drawings of the cell and its parts.  Everything that is to be drawn must be defined in the drawing editor.  You can select individual parts to draw, or the whole cell.  That is the default; click the "Define new" button.

 

Now you see your cell, without lens elements.  Click "Include lens elements" if you want them back again.  Click the "Color fill" checkbox, and your drawing is shown in vector form, with the lines between contiguous parts of the same color deleted.

 

 

You can send any of the drawings shown when this display is open to be converted to standard SYNOPSYS graphics.  Click the "Make graphics" button.  This picture can be printed, saved, and annotated as can all standard graphics.  You can add your company name, etc. to the drawing there -- but you don't have to draw things like dimension arrows, and such.  The drawing editor can do that for you.

 

Enter the number 1 in the box next to "Cell part No", and then click that button.  Click Define new again.  Now you see only part 1.  But that part is really a piece of a larger part.  Select the checkbox "Include all parts of the same color", and define your drawing again.  Viola!  All of the parts that are contiguous with part 1 are now shown as a unit.  Try the Color fill checkbox; you can see the part in color or in outline.

 

If you see any lines somewhere inside your part that really should be hidden, it is a sign that you did not make those pieces exactly contiguous.  Close is not good enough.  The Tie-to editor is intended to help you get things exact.  If this happens, close the drawing editor and fix up your cell.  Then come back.

 

Now we'll add some dimensions to this drawing.  Enter the number 6 in the box that says "Piece part". Select OD in the Dimension lines section, and the radio button that says "Left".  Then click the button "Add line".  If the checkbox "double lines" is selected (the default) you will now get two lines going to the left of surface 6.  Move the Line start and Line end sliders until the extra lines look the way you want.

 

 

Wait a minute!  The leftmost component of the cell was number 7, not 6.  Why did we need that number instead of 7?  Simple.  When you define a new part for the drawing editor, it makes a new cell design that only contains the requested part(s).  Since the retaining rings are not included in this new cell, the numbering is different from the original cell.  To see the renumbered parts, click the checkbox "Show cell part numbers".  (This only works if the Color fill box is also selected.)

 

There is also a checkbox "Numbers".  This shows you the item number of the individual lines that you have added to the picture.  It only works if the Color fill box is not selected.  Select these options, and you see that there are now two new item numbers displayed:

 

 

Item 59 is a dash-dot line defining the optical axis, put there by default.  Click Checkpoint, enter the number 59 in the "Edit item number" box, and then click "Delete this".  It goes away.  But you probably want it, so click "Load check" to restore your checkpoint.  The two fiducial lines you added are now item 60.

 

Let's add some dimensions.  Under "Dimension numbers and arrows", enter the number 6 in both the From and To boxes.  Select the OD button on the left, and the -OD button on the right.  Click "Add arrows", and you get your dimension arrows -- which you can hardly see since they are at the moment at the very edge of the cell.  Slide the "Arrow position" slider until they move where you want them.

 

Now just click the button "Add dimension".  The number 24.0000 shows up near the top of the cell.  Move the X-position and Y-position sliders until the number is within the gap in the arrows, and then go to item 61 in the Edit item number box.  (You can click on the spin buttons to the right of this box to increment or decrement the item number.)  Now move the "First break" and "Second break" sliders to close up this gap somewhat.

 

Now add lines giving the overall length of the cell.  Here you don't have the double line option, so you have to go to part 6, Start, and Up in the Dimension lines section.  Click Add line.  Do the same for the end of part 7, toggling the Color fill checkbox if you need to change from seeing the part numbers and the item numbers, to verify that part 7 is indeed the rightmost part.  Now add arrows from the start of 6 to the end of 7, and the dimension as well.  Move these with the sliders, as before -- and if you run to the end of the slider and want to go further, just click Update.  The sliders go back to the center.

 

 

We need to show the depth of the thread on the left end.  To do this we will use a different arrow option.  Click the radio button that shows two arrows pointing at each other. .  Under Dimension numbers and arrows, select part 6 start and part 6 end.  Click Add arrows, and then move the arrow pair to below the cell.  Adjust the lengths of both with the First break and Second break sliders.  In the Dimension lines section, enter 6 in the Piece part box, select Start, Down, and click Add line.  Adjust the line start and end, and then do the same at the end of part 6.  Click the Add dimension button, and the length of the thread is shown on the picture.  Move the number to the left of the arrows you just made.

 

The procedure should be clear now.  After adding the other lines and dimensions that are needed, and moving things around where necessary, we get the final cell drawing.

 

 

Of course this is not completely finished.  We need to click Make graphics, and then open the annotation editor to add the company name, tolerances on all dimensions, the thread specifications, and so on.  But most of the work has been done already.

 

You can save any of the drawing files, by clicking on the Save button,  but remember that if you want to come back later and edit something, you must first get the same lens, open PAD with the same display, open the cell editor and get the same cell back again, and then go to the drawing editor.  All of the components must be present for the program to work properly.

 

Recycling an old drawing

 

Okay, you've made a perfect drawing, and then realize you have to change the original cell design.  Now what?  You can change the cell and come back to the drawing editor -- but if you just click on Open or Load check, you get back all of the lines that were plotted on the old drawing, which were correct only for the old cell.  The Define new button makes a correct drawing, but all of your lines are gone!  Don't worry; you don't have to recreate everything from scratch.  You can recycle the lines and dimensions that you put on the previous drawing, and apply them to the new one.  But this is tricky, and you must do it right.

 

1.  Save the previous drawing that contains all of the user-entered data you want to recycle.

 

2.  Change the original cell design as you like.

 

3.  Open the drawing editor, select the parts you want to draw, and click Define new.  This makes new data files for the solid and outline versions of this set of parts, with no extra lines yet.

 

4.  Click the Recycle button, and select the file you saved in step 1.  All of your lines come back -- including those that point to things that you have changed.  Simply edit or delete those lines, and recreate whatever you need to bring the drawing up to date.  When done, save it again.  Whew!
