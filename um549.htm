5.49 Free-form surface utilities (FFA)                 Previous Next TOC

 

The FFA command  may also be run from the dialog MUT.

 

Recent developments in optical manufacturing have made it possible to create non-rotationally symmetric surfaces via precision milling or diamond turning, where only the active surface is machined.  This is in contrast with previous technology, which required making an off-axis asphere by coring out a section of a much larger rotationally symmetric parent.  The new process requires adjusting the position of the cutting tool in three dimensions as the part is machined.

 

To make a centered paraboloid on a precision lathe, one would move the tool along an arc that duplicates the non-circular cross section of the paraboloid, keeping it fixed at each zone.  The surface is thus rotationally symmetric but non-spherical.

 

If the goal is to use only an off-axis portion of this surface, one could cut out the desired portion after the parent was polished.  This requires manufacturing a much larger part than will actually be used, and it is very expensive.  One would like to redefine the surface so it corresponds only to that off-axis portion.  This is one important function of FFA.  The other function is to then analyze that shape in a variety of ways.

 

When you run FFA on a system where a surface is an aspheric with a decentered CAO, as shown below, the program will convert it to a Zernike surface centered at the on-axis chief-ray point.  This corresponds to the part that will actually be manufactured.  The surface following that asphere must be in global coordinates, since the selected surface will be redefined and the current TH will have little meaning afterward.  The program also deletes any EFILE data that may have been assigned to that surface, since those data do not correspond to the new clear aperture.

To make this off-axis section of a paraboloid, one orients the mirror blank so that the local surface normal is centered and aligned with the axis of the lathe, and then turns only that portion.

 

 

But now the part is no longer rotationally symmetric.  This means that, as the part is turned in the lathe, the cutting tool must move in and out with great precision, following the desired shape at each zone.  Achieving the kind of accuracy required for an optical surface becomes more and more difficult as this tool motion, which we designate the Total Indicated Runout (TIR) becomes larger.

 

(To define an off-axis mirror such as this, we assign a decentered CAO to the surface, or use the DCCR attribute, which does the same thing automatically.)

 

The command

 

 FFA SN [ RHO/0  [ SAG / RSAG / 0 /FIX [ CONTOUR/SURFACE  ]]]

 

will analyze the surface, replace the decentered portion with a new centered shape that is valid over that portion, and report some useful statistics. SN is the surface number to analyze, RHO is the normalizing radius to be applied to the final Zernike shape.  The optional SAG produces a table giving the total sag of the surface, parallel to the new surface normal at the optical center, while RSAG lists the “reduced sag”, which is the sag after all rotationally symmetric terms have been set to zero.  In other words, the latter table gives the departure from symmetry as the part is turned, and not the actual sag itself.    It is this value that determines the difficulty to be expected in machining the part.  The optional FIX will do the conversion and then exit.

 

A related command will show the fringes in double pass relative to a reference sphere.  This is intended for testing the finished mirror.  Input for this form is

 

 FFA SN [ RHO/0 ] RFRINGES [ RREF/0 ] [ NLNS ]

 

If RHO is zero, the value is set to 1.0.

 

To analyze the surface, the program first fits the shape to a set of Zernike polynomials, where the parameters are the dimensions in (X,Y,Z) in the lathe coordinate system.  The polynomials thus describe the surface in those coordinates, and those are the coefficients listed with the output and assigned to the new shape by FFA.

 

In RFRINGES mode, the program then finds the sag difference between that polynomial and a reference sphere at an array of points whose grid number is entered as NLNS.  If RREF is not entered or is zero, the program will estimate a useful radius by examining the computed sag at the top and bottom of the clear aperture.  This difference is then multiplied by two (for double pass) and divided by 0.6328 um, the HeNe laser line most often used in interferometric testing.  The results are plotted as a greyscale of 255 possible values from black to white.

 

 Note that while this feature does not assume rotational symmetry, it does assume bilateral symmetry.  If your design involves beta tilts or X-decenters, the results may not be meaningful.

 

In all the other modes, the symmetric Zernike terms, numbers 3, 8, 15, 24, 35, and 36 are then set to zero, and the program finds the highest and lowest point on the surface using just the remaining terms.  Those give the motion of the cutting tool parallel to the lathe axis, at a given zone, as the part turns, and it is this motion that is critical for obtaining optical precision.

From this listing we see that the optical center is at (0, 20) inches from the vertex axis of the paraboloid, the surface normal is at an angle of 4.7636 degrees with respect to that axis, and the program has calculated a value for RHO of 9.9985 inches.

 

Next we see the Zernike coefficients themselves, listed as a table.  To see the individual terms identified, one can ask for an ASY listing of that surface.  If one were making this part with a numerically controlled lathe, those coefficients could be entered into the controller for the machine to fully describe the shape.

 

In this RSAG analysis, we next see the reduced sag for a set of 69 data points distributed over the surface, centered at the optical center.  At the center, where X and Y are both zero, the sag is zero, and it can take both positive and negative values elsewhere.  Where the reduced sag is negative, the tool moves in the (-Z) direction, in this case away from the mirror blank.

 

It is wise to examine a sag table for the new surface, to be sure of the proper sign, since the axis direction can be changed by FFA.

 

The final lines show some statistics from the analysis: the highest and lowest RSAG values, the total excursion, or TIR, and the RMS excursion.  Observe that those figures refer only to the non-symmetric terms of the polynomial, and not the total surface shape, which is summarized in the line below that.  Note that the terms highest and lowest refer to the most positive and most negative sags found, not necessarily the highest and lowest point on the surface (a high point on the surface may well have a negative value of Z, for example).

 

For this example, the TIR value is 0.001815 inches, which is a fairly small excursion.  The total TIR, considering all Zernike terms, came out to be 0.20533 inches.  This is the actual total sag depth of the part measured with respect to a plane perpendicular to the lathe axis.

 

The optional CONTOUR directive in word 5 of the command produces a contour plot of the surface, where either the full shape is shown (with SAG) or just the non-symmetric terms (with RSAG).  The optional SURFACE shows the surface as an oblique-perspective plot.  The examples shown below were created with the commands

 

FFA 2 0 RSAG CONTOUR

FFA 2 0 RSAG SURFACE

 

 

 

It is also possible to control most of the statistics during optimization, as described in Section 10.3.3.  For this purpose one uses the construction parameter aberrations FFHIGH, FFLOW, FFTIR, FFRMS, FFALPHA,  and FFBETA.  When these are used, however, the program does not alter the system as described above.  That would make all global targets invalid on the target surface.  But it reports the statistics as though the change were made, which is a goal in optimization.
