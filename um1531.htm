15.3.1 Initializing the example file                                                Previous Next TOC

 

The example lenses used to educate XSYS must be SAVED in RLE format before you can create the XSYS example file.  Then you type INIT at the XSYS> prompt.  The program asks for the name of a lens file, then processes the file and adds it to the database.  You can add more lenses, and when all of the examples have been processed, enter END at the prompt.

 

If the list is a long one, you can make a MACro giving the filenames.  Suppose that three lenses have been saved, with names EX1, LENS2, and XXX3.  The MACro would then read

 

XSYS

INIT

EX1

LENS2

XXX3

END.

 

XSYS then studies the lenses and creates two disk files, one for the lens input data and one with the lens analysis data resulting from the study.

 

It is also possible to add a lens to an existing example file.  In this case you enter ADD, instead of INIT. XSYS prompts for the location in the example file where the new lens should go, or whether it should add the lens at the end of the file.

 

A list of the current lenses in the example file is produced by the command LIST, which prints the ID of each lens.
