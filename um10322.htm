10.3.2.2 Third- and fifth-order aberrations                                Next Previous TOC

 

The values of the third- and fifth-order aberrations at the final surface may be defined as aberrations to be controlled.  The input is

 

{ A / S / MUL / DIV } name

 

where name is one of the following (see sections 6.6 and 6.7):

 

SA3 CO3 TI3 SI3 PETZ DI3 PAC PLC SAC SLC

SA5 CO5 TI5 SI5 ECOMA PZ5 DI5 TOBSA SOBSA

SAP TIP COP SIP ASP PZP DIP

 

Another set of aberrations is available to help to reduce the sensitivity of the lens to manufacturing errors.  These are

 

SAT COT ACD ACT ECD ECT ESA ECO

 

A description of these aberrations with instructions in how they may be used is given in Section 10.13.
