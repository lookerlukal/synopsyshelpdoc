8.6.5 Diffraction Pupil Function (DPF)                         Next Previous TOC

 

The vignetted exit pupil normally calculated by the diffraction analysis programs may be stored for use by DIFF.  The input is

 

DPF { ICOL / P } HBAR NRYS 0 GBAR

 

See section 9.4 for a description of this feature.
