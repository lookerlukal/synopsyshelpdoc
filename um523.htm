5.23 Insert Real Glass (IRG)                        Next Previous TOC

 

SYNOPSYS can substitute a real glass type for any other, or for a glass model. ARGLASS can replace any or all of the model glasses subject to certain selection criteria, while IRG operates on a single surface at a time.

 

The command

 

IRG SN {S / O / H / F / U / C / G / R / M / P / N } [ AUTO ]

[ glassname ]

 

will replace the current glass at surface SN with the named glass.  The program then submits an optimization job that varies curvatures as necessary in order to maintain the focal length of the lens and the ratio of powers of the two surfaces.

 

The catalog from which the glass is taken is given by one of the following letters:

 

S

Schott

O

Ohara

H

Hoya

F

Corning France

U

Unusual

C

Custom

G

Guangming

R

LZOS

M

Sumita

P

Private catalog

N

Nikon glass

 

The AUTO option causes a search through the named glass catalog for the closest glass match.  In this case the glassname on the following line is not allowed.

 

The feature GSEARCH can substitute names glasses for selected elements and reoptimize automatically.

 

The glass-table display in PAD can show an entire catalog, and you can insert a selected glass with just a few mouse clicks.  This is much easier to use than the IRG command.  Really.

 

You may run your IRG command file via the MACro editor, but do not run it as the alternate MACro.  IRG creates its own alternate MACro to do the optimization, so this must be vacant.
