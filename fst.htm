5.51 Find Superachromat Triplets (FST)                Next Previous TOC

 

 

There are four categories of color correction that can be applied to a lens design.  Uncorrected, achromatic, apochromatic, and superachromatic.  An achromatic lens is corrected so that two different wavelengths focus in the same place.  The apochromat focuses three, and the superachromat keeps four wavelengths in focus.  That is the highest degree of color correction that has yet been developed.

 

Finding glass types suitable for this kind of correction is not difficult, but one must employ just the right combination or the correction will not work.  Max Herzberger published a paper in 1963 that describes a simple geometric rule that can be used to select these combinations.  Briefly stated, if one makes a plot of the quantities P* vs. P**, one gets a spread of values for all the glasses in a catalog.  Then selecting three types that lie on a straight line yields a combination whose correction can approach the superachromat level.

 

The on-screen glassmap feature (MGT) can show this display, and you can position a straight line anywhere on the map by <Ctrl>-clicking where you want the lower end and <Shift>-clicking where you want the upper end.  Then select any other glass that is close to the same line, and you have a potential triplet.  For practical reasons, it is best if the line is a long one, and you want the dispersions of the end points to differ by a wide amount (or else the curves of the elements may become very steep.)  This approach lets you see exactly what is going on, and you can restrict your choice to glass type with low cost, chemical stability, and so on, by graphing those properties onto the P*-P** diagram.

 

There is also a command form will do much the same thing automatically.  It has the input

 

FST

[ PRICE PTIMES ]

[ BUBBLE BLIMIT ]

[ STAIN SLIMIT ]

[ACID ALIMIT ]

[ ALKALI ALLIMIT ]

[ HUMIDITY HLIMIT ]

[ PREFERRED ]

[ SAFE ]

 

CATALOG { S / O / H / F / G / R / M / P / N )

CATALOG { S / O / H / F / G / R / M / P / N )

...

GO

 

This command allows one to search more than a single catalog -- which the glassmap display cannot do.  The catalogs are specified by

 

S        Schott

O        Ohara

H        Hoya

F        Corning France

G        Guangming

R        Russian LZOS

N        Sumita

P   Private catalog

N   Nikon

 

The program will search all suitable combinations of glasses in the requested tables (up to a maximum of 1000 types) and then report the 10 best combinations it found.

 

We have to define "suitable".  Because a practical design requires a large dispersion difference between the top and bottom glasses at the ends of the straight line, the program only tests combinations where

 

1.  The Abbe number of the top glass is in the upper half of those in the catalog.

2.  The bottom glass has an Abbe number in the lower half

 

In addition, the glass properties must satisfy all entered requirements, such as relative price, acid sensitivity, and so on.  (See ARGLASS for a definition of those properties.)  Note that most glass catalogs do not contain price information, so they will not be filtered on that issue.  (The easiest way to check is to open the glass map MGT, select a table, and ask to see the list.  If the price is given as zero, that glass has no price data.)

 

The score of each triplet tested is based on a combination of the difference in Abbe numbers between the three and the distance of the middle glass from the straight line.  More widely separated is better, and closer to the line improves the score.

 

Here is an example:  We ran the following input:

 

FST

PREFERRED

CAT O

CAT S

CAT M

GO

 

and got the following combinations:

 

--- FST

--- PREFERRED

--- CAT O

--- CAT S

--- CAT M

--- GO
