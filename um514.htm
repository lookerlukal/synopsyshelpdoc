5.14 Combine Two Lenses (COMBINE)        Next Previous TOC

 

Two separate lenses may be combined with the command

 

COMBINE [ SN ]

filename

 

where filename is the name of a lens file previously saved with the SAVE command.  The lens currently in memory will constitute the front part of the combination (optionally truncated at surface SN -1), and the lens from the saved file will be the rear part, beginning at surface SN (or at the last surface of the first section, if SN is not entered).

 

The object geometry is taken from the first part, and the image geometry from the second; these and the aperture stop specification should be adjusted manually, following the COMBINE command, if necessary.

 

There is another to combine lenses, using the lens editor, which gives you more control over which surfaces to process.  Place both lenses in separate ACONs, and then activate the lens that is go on the output end of the combination.  Click the Lens Edit button , and the data for that lens are displayed in an editor.  Select with the mouse the portion that you want to put into the other lens file, and press Ctrl+C.  Those data are now on the Windows clipboard.

 

Now activate the ACON for the first part, and again send the data to the LE editor.  Here it gets a little tricky.  Find the spot where you want the other lens to be inserted, and note the surface number.  Delete all surfaces beyond that section, and type

 

 ADD SNA

 

in their place.  Then paste the data from the other lens at the end of the file and execute it.  The value of SNA will be added to all surface numbers that appear in the input file beyond that point, which will renumber the surfaces you have just added.   Usually, SNA will equal SN – 1.  For example, if the first portion goes from surfaces 1 to 6, as specified by

 

 COMBINE 7,

 

you want the added surfaces to start with number 7.  If the added section starts with surface 1 in that lens, the correct SNA will be 6, so the added surfaces will begin from 7 on up.

 

When this file runs, the new data will be renumbered starting with the requested SN, and will become part of the combined lens.
