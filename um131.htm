13.1 Help Files                                                                Previous Next TOC

 

In keeping with industry trends, SYNOPSYS no longer issues hardcopy manuals.  The Tutorial Manual as well as this User’s Manual are now both available as on-line Help files, giving you immediate access to complete instructions.  Users who prefer a printed document can make their own: the Tutorial Manual is meant to be printed on your system printer.  Beginning users are encouraged to make a hardcopy printout and then follow along with the text as various features of the code are exercised.  Most users will not need to print the entire User’s Manual, although individual sections can be printed for easy reference.

 

There are two advantages of this approach to documentation: first, each update automatically contains the latest documentation, and second, any reference in the online manuals is always accompanied with a link to the referenced section.  You should have no difficulty in finding any topic you want, virtually instantaneously, without wrestling with the index to a book.

 

You can activate this online documentation in many ways.  Probably the easiest way is just to type in the Command Window

 

 HELP topic

 

where topic is the name of a command or a word describing your question.  Thus, to read about the SPT command, you would type HELP SPT.  That section then opens.

And if you are entering a command yourself, either in the Command Window or in a MACro editor, just type the command, and then look down at the tray.  The format is display for you.  Read about this very friendly feature in Section 13.20.

 

Clicking on the menu item Help|Help Topics gives you a choice of the contents menu or the index.  You can also press the F1 key any time a dialog window is open to see the Help section for that feature.  (Some windows will show the Help section for the Command Window, because that window is still accepting keyboard input, even though you may be working in the other window.)  Or you can type <shift>F1 or click on the button , and then click on any other button, window, or menu item to see what it does.

 

Many of the dialogs in SYNOPSYS have a button labeled Help, and if you click there you will go immediately to the User’s Manual section for that topic.  In some cases the button is very small with only the character “?”.  Yet more options are explained in the next section.
