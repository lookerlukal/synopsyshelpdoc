8.5.4 Through-Focus Geometric MTF (TFG)                 Next Previous TOC

 

The program TFG will calculate the GMTF at up to five focus positions in up to ten wavelengths, obtaining the weighted MTF at each position.  The rays used for this calculation are traced only once, and the extra-focal images are determined by extending the ray paths forward or backward by the requested amount until they intercept the shifted focal plane. The results can be plotted on a composite graph showing all of the MTF curves. TFG may be run interactively from the dialog MTG.  The command-mode input for this program is:

 

[ FCO FREQ ]

 

TFG { ICOL / M / P } HBAR NRYS 0 GBAR [ P ]

 

DF { 1 / 3 / 5 } DELTAF

 

where DELTAF is the focus shift which will be applied to 1, 3, or 5 separate evaluations, according to the DF input.

 

A single color will be traced if ICOL is a color number or “P”, while the letter “M” will generate a multicolor GMTF.  The images may contain up to 20,000 rays each, and the defocus motion is along the local Z axis of the image surface.  All apertures, vignetting, and feathering are taken into account, and use of the FCO input is recommended.  If a plot is desired, the optional [ P ] should be entered.
