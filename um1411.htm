14.1.1 Generating the perturbation table                                        Previous Next TOC

 

One method of describing the change in shape of a surface is with a polynomial where you enter a set of coefficients.  The polynomial used by TOSP has the form

 

DZ = Al + A2R**2 + A3R**4 + A4R**6 + A5X + A6Y + A7X**2 + A8Y**2 + A9XY

 

where       R**2 = X**2 + Y**2.

 

The data are entered following the TOSP command in the format

 

DATA

ISN JZN NPS CAP

A1 A2 A3 A4 A5 A6 A7 A8 A9

ISN …

END

 

or

DATA

ISN JZN NPS CAP

EXACT

ΔZ1 ΔZ2 ΔZ3 … ΔZ10

ΔZ11 ΔZ12 … ΔZ20

…

ISN …

END

 

where ISN is the surface to be perturbed, JZN equals 1 for surface deformations and 2 for index changes (enter the deformations first, for surfaces with both types of perturbations), NPS is the number of rows or columns into which the surface is partitioned, and CAP is the semi-aperture over which the data apply.

 

The value of CAP should be at least as large as the maximum aperture required by the rayset; an error message will be displayed otherwise.  The data are interpolated linearly between nodes, so it is wise to generate a fine enough mesh to give a smooth wavefront. (The maximum value for NPS is 25 for deformations and 40 for index changes.)

 

To illustrate the use of this feature, suppose that a lens is subject to a thermal gradient from top to bottom, resulting in a thickness change of .000025 inches at the top and an index change of .00148.  Let these changes occur in a linear fashion from top to bottom, with the center point unchanged.  Then the data could read

 

IMAGE

TPF 2 1 300

TOSP

DATA

1 1 2 1.25

0 0 0 0 0 -.00002

1 2 2 1.25

0·0 0 0 0 .001184

END

GO

IMAGE

GPP ...

 

Here we assume that the clear aperture of surface 1, which is to receive the perturbations, is 1.25 inches.  The entered value of A6 for the surface deformations will give a linear change in Z of -.000025 inches at an aperture of 1.25 (since the coefficient A6 is multiplied by Y in the polynomial).  The minus sign indicates a negative Z at a positive Y, which gives a thicker lens at the top since surface 1 is the left side of the element.  The index perturbation follows, and gives an index change of +.00148 at the same aperture.  Note that since these perturbations are linear in this case that only two nodes are adequate – and a linear interpolation will give the exact answer.
