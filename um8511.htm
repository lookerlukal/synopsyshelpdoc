8.5.11 Foucault Knife-edge Tool (MFK)                                         Next Previous TOC

 

The command MFK will open a special window that shows the pattern at the pupil one would get in a classic Foucault knife-edge test.  You can select the number of rays to trace and the knife-edge scan direction, in X or Y.  The program traces the requested number of rays in the primary color and displays the intensity distribution on the screen.  You can scan the knife edge in X or Y with the horizontal slider and alter the focus position along the Z axis with the vertical slider.  In this way you can simulate the test you would use to verify that a given zone focuses at the correct Z-axis position.

 

Although this is a geometric analysis, it makes allowance for the effect of diffraction on the results.  By pure geometry one would expect that a given ray would either pass the knife edge or would be stopped there, and the display would be all black and white.  But the program calculates the estimated radius of the Airy diffraction disk, based on the current FNUM and wavelength, and then convolves the ray intercept with a knife-edge trace of a perfect diffraction pattern.  If a ray passes well clear of the edge, it gets a grayscale value of 1.0.  if it hits well within the knife, the value is 0.0.  If it passes (or is blocked) close to the edge, it is given a weight corresponding to the perfect knife-edge trace value at that distance.  You can print the display or convert it into a bitmap, which can be loaded into a Graphics Window, where it may be annotated and then printed.

 

An example MFK display is shown below.  This system is an F/8 paraboloid of 16 inches diameter.  The RLE file is

 

RLE

ID PARABOLOIDAL MIRROR

OBA 256 .5 8

UNI INCH

WAVL CDF

1 RD -256 CC -1 YMT

1 REFLECTOR

2

END

 

MFK

 

Notice that in this example, since the Foucault test must be performed at the center of curvature, not the infinite-object focus, that the F/number as tested is actually F/16 instead of F/8.  You can of course use this feature for the other focus, but you must then test the actual mirror with a collimated input beam in order to see the same thing.

 

The picture below shows the pattern when the knife is at the axial position, but shifted in Z by –0.16 inches, which is the focus point of the 0.8 zone.  The trace at the bottom of the display helps you to identify which zone is currently in focus.

 

 

       Ronchi Test

 

A related analysis is the Ronchi test, where instead of a single knife edge one uses a mask with a set of fine parallel lines.  This test can also be emulated by MFK.  Simply enter the desired line density (always in units of lines/inch), click Update, and then use the sliders as above.
