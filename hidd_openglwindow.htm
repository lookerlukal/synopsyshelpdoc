RSOLID Dialog Window

 

When your RSOLID picture is displayed in this window, you have the following options:

 

1.  Click and drag with the left mouse button in the window to rotate the picture.

 

2.  Click and drag with the right mouse button in the window to pan the picture.

 

3.  <Ctrl>+ right-click to enlarge the picture.

 

4.  <Shift>+<Ctrl>+ right-click to reduce the picture.

 

5.  Zoom in and out with the mouse wheel.

 

6.  Hold down the <spacebar>; this will rotate the picture slowly about the rendered Y axis; use <Shift>+<spacebar> to rotate about the rendered X axis.  If the RSOLID command specified angles of (0,0), those axes are aligned with the Y and Z axes of the first surface displayed.

 

A toolbar is displayed above the window.

 

 

The leftmost two buttons change the background color to white or black.

 

The four arrow keys quickly rotate by 90 degrees in the indicated direction.

 

The camera icon creates a bitmap copy of the display.  This may be pasted into any graphics program that can accept such things – or into a SYNOPSYS Graphics Window, where you can annotate and print it.  The second camera icon (rotated 90 degrees) produces a bitmap that is rotated (by turning on switch 81), which may better fill a page when pasted.  If you elect to paste the picture rotated 90 degrees and wish to add text annotation, be sure to select that option in the text editor as well.

 

There are two printer icons, both of which send the picture to the current printer.  The top one turns off switch 81 first, so the picture is oriented upright on a vertical plot page.  The lower print button (showing the printer on its side) turns switch 81 on first, and the picture is rotated by 90 degrees, thus filling the page more fully.

 

The “+” and “-“ buttons enlarge and shrink the display.

 

The home button restores the display to the size and orientation of the original.

 

The print options require Windows 7 or higher.  We have seen the printer drivers with earlier version crash, so users of those versions are advised to instead use the bitmap buttons and then paste into a new graphics window.  That window can then be printed.
