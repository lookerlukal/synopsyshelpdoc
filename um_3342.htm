3.3.4.2.  GRIN ENM

 

ENM can model environmental effects on surfaces and lenses, as explained in Section 14.2.  To model a change of index, it defines the current lens as a special type of gradient-index material called GRIN ENM.  This model uses the same polynomial representation of the index as that used for surface deformations by ENM:

 

INDEX = (nominal index at current color) + A2R**2 + A3R**4 + A4R**6 + A5X + A6Y + A7X**2 + A8Y**2 + A9*X*Y

where R**2 = X**2 + Y**2, R4 = R**4, and R6 = R**6.

With these data one can model the case where one side of a lens is hotter than the other, or where the rim is at a different temperature than the center, for example.  Input for this kind of GRIN is as follows:

SN GRIN ENM DT N1 N2 N3

A2 A3 A4 A5

A6 A7 A8 A9

DT is the integration step length, and N1 to N3 are the nominal index values at the axis.

Note that, with all GRIN materials in SYNOPSYS, the system must have exactly one or three wavelengths defined.  Once a material has been declared in this way, the data apply only to the current wavelengths.  To change wavelengths, you have to go back to the system before it was modified by ENM, change the wavelengths there, and then run ENM again.

Example:

Suppose the element at surface 6 of the relay telescope is exposed to sunlight on the top, so the index is slightly higher there.  The input would be

FET 4

CHG

NTOP

END

ENM

DATA

6 2

0 0 0 0 

0.002 0 0 0

 

END
