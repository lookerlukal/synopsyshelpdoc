5.20 Custom Glass Table                        Next Previous TOC

 

Index data for materials not in the standard glass catalogs may be input to SYNOPSYS using one of several formats described in section 3.3.4.  Glasses entered with the formats GLASS, GDF, and GFlT may then be entered into a custom glass table.

 

To do so, the glass must currently be present in a lens, and must be assigned a desired name with the GID format.  Then it is put into the glass table with the command

 

 AGLASS SN                (Add GLASS)

 

where SN is the surface number with the desired glass.  If the current directory does not already contain a custom table, a new one will automatically be created and the desired glass entered.  Thereafter, the new glass may be accessed simply by giving the glass table name "C" in the RLE or CHG file, along with the assigned name on the next line.

 

If the material has been assigned thermal-index properties (DNDT) or a thermal expansion coefficient (CTE), those data are saved in the catalog as well.

 

The contents of the custom glass table may be printed with the command PCGLASS, and a glass may be deleted with the command DGLASS followed by the GID name on the next line.

 

Private catalog

 

SYNOPSYS can also create a Private glass catalog, which is not the same as the custom table described above.  This is intended for users who have chosen a favorite set of glass types and want to see only those glasses in the glass map display and make them available to programs such as IRG, VNUM, etc.

To create a private glass table, use the following syntax:

 

PCAT INIT

 

PCAT ADD

[ S / O / H / F / G / R / M ] glassname

[ S / O / H / F / G / R / M ] glassname

[ S / O / H / F / G / R / M ] glassname

...

 

END

 

The table is created in the current directory and is only available to users while in that directory.  Note that the Unusual material catalog (U) and custom catalog (C) are not permitted in this list.  They are formatted differently and will not work in this context.

 

To use the private table, specify the letter "P" for the table name.

 

For example, to create a private catalog containing three glass types, we enter

 

PCAT INIT

PCAT ADD

S N-LAK10

O S-NBH5

S N-SK16

END

 

Now we can apply these glasses to our lens with input such as

 

CHG

3 GTB P

S-NBH5

END

 

There is no new information in the private catalog, and the only reason you would ever use it is so you can avoid seeing all the glasses in the standard table when you want to use just a selected group of them.
