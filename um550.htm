5.50 Find Glass Transmission (FGT)                Next Previous TOC

 

Several features can evaluate the transmission of a lens.  Most of them, such as RTRANS, can be run from the dialog MTR.  Most of those features evaluate the entire lens.

 

It is sometimes useful to know the transmission coefficient of a single element; this is the function of FGT.  Input is

 

 FGT ICOL TH SN

 

In this command, ICOL is the color number, TH is the glass thickness whose transmission you wish to evaluate, and SN is the surface assigned the glass type to be treated.  If TH is zero, the program will use the current thickness of the glass at surface SN.

 

 

In the lens above, we wish to know the transmission of element 1.  The result, shown below, is 0.98718.

 

SYNOPSYS AI>FGT P 0 1

 

TRANSMISSION OF GLASS

ON SURFACE            1

GLASS TYPE            BSM51Y

IN COLOR              2

THROUGH THICKNESS    14.075480

EQUALS                0.987180

 

A novel use of this feature is when you want to control the overall transmission of a lens system.  Suppose the three elements show above are made of two glass types: elements 1 and 3 are glass BSM51Y, and element 2 is PBL25Y.  First we need the transmission coefficients for a thickness of one mm for each glass.  We run FGT twice and transfer the coefficients to variables Z1 and Z2.

 

FGT P 1 1

Z1 = FILE 1

 

FGT P 1 3

Z2 = FILE 1

 

Now we can create an entry for the AANT file that will attempt to control the total transmission of those three elements.

 

AANT

...

 

M .99 1 COMPOSITE

CD1 TH 1

CD2 TH 3

CD3 TH 5

= Z1**(CD1 + CD3) * Z2**CD2

 

...

END

 

This entry employs the COMPOSITE form of aberration definition, using the fact that the total transmission equals the coefficient just found raised to a power equal to the sum of the thicknesses of elements using that glass.  When the lens is optimized with this additional requirement, we would expect transmission to be improved, and of course the elements will become thinner.  But the program will reduce the thicknesses preferentially where it will do the most good.
