13.23 Annotation Strings (ASn)                                        Previous TOC

 

The annotation editor, which opens when you click the button , lets you enter a text string that will show up on your drawing at the place where you clicked.  If you want to add the same text string to many pictures, however, it is more convenient to define the string only once instead of typing it in all over again each time.  This is the function of the Annotation String.  The commands are:

 

 ASn “any line of text, up to 90 characters long”.

 

Here, ASn is a command of the form AS1 through AS9, and the text string must be contained within quote marks (and must not contain any quote marks of its own).  In this way you may define up to nine character strings to be used later.

 

Once these lines are defined, you click the appropriate button on the annotation editor dialog:

 

 

and the program will insert the string you previously defined into the text editor pane.
