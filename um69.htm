6.9 Strain Evaluation (STRAIN)                Previous TOC

 

The STRAIN evaluation provides information that may be helpful in deciding how to modify a lens in order to improve its performance.  The idea is that, if a given element contributes a large amount of some 3rd or 5th- order aberration, it will be difficult for the other elements to compensate exactly – while, if the individual contribution is small, the compensation need not be so exact.  This feature evaluates the aberrations of each element in the lens and prepares a table giving the values of the aberrations, the power of the element, and the sum of the squares of the aberrations listed, identified as the “STRAIN”.  It can optionally produce a plot of the results.  Input is

 

 STRAIN [ P ].

 

If your goal is to improve the lens and you can allow an additional element, from this table you can identify which element has the largest strain and should be split.  Doing so allows the power of that element to be shared by two elements, each with presumably smaller aberrations.  Conversely, if the lens performance is adequate and your goal is to attempt to simplify the design by removing an element, then you should examine the table to see which element shows the smallest strain.  If that element also has relative little power, it is a good bet you can remove it from the lens and reoptimize.  The AED feature is a very handy way to alter the lens so the element can be deleted without destroying the image correction.

 

An element with a large strain is also an element that will have tight tolerances.  This table can identify such elements, so you will know the problem areas before you spend time calculating a budget with BTOL.  You can examine the individual aberrations of such elements, in the listing, and perhaps improve things by controlling the value of the largest contributors to the strain.  The strain of an element can be controlled during optimization with the STRAIN aberration, and individual contributions can be controlled as well, as the following example shows.

Now the lens has entered a different solution region, and the value of SA3 is even lower than we requested.  The largest strain is now 4.37, while before it was 60.3.  We would expect looser tolerances on this version of the lens.

 

The optional “P” in the command line produces a plot, which lets you quickly identify opportunities.  Here is an example, which is part of a microscope objective:

 

 

 

Looking at the lens drawing, one would never suspect that the element at surface 9 is not doing much work and can be deleted.  But the STRAIN plot shows that the power and strain are both very low on that element.  So we put an AED command at the top of the optimization MACro and reoptimized.  This case required a great many passes, since the output end has a high NA and is very sensitive to construction changes – but eventually the merit function approached its previous value and we were able to delete that element.
