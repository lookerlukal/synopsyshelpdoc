8.5.6 RMS Spot Size                                         Next Previous TOC

 

The RMS spot size at the current focus position may be calculated with the input

 

{ RMS / XRMS / YRMS } { ICOL / M / P } HBAR NRYS 0 GBAR [ XIP YIP ]

 

This command prints the RMS size measured from the principal ray in the requested color, unless the multicolor “M” option is selected, in which case it is measured from the chief-ray point in the primary color*.  It also lists the spot variance, standard deviation measured from the image centroid, and the location of the latter from the chief-ray point. XRMS and YRMS consider only one component of the image.  A copy of the results is put into the Al file buffer (see 15.1.2.3).

 

You may evaluate the image relative to a point that is offset from the location described above, by entering the X and Y offset dimensions in words 7 and 8 of the command.

 

RMS may be run interactively from the dialog MGI.

 

 

A plotted version is also available:  The input

 

 FRMS { ICOL / M / P } { 0 / P } NRYS 0 GBAR

 

will calculate how the RMS spot size varies over the field.  The optional “P” in word 3 of the command requests a plot.  This form always prints the RMS at 51 field positions, at the entered GBAR, if any.

 

FRMS may be run interactively from the dialog MGI.
