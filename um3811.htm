3.8.1.1 Examples of tilt and decenter input                Previous Next TOC

 

Suppose the triplet shown on the left has already been entered in an RLE file, and now the center element is to be decentered.

 

 

The input

 

CHG

3 DEC 0 -.005 0 2

END

 

would cause surfaces 3 and 4 to be decentered downward by .005 lens units as shown (exaggerated) on the right.

 

If the decenter data had read

 

3 DEC 0 0 .005 2

 

the central element would be shifted .005 lens units to the right instead of downward.  Note that if the lens has a paraxial thickness solve on surface 6, the result of the solve – and the position of surface 7 – will be unaffected by the Z-decenter since asymmetries are ignored by PXT.  The paraxial solve will therefore not yield the new paraxial focus in this case.  For zoom lenses, where the effect of Z-axis motion must be taken into account, the ZFILE description of element motion is to be preferred over Z-decenters.  It has its own format that does not involve Z-decenters, and the paraxial raytrace honors the zoom settings.

 

If the third element were to be tilted, as shown below, the input would read

 

5 AT -10 0 2

 

 

to give a -10-degree tilt about the vertex of surface 5.  Note that surface 7 was unaffected by the above instructions since only two surfaces were tilted.  If you want surface 7 to follow along the same axis, as shown on the right, the input would read

 

5 AT -10 0 100

 

where the entry "100', is more than the total number of surfaces.

 

A more complicated example is shown below.  In this example a folding mirror is to be placed between two identical simple lenses.  For this example we will use the remote tilt option.  The RLE file for such a case might look as follows:

 

RLE

ID SPLIT LENS W/ MIRROR

WAVL CDF

OBA 1.3 .25 .5 0

1 RD 1.89008 CC -23.0783 TH .35 GTB S

BK7

 

 

2 UMC 0 TH 1.5 AIR

the UMC solve collimates the light

 

3 CV 0 TH -1.12 AIR

 

3 REFLECTOR

 

4 PCV 2 PTH -1 PIN 1

surface 4 will have the same curvature as surface 2, the same thickness (with a sign change) as surface 1.

 

5 PCV 1 YMT 0 TH -1.3 AIR

surface 5 will have the same curvature as surface 1.

6 CV 0 TH 0 AIR

 

 

3 AT 45 0 100

this tilts surface 3

 

4 AT 45 1.12 100

this tilts surface 4 about the vertex of surface 3 (1.12 inches to the right).

END

 

 

 

This example shows how the AXIS specification may be used to locate a surface with respect to a distant rotation point.  A simpler way is to place two surfaces at the mirror; the first is the mirror itself, and the second is a dummy surface given another 45-degree tilt to align the axis with the reflected center ray.  Although this method requires an extra surface, the geometry is simpler to understand, and some features of SYNOPSYS, such as the REVERSE and UNFOLD commands work with this method, rather than the remote axis input described above.  Also, the Worksheet program can insert and remove fold mirrors or change the direction of the fold, etc. – and you can do it with just a few mouse clicks.  If you want to insert a fold mirror into your system, by all means have a look at the WorkSheet.
