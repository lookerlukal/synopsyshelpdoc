12.4 BTOL Example                                                        Next Previous TOC

 

The simplest form of BTOL input would simply be

 

BTOL

GO

 

which would allow the wavefront variance to degrade by about 10% at three field points in all of the defined colors, and only utilize whatever paraxial solves were already present in the lens for adjustment purposes.  Usually, a more elaborate set of commands is called for.  An annotated example of a BTOL run is given in Chapter 8 of the Tutorial Manual.
