5.35 DOE OPD Table (DSAG)                Previous Next

 

The DSAG command will produce a table much like the SAG table, except that it lists the phase shift in a DOE element instead of the surface shape of an element.  Input is

 

 DSAG SN [ HAAP [ NSTEPS ] ]

 

Surface SN must have been declared a DOE, and the default semi aperture is the current CAO of the surface.  NSTEPS is the number of zones to be analyzed, with a default of 10 zones.  The analysis is performed along the Y-axis from center to HAAP.

 

Since the DSAG results are not directly available to the AI program, SYNOPSYS puts a copy of the results into the AI buffer, which can be accessed by AI.  Here is how one might produce a plot of the phase shift in a DOE on surface 3 as a function of the Y-coordinate from Y = 0 to 5.0:

 

MACro:

 

DSAG 3 AIP 1

ORD = FILE 1

 

Run the MACro once, and then type

 

DO MACRO FOR AIP = 0 TO 5

 

and the plot is produced.
