Printing the Command Window

 

You can print the contents of the active window by clicking on or typing Ctrl+P, or with the SYNOPSYS command PCW.  This, however, prints the entire contents of the current window.  If you want, for example, a printout of a SPEC listing, and nothing else, there are three ways to do it. One way is to create a new window, type the SPEC command, and then print that window.  Then you may continue to work in the new window or click in the previous window to resume work there.

 

A second way uses the PON and POFF commands to capture a portion of the printout and then send it to a printer. These commands can be included in a MACro, as can all commands, to automatically extract a desired portion of the listing that is produced by the MACro.  Click on the link to PON for a description of this feature.

 

A third way, slightly more complex, is available when you want to extract a portion of an existing Command Window without printing the whole contents or re-running the commands that produced the listing.  You do this by selecting the text you want, copying it to the clipboard, and then pasting it into a new, empty window, which you can then print.  Proceed as follows:

 

1. Select the text you want to extract by dragging through it with the mouse.  Type Ctrl+C to copy this text.

2. Create a new Command Window by clicking on or by typing Ctrl+N.

3. Type Ctrl+V to paste the selected text into this new window.

4. Type Ctrl+P to print the text.
