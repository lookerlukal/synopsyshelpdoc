15.1.5.3 Replacing A Lens With Al                                        Previous Next TOC

 

The verb REPLACE performs functions similar to INSERT:

 

REPLACE { ELEM / SURF } NB [ TO NB ] WITH CHOICE NB

REPLACE { ELEM / SURF } NB [ TO NB ] WITH MELLES 'lensname'

REPLACE { ELEM / SURF } NB [ TO NB ] WITH { ELEM / SURF } NB [TO NB ]

REPLACE { ELEM / SURF } NB [ TO NB ] WITH FILE 'filename'

{ ELEM / SURF } NB [ TO NB ]

 

 

For example:

 

REPLACE ELEMENT 5 WITH ELEMENT 9

REPLACE SURFACES 5 TO 8 WITH FILE ‘LENS23.RLE’ ELEMENT 4
