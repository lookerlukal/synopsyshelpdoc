8.6.1.2 Fringes at Intermediate Surface (FIS)                                 Next Previous TOC

 

The PUPIL feature described in Section 8.6.1 can show the fringes of a wavefront that is calculated at the final surface and drawn in a variety of ways, including as fringes.  The location of each ray point at the fringes is taken from the ray coordinate at the entrance pupil, however, and the picture does not show any distortion of the exit pupil, if any.

 

The feature described in this section can plot the fringes of the wavefront at any intermediate surface in the lens, with the ray locations taken from an exact raytrace.  Thus, it shows the position and shape of the beam at that location, along with the OPD displayed as a fringe pattern.  It can be run from the dialog MDI.

 

Bear in mind that the OPDs are calculated at the target surface, but not on the surface.  They give the OPD relative to a reference sphere located at that point in the lens, but ignore the actual shape of that surface.  If it is assigned a relative tilt, the program will emulate a tilted wavefront there with a calculated wedge in X or Y that gives the same OPD shift at each ray point as one would expect from that tilt angle.

 

If the beam is not centered on the target surface, the entire plot is shifted in X and Y accordingly and shows the actual beam location at the surface.

 

Input is:

 

 FIS ICOL HBAR ISN SCF GBAR { REF / PARAX } WX WY NGRID [ FULL ]

 

In this input,

 

ICOL

is a color number.  “P” may be entered to specify the primary color, but there is no multicolor “M” for this feature.

 

HBAR

is the fractional field in the Y-Z plane.

 

ISN

is the surface following which to find the wavefront fringes.

 

SCF

is a scale factor.  Entering 1.0 will draw the picture at 1x scale.  Entering zero will produce a default picture that is about five inches across.

 

GBAR

is the fractional skew field point.

 

REF

determines how the program will interpret the OPDs of the rays.  If this entry is zero, the beam at the requested surface will be treated as though the lens were in AFOCAL mode at that point, and the fringes will therefore be relative to a flat reference wavefront.  If a nonzero value is entered, that will be the distance to the center of the reference wavefront.

 

If the word “PARAX” is entered for REF, the program will calculate the location of the paraxial focus position following surface ISN and will make that the reference surface.  This option is probably easier for most users, when paraxial raytracing is meaningful.

 

WX, WY

These entries apply an optional wedge in X and Y to the wavefront fringes.  Here things get a little tricky, however.  In the case of the PUPIL analysis, the wedge applies across the entrance pupil, which is normalized according to the pupil definition.  But FIS shows the local wavefront at a different point, which is usually of a different size.  So the program has to know at what aperture the requested wedge will show up, and to do this it divides the current ray position by the current clear aperture assigned to that surface (or calculated by default).  If the wavefront nicely fills that CAO, the fringes will be as expected; if it is smaller than that value, then fewer fringes will show up.

NGRID

This entry controls the grid number of the evaluation.  The default is 100 x 100, and you may enter any number up to 500.

 

[FULL]

FIS also calculates a set of Zernike coefficients that model the observed fringe pattern.  They will be printed in compact form unless the optional FULL is entered in word 11.  That will produce a full listing of each polynomial.

 

Note that the Zernike polynomials are usually cast over a unit circle, which is not appropriate here, since the actual beam dimensions are likely not that size.  So the program scales the ray coordinates on the target surface by the total beam size in X and Y, thereby normalizing the data input to the Zernike analysis code.  You have to keep this in mind if you wish to utilize the coefficients later.  (The X and Y parameters to the expansion are normalized by that beam size.)

 

 

To illustrate this feature, we get the relay telescope out and find the fringe pattern and location at surface 11, in color 3, at full field.  The image is formed at a distance of 12.711 inches from surface 11, so we enter this reference dimension with the input.

 

 FETCH 4

 FIS 3 1 11 0 0 12.711.

Special caution is in order for nonsequential systems.  FIS alters the lens according to the details of the analysis, and it can happen that an essential surface for nonsequential raytracing is no longer present.  In that case the results will have little meaning, and you are advised to request a surface number that is not coincident with any surface declared CAPTURE, LOOSE, etc.  This will ensure that those surfaces remain undisturbed.
