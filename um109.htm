10.9 Final Results                                                        Next Previous TOC

 

When an optimization is complete, it is useful to obtain a listing of the final values of the aberrations and the relative impact of each on the merit function.  This can be done with the commands

 

FINAL [ n ]

 

FINAL NUMBER nb

 

FINAL NAME "name".

 

The first option shows the n largest aberrations (or all, if n is omitted or zero); the second option only shows the aberration numbered nb; and the third only the aberration that has been assigned the entered name string with NAME ( "name" must be in quote marks).

 

This listing shows the raw value of each aberration, the weighted value measured from the target, and the relative effect on the merit function.  The latter is the fraction of the value of the merit function attributable to that component, with the sum of all fractions equal to 1.0.  If a number is entered for n, only the n largest aberrations will be printed, in their numeric order (not ordered by size).  Another listing format is the ALIST, described in section 10.11. FINAL also copies the first 200 aberration values into the Al buffer (see 15.1.2.3).
