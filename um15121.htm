15.1.2.1 DO MACro Commands                                        Previous Next TOC

 

The plot commands described in the last section require that all of the plot definition be encompassed in a single input line.  It is also possible to make plots with a more elaborate definition requiring several lines.

 

To do so, first make a MACro containing all of the commands required for the plot (including refocusing, optimization, evaluation of parameters, etc.).  See section 13.2.  Now you have to get the MACro into the computer’s memory.  This happens automatically whenever you click the Run button in the editor.  You can also load it into memory without running it immediately with the command LTM name (Load This MACro) or by clicking the button in the editor toolbar.

 

Then type an AI sentence, such as

 

 

Do MACro for 1 TH = 2 TO 3

 

or

Do macro as the object distance varies from 200 to 2000

 

 

A lens parameter may be incremented by the implied loop, as the above examples suggest, and this parameter by default becomes the abscissa of the plot.  You may specify an ordinate (or a different abscissa) by including in the MACro the Al commands

 

Ordinate = parameter

Abscissa parameter

 

or equivalent sentences, where parameter is any valid AI noun or sentence that would ordinarily produce a numeric value as output.  As the MACro loops, a file is created in which the values of the loop parameter, ordinate, and abscissa are stored.  When finished, the plot is prepared and a table is printed of the two (or three) parameters and their values.  Some examples of this feature are given in section 15.22.

 

Note that other lens parameters may vary in the above process, besides the loop parameter.  The MACro might contain a CHG file, for example, with a NEW command – allowing you to input new data each time through the loop, or it might contain AI sentences changing one or more lens parameters, like

 

Increase the index of surface 5 by .01.

 

Often you will change a data item by replacing it with the AIP symbol, and then use the loop form

 

 DO MACRO FOR AIP = start TO end.

 

Plots may be superimposed using the verbs multi do macro and ADD DO MACRO ….  (See section 15.1.2 regarding the use of this feature.)
