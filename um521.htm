5.21 Adjusting magnification (MSET)                Next Previous TOC

 

The magnification of a lens is the ratio of the image height to the object height, and is printed with the SPEC listing.  The utility

 

MSET MAG [ A ]

 

can be used to adjust the object distance so as to achieve a desired paraxial magnification.  This command requires that the back focus distance be controlled by a YMT solve (and therefore does not work with AFOCAL lenses).  The entered magnification is the absolute value of the desired ratio, so the program will keep the same sign of the magnification as before.

 

The program prepares and executes an optimization run when this command is entered; the variable is the object distance and a single aberration, the magnification error, is corrected. The object height (YP0) remains constant unless the optional [ A ] is input; in this case the object angular extent (measured from the vertex of surface 1) remains constant.  Since the magnification is based on the paraxial Gaussian image height (GIHT), this feature cannot be used with the NPXT mode.

 

You may run MSET command via the MACro editor, but do not run it as the alternate MACro.  MSET creates its own alternate MACro to do the optimization, so this must be vacant.
