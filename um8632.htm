8.6.3.2 Single-field of view convolution MTF                 Next Previous TOC

 

There are two forms of single-field MTF, which are found in the dialog MOP, and which can also be run with the commands

 

[ Z]MTF { ICOL / M / P } HBAR FREQ [ GRID ] GBAR

 

printed analysis at a single frequency

 

[ Z]MTF { ICOL / M / P } HBAR 0 [ GRID ] GBAR [ P ]

 

analysis at 21 frequencies, with optional plot

 

The first form calculates the MTF in the tangential and sagittal directions at a spatial frequency of FREQ, which is in cy/mm for focal lenses and cy/mr for AFOCAL.  The mnemonic "MTF" uses the exact-ray option, while "ZMTF" the wavefront interpolation method.  This will be a linear spline fir to the wavefront.  An explicit color number may be entered for ICOL, the letter “P” for the primary color, or the letter “M”, indicating a multicolor analysis.  The second form gives the MTF at 21 frequencies, and optionally a plot, if the P is input.

 

If the lens is poorly corrected and you want to examine the MTF at low frequencies, you will get better accuracy with the ZMTF option.

The interpolation options can take the optional GRID input; the default is 80 (by 80), a smaller grid is somewhat faster, but the output can become irregular if the grid is too small or the system is poorly corrected.

 

The default cutoff frequency is the natural cutoff at the F/number of the lens and the requested wavelength (or the primary color in the case of a multicolor analysis).  You may enter a desired cutoff with the command

 

FCO FREQUENCY

 

prior to the MTF command.  This cutoff remains in effect until changed, or deleted with FCO 0
