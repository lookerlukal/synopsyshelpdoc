6.2.2 EDGE Thickness                                Next Previous TOC

 

The commands

 

EDGE [ SN ]

EDGE [ JSSS JSPS ]

 

Produce a listing of the center and edge thicknesses for all adjacent surfaces which are not tilted or decentered.  The calculation is performed in the Y-Z plane.  The program examines the apertures on either side of an element or airspace, using the CAO values or EFILE point A if defined.  Then it calculates the sags on both sides and finds the edge thickness. EDGE may be run interactively from the dialog MUT. Individual edges may be controlled in optimization with the construction-parameter ETH, and edges can be monitored in the whole lens with the mnemonics AEC, AGE, AFE, and AAE.

 

EDGE does not support hyperhemispherical surfaces, where there are two different sags for a given Y-coordinate.  In such cases, feathering can be controlled as outlined in Section 10.3.7.
