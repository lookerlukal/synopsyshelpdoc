8.5.2 Knife-edge Trace (KNI)                                 Next Previous TOC

 

The command

 

KNI { ICOL / M / P } HBAR NRYS SCF GBAR { X / Y } [ P ]

 

will calculate a geometric knife-edge trace.  The direction of motion of the knife is given by the { X / Y } choice.  The results are printed and will also be plotted if the optional [ P ] is input, with a scale factor of SCF equal to one inch on the abscissa of the plot. “P” for a color number is just the primary color. If a multicolor KNI is desired, M should be input instead of the color number. In this case only the weighted sum will be printed and plotted.  The zero point of the knife edge is the chief ray position in the primary color.  A sample KNI plot is shown below.  Another feature that can do knife-edge traces as well as other kinds of image analysis is described in Section 9.3. KNI may also be run interactively from the dialog MGI.
