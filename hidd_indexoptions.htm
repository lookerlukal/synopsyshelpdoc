Index Options Dialog

 

Clicking on the index flags column for a given surface opens a dialog where you can select from a variety of index options.

 

The glass model GLM option requires some explanation.  SYNOPSYS can model the partial dispersion of glasses over the entire glass map with an interpolation formula.  This feature is used when you want to vary the glass properties during optimization.  When you later substitute a real glass for the model glass, the secondary spectrum should be roughly what was predicted by the model.  This will not be true, however, if you select an anomalous glass, which is valuable precisely because it does not follow the same behavior of other glasses.  Moreover, if your lens contains a model glass, you will see the model parameters, i.e. the Nd and Vd input to the model, and not the resulting index values on the SPEC listing.  This can be disconcerting if your central wavelength is not at the sodium d line (0.58756uM), because the actual index there is not the same as the model index.  To see the actual indices, use the PRT or RIN command instead.

 

This dialog is reached via the SPS dialog.
