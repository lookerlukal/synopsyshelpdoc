5.25 Print Glass Attributes (PGA)                        Next Previous TOC

 

The command

 

PGA { SN / ALL }

 

prints a table of glass attributes for surface SN taken from the appropriate glass table.  If ALL is entered in word two, every glass-table glass in the lens is listed.  An example of PGA output is shown in below.

 

 

SYNOPSYS AI>PGA 3

The price is relative to glass type BK7.

 

Transmission properties are encoded in one of two ways:  newer glasses have a table of values from which the program can calculate transmission at any wavelength by interpolating over the spanned region, while older and obsolete glasses use a set of coefficients to a custom polynomial that applies to the entire transmission region with somewhat less accuracy.  These two schemes are also employed by some other features, such as GTRANS, which will display the transmission over a selected wavelength range.

 

To examine the transmission table, open the glass table (command MGT), select the desired catalog, click on the desired glass, click Properties, and click List.  A table will be displayed in the Command Window.
