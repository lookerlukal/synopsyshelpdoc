6.3.2 Global Raytrace (GRAY)                                Next Previous TOC

 

The path of a ray given by the previous commands (RAY, TARGET) is in the local coordinate system of each surface.  Other coordinate systems may be selected by the input

 

GRAY { ICOL / P } HBAR XEN YEN [ { SURF / 0 } GBAR [ JCOORD ] ].

 

The lens system will be put into GLOBAL mode if it is not already. GRAY always displays transverse coordinates (there is no OPD mode), in the coordinate system of surface JCOORD (default is surface one). GRAY may be run interactively from the dialog MRR.  An example GRAY output is shown below.

 

SYNOPSYS AI>
