8.0 IMAGE ANALYSIS ROUTINES – BASIC SET        Next TOC

 

The basic image analysis routines described in this section are accessed by entering any of the commands described below.  Previous releases of SYNOPSYS required entering the command

 

IMAGE

 

before the analysis command, but this is now optional.

 

They may also be accessed through one of the image-oriented dialogs, which start with the menu MIM, and are described in the next section.  Routines are available for performing either geometric or diffraction-based calculations on monochromatic or multicolor images.  All of the routines described in this chapter operate on a single object point at a time.  To analyze the effect of lens aberrations on an extended target, the MIT dialog, MFB, the DETECTOR feature, or the EIA command can be used instead.

 

Most of the image analysis routines first create a mathematical model of the image and then perform whatever analysis is required by the command.  A geometric knife-edge trace, for example, starts out with a spot diagram (the model) which is then masked in steps to give the analysis.

 

There are two exceptions to this rule: the partial-coherence program PARTC and the convolution form of MTF analysis employ algorithms that do not require an image model.  Most MTF analysis will use this convolution method, described in Section 8.6.3.  For special purposes, however, the image-model method is available for MTF analysis as well.  It is useful, for example, for vector-diffraction analysis in cases where polarization plays a role.  This is described in Section 8.6.3.4.

 

The programs generate a grid of rays and trace all of the requested rays exactly, deleting any that violate apertures or encounter feathering – unless the lens file contains the optional NOVIG entry, in which case only ray failures are detected.  The image is normally examined at the final focal plane, but the footprint routine allows you to specify an intermediate surface instead, for examining the ray pattern elsewhere in the lens.

 

All of the routines in this chapter utilize geometric raytracing to derive the ray grid geometry or the exit pupil configuration.  Another feature is available that can do diffractive propagation from one surface to the next.  This is described in Section 9.6.
