MPF PSPRD over Field Menu

 
The diffraction point-spread can be plotted in several ways, all of them accessible from the menu MDI.  The MPF menu shows the PSPRD in a variety of forms at several points in the field.  This dialog submits the OFPSPRD command with all of the necessary arguments.

 

The arguments available on the dialog are as follows:

 

Number of rays

Enter a number from 600 to 20000.  A large number will avoid aliasing but will run more slowly.

 

Height

This entry controls the height of the individual plots when drawn in oblique-perspective form.  If you enter a number other than 0, all of the plots will be scaled to that height.  If you enter 0, the program will scale each picture so that the relative intensity of each image controls the height.  A perfect image in this case has a height of about one inch.

 

Straight/Right/Left

This selection controls the view angle of the surface plots.

 

Show as surface/visual

These radio buttons control how each picture is displayed.  The surface selection draws a pattern of lines, while the visual selection shows what the image would look like to the eye.

 

Magnify, Gain

These data allow you to zoom in on the visual form of the display.  Magnify controls the zoom, and Gain adjusts the intensity of the picture.  If the images are not all nearly perfect, you should enter a Gain larger than 1.0 or the image may be too faint to see clearly.

 

Fields

Here you specify how many field points to display; the number must be from 2 to 20.  The first plot is always the on-axis image, and the last is at the edge of the field (HBAR = 1), with intermediate images showing intermediate points.  This analysis is performed in the meridional field only.

 

If you enter the word "FULL" instead of a number, the analysis will cover the full extent of the X-Y field.  In this case, the axis point is in the center and the corners of the field are at the corners.  The sampled field is rectangular if the system specification RFOV is in effect, and circular if not.

 

ICOL

This specifies which wavelengths to analyze.  A single number in the data boxes calls for analysis at that color, while entering “M” does the analysis in all defined colors.  Entering “P” produces a plot at only the primary color.
