8.5.1.2 Through-focus Spot Diagram (TFS)                 Next Previous TOC

 

A composite picture of the spot diagrams for up to 5 focus positions at up to 3 fields of view may be obtained with the input shown below.  It may also be obtained interactively from the dialog MTS.

 

TFS NRYS [ SCF ]

 

 

[ DF { 1 / 3 / 5 } DELTAF ]

(defocus data)

 

[ HBAR HBAR1 HBAR2 HBAR3 ]

(fractional field height)

 

[ GBAR GBAR ]

(fractional X object)

 

[ ICOL { ICOL / M / P } [ ICOL … ]

(color numbers)

 

PLOT

 

 

where NRYS is the number of rays in each spot, and SCF is that dimension that will be scaled to one inch on the plot.  The program will generate spot diagrams for each HBAR that is entered (default: 0, .75, 1.0) in each color ICOL (default: the primary color only), and plot them on a single picture.  “M” requests a multicolor plot in all defined colors, and “P” is just the primary color.  If the number of defocus positions entered on the DF line (default: 1) is 3 or 5, the program will also calculate the spots as they would appear at distances of plus and minus DELTAF from the image surface, based on the ray angles as they cross this surface.  These spot diagrams are then plotted alongside the first.  The procedure is repeated a second time if 5 defocus positions have been requested.  If just the word TFS is typed, the program will prompt for the other data items.  If SCF is omitted or zero, the program will use a reasonable default.

 

The program makes the assumption that the Z coordinates of the intercept points all shift together with the surface when defocusing, as is normally the case.  If the defocus is large and the focal plane sharply curved, however, this may not be true, and the new defocus should then be input with CHG and the spot diagrams calculated exactly.

 

The number of rays traced in each color is adjusted to represent the spectral weighting, for multicolor plots.  The ray grid geometry depends on switch 90: if that is turned on, a hexapolar array is used; if off, a square grid.

 

TFS plots will be shown in color if switch 35 is ON.  The displayed colors come from a built-in color table, and will not necessarily be the same color as the wavelengths analyzed.  To display the spots in approximately the visual colors, turn on switch 85 as well as 35 before requesting the plot.  In this mode, the program draws a black area before drawing the spots, since a good image could appear white, and would be invisible otherwise.  Wavelengths in the UV are shown in violet, and in the IR are drawn in red – even though in fact those wavelengths are invisible to the eye.

 

The shape of the spot is controlled by switch 27 (see section 8.5.1.1).  The spot symbol size (with 27 OFF) is controlled by the SSSIZE command.  An example of TFS output is shown below.
