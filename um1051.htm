10.5.1 Derivatives                                                                Next Previous TOC

 

It is sometimes helpful to see the derivatives of each aberration with respect to each variable. This can be produced if the command

 

DERIVATIVES [ 0 MULTI ]

 

is entered instead of the command SYNOPSYS.  No optimization will be performed in this case.  The optional MULTI must be input if the AANT or PANT files contain any ACON entries.  An Example of a DERIVATIVE printout is shown below.  (Note that for RAD's, the variable is actually the quantity ( 1 / 2RAD ).
