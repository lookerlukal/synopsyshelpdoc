10.3.17 Automatic Monitor of Meniscus Lens Center Separations (AMS)                        Next Previous TOC

 

If the centers of curvature of meniscus lenses are too close together, it becomes difficult to remove the residual wedge by centering and edging.  If they coincide, it becomes impossible.  The AANT file input

 

 AMS [ TARGET WEIGHT [ WINDOW ] ]

 

will monitor the absolute value of the separation for all elements whose thickness is a variable, and compare the value to TARGET.  If the value is larger, there is no penalty.  The penalty goes as the square of the violation if the error is less than WINDOW, and linearly thereafter.

 

To ensure that the separations are all no less than 2 mm, for example, the input would be

 

 AMS 2 1 1.

 

The target value is always a positive number in lens units.  This monitor does not apply to DOEs, HOE, or gratings.

 

This monitor should be used sparingly, and never when roughing out a lens.  A given element may want to change from, say, a positive meniscus to a negative, and in so doing it might have to pass through the forbidden area where the centers coincide.  If the monitor is then in effect, this solution might never be found, even though there is an excellent design on the other side.  So save it for when the design is otherwise finished but needs some tweaking.

 

In the drawing below, a lens has two radii with almost the same center location, separated by a small distance d, and a wedge error of W.  In order to remove the wedge, the lens has to be edged by an amount H, given by

 

H = W*R**2/d

 

If d goes to zero, H becomes infinite.  Since one usually has only a small amount of extra glass that can be removed in edging, the quantity d has to be greater than some amount.  That amount is the target for AMS.

 

 

This is one of several kinds of monitors that operate continuously during optimization.
