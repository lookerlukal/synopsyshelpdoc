10.3.2 Paraxial aberrations                                                        Next Previous TOC

 

Paraxial aberrations are specified by the input

 

{ A / S / MUL / DIV } name

 

where name is one of the following:

 

FOCL

focal length

BACK

back focal distance

TOTL

vertex-to-vertex length

GIHT

Gaussian image height (UB if AFOCAL)

(see section 6.1)

EPP

exit pupil distance

ENP

entrance pupil distance

DELF

shift from paraxial focus (-UA if AFOCAL)

FNUM

F/number (-0.5 / YA if AFOCAL)

 

YP0 or YPP0

object height, not for OBC or OBD

TH0

object distance

UP0 or UPP0

for OBC and OBD objects, is object angle in degrees

 

XP0 or XPP0

object height in X-direction; not for OBC or OBD

 

UXP

object angle in X-direction; only for OBC, OBD

 

YP1

chief-ray height

object parameters (see section 3.1.1)

XP1

chief-ray height, X-direction

YMP1

marginal ray height

XMP1

same, in X

 

GBR

Gaussian beam radius

GWR

Gaussian waist radius

(see section 5.12)

GBU or GBD

Gaussian beam divergence

GWL

Gaussian waist location

 

 

GXR

Gaussian beam radius in the X-direction

GXW

Beam waist radius in X

GXU

Beam divergence in X

GXL

Waist location in X

 

 

ACCOM

AFOCAL accommodation

(see section 3.2)

BTH

shift from paraxial (YMT) focus

BCL

back clearance.  See below.

SGTH

sum of glass thicknesses.  See below.

SYNC

this aberration does not actually calculate anything.  Its purpose is to synchronize a thermally shadowed zoom lens.  See Section 5.9.

 

For AFOCAL lenses, the mnemonic FNUM controls the quantity (1 / DIA) where DIA is the diameter of the collimated output beam, and ACCOM controls the accommodation, in diopters.

 

Another set of paraxial aberrations is also available.  These can target the paraxial ray coordinates at a selected surface.  See Section 10.3.2.1.

 

Although it is not strictly a paraxial quantity, the program will accept a target for BCL, which stands for Back Clearance.  This is an alternate way to keep the back focal distance greater than a desired minimum value.  Unlike BACK, which calculates and controls the paraxial back focus value – which can be infinite in some cases – BCL traces the axial marginal ray instead and tests to see if it intercepts the axis beyond the location of a dummy surface located at TAR from the last lens surface.  If so, the error is zero.  If not, the error is just the Y coordinate of that ray on that surface times the entered weight.  This should be a more reliable way to get the desired results, since the error is not likely to give a very large number at any time.

 

The quantity SGTH is the sum of all glass thicknesses.  This is intended for when you want to increase the transmission of the lens by reducing the total path through glass.
