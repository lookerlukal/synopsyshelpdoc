10.3.16 Automatic Monitor of Diameter/Thickness (ADT)                        Next Previous TOC

 

One often wants to control lens thicknesses so that the ratio of diameter/thickness does not exceed some value determined by shop practice.  A lens that is too thin will likely bend in polishing, making a good figure hard to achieve.  Just monitoring thicknesses is not always a good solution, since small lenses can indeed be thinner than large ones.  The AANT file input

 

 ADT [ TARGET WEIGHT [ WINDOW ] ]

 

will monitor this ratio for all elements whose thickness is a variable, and compare the value to TARGET.  If the value is smaller, there is no penalty.  The penalty goes as the square of the violation if the error is less than WINDOW, and linearly thereafter.

 

To ensure that the thicknesses are all at least 1/6 of the diameter, for example, the input would be

 

 ADT 6 1 1.

 

The program selects the larger of the current CAO values on either side of the element.  The target value is always a positive number.  This monitor does not apply to DOEs, HOE, or gratings.

 

A second form is also available:

 

 ADS [ TARGET WEIGHT [ WINDOW ] ]

 

This form adds the sags of the surfaces if they are concave to the thickness before evaluating the aberration.  This is so that meniscus lenses, which are inherently stiffer than flat plates, do not wind up with thicknesses greater than necessary.  A steep meniscus lens, or a negative lens with a thick edge, will likely not flex under polishing, and this form takes that into account.

 

The two forms are mutually exclusive: if you select one you cannot select the other.

 

This is one of several kinds of monitors that operate continuously during optimization.
