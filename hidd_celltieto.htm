Cell Tie to dialog

 

This dialog provides an easy way to establish the dimensions of cell parts.

 

Target: OD, ID, Cell start, Cell end

 

Select the cell dimension you wish to tie to something else.

Source: Surface

Here you control the parameter by reference to a lens surface dimension.  EFILE point C is just the lens OD, while the sag at a given Y would be used to put a shoulder on a part that just touched the lens at that point.  EFILE point A is the break between the surface and the flat portion (or bevel if there is no flat.)  Note that the lens must have EFILE edges defined to use this option.

 

Source: Part

This lets you tie a dimension of a part to another dimension of another part.  Since parts must be contiguous for the drawing editor to consider them parts of a larger piece, this option is used to render that so.

 

If the cell that is the source of the dimension has a nonzero slope angle, you may want to specify whether to pick up the value at the start or end of that part – or at a position midway that corresponds to the start point of the target part.  Simply select the appropriate radio button.

 

Honor clearance

This box adds twice the clearance specified in the Cell Editor to a lens OD before tying that dimension to a cell ID.  The clearance value is thus the radial separation between the two parts.

 

OK

This completes the data transfer and returns you to the Cell Editor.  Note that any ties that you create here are not pickups in the sense of lens files.  They are a one-time data transfer only, and if you later change the source dimensions, the target does not automatically change to keep up.  You have to go to the Tie-to dialog once more to pick up the new values.

 

Cancel

Click here to return to the Cell Editor without changing anything.
