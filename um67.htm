6.7 Fifth-order analysis (FIFTH)                        Next Previous TOC

 

Third and fifth-order coefficients are produced by the commands

 

{ FIFTH / NFIFTH } [ ELEMENT ]  [CHROM [ ICOL1 ICOL2 ] ]

 

or

 

{ FIFTH / NFIFTH } [ SN / SURFACE ] [ FULL ] [ CHROM [ ICOL1 ICOL2 ] ]

 

or

 

{ FIFTH / N FIFTH } [ JSSS JSPS / P ] [ ELEMENT / FULL ] [ CHROM [ ICOL1 ICOL2 ] ]

 

If JSSS and JSPS are entered, a listing is given only for surfaces in that range.  If the optional [ P ] is entered, the listing gives the contribution of each surface.  If only SN is entered, the listing gives the fifth-order contribution of that surface only.  The FULL option prints the coefficients broken down into the intrinsic and transferred parts; and if only the FIFTH or NFIFTH mnemonic is entered, only the system totals are printed.  The CHROM option prints chromatic differences.  NFIFTH gives aberrations normalized to F/1 in air.  If two color numbers are entered with the mnemonic CHROM, the chromatic difference in fifth-order aberration will be printed for the entered color numbers (the letter “P” is not allowed in this case).  Definitions of these aberrations are given in App. D. FIFTH may be run interactively from the dialog MFT.  The output is in lens units.

 

The program calculates the 5th-order aberrations with an algebraic method that works for all simple shapes and some of the more complex ones -- but is not applicable for all shapes. Surfaces defined as HOEs, Forbes polynomials, and the like.do not lend themselves to this calculation, so the program instead evaluates the aberrations at the final surface by tracing a set of rays and fitting the intercepts to a power series, the terms of which correspond closely to the desired aberrations.  This approach itself has limitations however: it does not return the aberrations of only a section of the lens, so the requested surfaces, if any, must then include the entire lens.  Also, it does not return the pupil aberrations, which are therefore unavailable with those shapes.

 

 

The fifth-order aberrations are:

 

SA5

fifth-order spherical aberration

 

CO5

tangential coma

 

TI5

tangential astigmatism

 

SI5

sagittal astigmatism

 

PZ5

Petzval curvature

 

DI5

distortion/GIHT (distortion / 1.0 with NFIFTH)

 

TOBSA

tangential oblique spherical

 

SOBSA

sagittal oblique spherical

 

ECOMA

elliptical coma
