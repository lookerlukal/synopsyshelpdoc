7.3.2 Long form input                                        Next Previous TOC

 

(See 7.3.1 for a short form of this feature).  This more lengthy form of fans plot input allows you to specify the field angles to be plotted in addition to the other data.

 

ICOL ( ICOL / M / P }[ ICOL … ]

FOV HBAR GBAR

[ FOV HBAR GBAR ]

[ FOV HBAR GBAR ]

[ FOV HBAR GBAR ]

PLOT

 

Here, the ICOL entries specify the color numbers that are desired, “M” requests a multicolor analysis in all currently defined colors, and “P” requests the primary color. Up to four field points may be requested with the FOV entries, each defined with a fractional field in the Y direction (HBAR) and in the X direction (GBAR).  The resulting plot will be in color if switch 35 is ON.  A SCF of zero gives a default scale.

 

If switch 59 is on, pupil reduction is shown when the pupil is reduced in size by any of the pupil options, even though you may have selected an option that normally shows the reduced pupil covering the full axis (such as the VFIELD and WAP options).  In this case, the abscissa of each plot gives the full pupil size, rather than the vignetted size, and the TFAN extends over only a portion of the whole axis if the pupil has been reduced.
