7.14 Longitudinal spherical aberration (LSA)                         Next Previous TOC

 

The command

 

 LSA SCF { P / 0 }

 

will produce a table and a plot (if P is input) of the longitudinal spherical aberration for 25 points in the aperture.  For each point, the real-ray intercept in Y measured from the on-axis chief ray point is divided by the difference of the tangent of the ray angle (HH) measured from the chief ray.  The output gives two columns, listing the entrance pupil coordinate and the longitudinal aberration.  A zero SCF gives default scaling.  This command can also be run from the MRR dialog.  It only works on lenses that are either FOCAL or APERFECT.  The analysis is always performed over all defined colors.
