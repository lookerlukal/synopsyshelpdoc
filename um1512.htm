15.1.2 Parametric Plots, Listings                                        Previous Next TOC

 

AI can analyze the values of two parameters as a third (or one of the two) is varied incrementally, and plot or print the results.  You may input the number of steps as well as the scales and labels of the two plot axes.  The default STEPS is 20, and the scales are adjusted automatically unless entered explicitly.  You can run this feature with a single line of input, as described below, or with a MACro, described in the next section.

 

Plot HH on surface 2 VS FOCL for 3 RAD = 1.2 to 3.4 at full field

Print the SAG of 1 versus aperture for apert equal .1 to .2

 

MULTI Plot YA for YEN = 0 to 1

Add plot ZA on 2 for YEN = 0 to 1

end

 

Plot BACK as the object distance varies from 500 to 1000.

Plot the spherical aberration for a conic const on 6 = 0 to 1

What is STEPS?

STEPS = 100

SCALE?

SCALE =.001 by 50.

 

Note that the SCALE directive above would be confused with the SCALE command if you did not have the “=” sign or another verb that clearly makes it an AI sentence.

 

 

AI allows you to add another plotted curve to a previous plot, but first you have to make sure that it does not show the first plot before you make the second.  In the above examples, the third line makes a plot with the verb “multi plot” instead of “plot”.  This allows the fourth line to add a curve to the plot made by the third line, and the END statement following it then shows the picture.  The abscissa of the second (and subsequent) plot should be the same as on the first plot, to guarantee the same horizontal scale factor for the two curves (or an explicit scale may be entered for the first plot)

 

The last sentence in the above examples will cause the plot scale to be. 001/inch on the abscissa (horizontal axis), and 50./inch on the ordinate.

 

SCALE = 0 by 0

 

restores the default automatic scaling of both axes and also restores the default automatic origin location.

 

Another directive can reuse the previous scale when plotting more than one curve on a single page with the input ADD PLOT … or ADD DO MACRO….  If you want the scale and origin of the added data on both axes to be the same as the previous data, you may use the input

 

SCALE = LAST,

 

which may be entered after the first plot command is executed, but before the second.  The program will reuse the scale and origin for all following AI plots until the defaults are restored as described above.

 

Setting the plot origin and labeling of axes are automatic, unless explicit requests are entered.  To specify a particular origin, which is frequently needed when a particular scale is entered, enter a sentence such as

 

 

Origin = 2, 15

 

or

Set the origin equal -1 by 10.

 

 

The origin, like the scale, remains in effect until default scaling is requested by entering zero scale for both axes.  Note that ORIGIN = 0, 0 does not restore the default automatic origin location: it places the origin at exactly (0, 0).

The axis labels are taken by the program from the data plotted, and are usually satisfactory.  In some cases, however, such as when the variable is the AIParameter (see 15.1.2.2) you will prefer to enter your own label.  To do so, enter a sentence such as

 

Label of ordinate = 'Relative field point'

Abscissa label equals 'Here is a sample label'

OLAB = ‘This is the ordinate’

Change ALABEL to 'and this is the abscissa'

 

A character string up to 24 characters long may be given, and must be put in quote or apostrophe marks. The mnemonics OLAB and ALAB may be assigned the desired string, which will then appear on the next plot.  (The axis labels are entered before the PLOT command.)  The labels are in effect for only a single plot, and must be reentered if the same labels are needed for the next plot.  A picture may be replotted with a different scale or origin, or with different labels, by entering the new labels or data and then typing AGAIN -- see 15.1.2.4.
