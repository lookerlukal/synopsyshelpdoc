7.10 Combining Lens Drawings (CDWG, LDWG)        Next Previous TOC

 

SYNOPSYS can show several different lenses on the same picture.  There are two formats; the commands for the first are

 

CDWG { 5 / 10 }

…

CDWG [ TSCF [ JSSS JSPS [ HBAR NB NB NB ] ] ]

…

CDWG …

 

CDWG PLOT

 

The first CDWG command specifies how many pictures you want to fit on the page.  If you need up to 5, enter 5, and if you need up to 10, enter 10.  The latter will of course produce smaller individual pictures.

 

Then you enter additional CDWG commands in the same format as the DWG command, up to the number that you allowed for before.  Each new drawing gets placed on the same page as the previous one, in a lower position.  When you have created all of the pictures you want, enter CDWG PLOT to produce the composite picture.

 

If the entered scale factor on any CDWG command produces a drawing that extends beyond its allotted area, the program will automatically reduce the scale so that it fits.  There is little room for each lens on the 10-picture drawing, however, so we allow some overlap in this case – which may be all right if the overlap of one lens happens to be in a narrow section of its neighbors.  If you wish to avoid this overlap, it is best to give your own scale factors in this case.

 

Here is a sample MACro that creates a drawing that shows five lenses:

 

GET 1

CDWG 5

CDWG 1

GET 4

CDWG .25 1 99 HBAR 0 -1 1

GET 7

CDWG  1

GET 2

CDWG .33

GET 5

CDWG .33

CDWG PLOT

The other format is

 

LDWG [ ALL / nb nb ... ]

 

This form draws all the lenses currently in the lens library, or only those whose numbers are listed.  The default scale and surface range will apply to all of the drawings.  In the event those choices are not what you want, use the CDWG format instead.
