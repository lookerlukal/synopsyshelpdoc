8.6.14 Pupil plots over field                                Next Previous TOC

 

SYNOPSYS can create a plot that shows the shape of the exit pupil wavefront at several places in the field.  This feature can be run from the MDI dialog by selecting the Over Field checkbox in the PUPIL section and then clicking the PUPIL button.  The coordinates displayed on the plots are taken in the entrance pupil, while the wavefront is at the exit pupil.  (To see the wavefront at the exit pupil, use the PUPIL command and specify EXPUPIL.)

 

Command-mode input is

 

[ FRINGES ]

[F]FPU ICOL SCF GBAR WX WY NFOV.

 

An oblique-perspective plot is produced unless the command is preceded by the command FRINGES.  The FFPU form analyzes from -1 to 1, while the FPU form goes from 0 to 1.

 

The entries ICOL and GBAR have their usual meaning: color number and fractional skew field.

 

This feature does not have a multicolor option, and it does not support contour plotting of the wavefront (as does PUPIL).  You may enter the desired number of field points as NFOV in word 7; if entered the number must be between 2 and 15.  The default number is 15 field points.

 

The scale factor SCF applies only to the oblique-perspective view and specifies the number of waves plotted per 0.25 inch on the plot.

 

The entries WX and WY give an optional wedge that will be added to the wavefront in the X or Y directions; a value of 3 to 5 can make the fringes easier to interpret.  An example FPU plot is shown below, in this case without the added wedge.

 

 

The command PUPIL will plot a single field point.
