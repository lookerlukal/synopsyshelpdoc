15.3.2 Starting a lens from scratch                                        Previous Next TOC

 

If you enter START at the XSYS> prompt, the program will prompt for the object coordinates and the overall requirements.  All input to XSYS must be in units of MM (the example lenses and your starting lens, if any, are automatically scaled to MM if the lenses are not in those units).  When the general specifications are in, the program prompts for the weights to be used when it compares the new requirements to the example lenses.  The input from this point is the same as for the ALTER command, and is discussed in section 15.3.4.
