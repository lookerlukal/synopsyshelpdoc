The Pupil Wizard

 

The Pupil Wizard can be found in the top menubar under “Wizard”, and may be accessed via the command MPW or with a toolbar button in the WorkSheet.  This Wizard will help you to set up the object, the stop surface and aperture, and control the paraxial and real entrance pupil definitions.

 

Another feature, the Object Wizard, MOW, can do much the same thing but is organized differently.  You are encouraged to try both MPW and MOW and select whichever best suits your needs.

 

The Wizard works best if your lens already has all of the surface data.  Then it will let you select from the more exotic options that you may need.

 

This dialog is intended to be self-explanatory.  It differs from the SYS dialog by having a “Cancel” button.  This button will discard any changes you make while in the dialog (except for the VFIELD option, which opens its own dialog with its own Cancel button, and changing the stop surface, which is done immediately.).  The SYS dialog, on the other hand, is more like a spreadsheet, and any changes you type are immediately entered into the lens and may in some cases alter the data elsewhere in the dialog.

 

You can see an echo of the lens changes that are generated by this Wizard by placing switch 23 ON.

 

The Wizard will turn off XPXT mode, if it is on, if the pupil is delared circular and there is no skew field point defined.  If you really want this option to be on, you must reset it yourself after running the Wizard.
