13.2.1 The Startup MACro (SYSTART.MAC)                                Previous Next TOC

 

Program initialization is performed via a special MACro called SYSTART.MAC.  This file is automatically run when SYNOPSYS is started, to set various default options.  It also calls a separate file, ICONS.MAC, which contains the definitions of the character strings that are produced by the buttons on the SideBar.

 

Some users prefer to initialize the program with different settings.  It is not a good idea to put your custom settings into the SYSTART file, since every time the program is updated this file is replaced during installation and your settings are then lost.  Instead, you should create a separate MACro called CUSTOM.  The startup MACro will call this file, if it exists, when it has finished with its other work, and the installation procedure will leave this file alone.

 

In your custom MACro you may set any of the mode-control switches that you want changed from the default, and also define any AI symbols that you want to have automatically recognized by the AI program.
