3.8.4 External coordinates                                        Previous Next TOC

 

It is sometimes necessary to examine the path of a ray or the positions of elements in a coordinate system that is removed from the lens itself.  This might be the case with an alt-azimuth telescope mount, for example, where you want the path in the coordinates of the mount rather than surface 1.

 

This is done by placing the entire system in EXTERNAL mode with the input (in the RLE or CHG file):

 

 EXTERNAL ANGLE AE BE GE

 EXTERNAL POSITION XE YE ZE.

 

Here, too, the Euler-angle order may be specified by substituting for the mnemonic ANGLE one of ABG, AGB, BAG, BGA, GAB, or GBA.  This input gives the position of surface 1 in the external coordinate system.  After these data are entered, you may ask for the path of a ray in external coordinates with

 

 ERAY { ICOL / P } HBAR XEN YEN GBAR [ SURF ]

 

You may also vary and target any of these quantities with the above mnemonics, or ask AI to set or tell you their values.  To see the location of the vertex of each surface, use the command

 

 ASY EXTERNAL.
