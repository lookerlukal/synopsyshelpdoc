10.3.5.1 Limit Input                                                                Next Previous TOC

 

Sometimes you will want an aberration to be greater than or less than a particular value, but do not care what it is otherwise.  It is unwise to give a specific target in such cases because that might constrain the solution unnecessarily.

 

To permit specification of a “one-sided” requirement, three input formats are available:

 

 { LUL / LLL / LUA / LLA } TAR WT WINDOW

 

{ L / LIM } TAR WT

 

and        LL A1 f1 A2 f2.

 

These are used instead of the usual M TAR WT before an aberration definition.

 

The simplest of the three is the LUL ... format.  Here you specify a target value, a relative weight, and a window size.  The LUL option specifies an upper limit, while the LLL form gives a lower limit.

 

Considering the LUL form first, if the quantity to be controlled is less than TAR, the aberration is zero because this is an upper limit and we don’t care in that case.  For values that exceed TAR, the aberration varies as the square of the departure from TAR, calculated so that if the excess is just equal to WINDOW, the aberration value is equal to WT.

 

For the LLL form, the rules are reversed; the aberration is zero if the quantity exceeds TAR, and varies as a quadratic if the quantity is less than TAR.

 

The LUA form gives an upper limit to the absolute value of the aberration, while LLA gives a lower limit to the absolute value.

 

Thus, to control airspace number 4 so that it is larger than 10 mm, one could enter in the AANT file

 

 LLL 10 2 1        ! the lower limit is 10 lens units; weighting factor = 2.0, and the window size is 1 unit.

 A TH 4

 

If the value were to decrease to, say 9 mm, then this aberration would get a value of 2.0.  One can adjust the importance of this control with the WT input, and get as precise a result as required with the WINDOW parameter.  (As with all aberrations, one must be mindful of the need to balance things: a very tight control somewhere may result in other goals being neglected.)

 

The second form defines the aberration as an exponential, which is also a one-sided control.  The aberration in this case is defined to be

 

fi = exp( WT (Aberration - TAR ) )

 

Either a positive or negative limit may be imposed, depending on the values of TAR and WT.  You can enter these values yourself using the following steps, or you can select the "LL" option and the program will find them for you.

 

In either case, you will identify two values of the controlled quantity and two values of the aberration that will correspond to those values, as follows:

 

1.

Choose A1 f1, A2, and f2 where A1 and A2 are values of the quantity near the desired limit, and f1 and f2 are the aberration values you wish to go into the merit function corresponding to those values.

 

Then, using the LL method, you simply enter those values, according to the format above.  The program then calculates the values of TAR and WT that yield an exponential function that fits those data.

 

2.

If you prefer, you can calculate the values of TAR and WT yourself, as follows:

 

WT = ( ln ( f1 / f2 ) ) / ( A1 - A2 )

 

TAR = ( ln f1 + WT A1 ) / WT

 

For example, suppose you wish the narcissus value on surface 9 to be greater than 0.01.  Suppose that the other aberrations in the merit function have values of about 0.02. Then you might pick values for the parameters A and f

 

A1 = 0.008, f1 = 10

A2 = 0.010, f2 = 0.02

Then

 

WT = ( ln ( 10.0 / 0.02 ) ) / ( 0.01 - 0.008 ) = 3107.304

and

 

TAR = ( ln ( 10.0 ) + 3107.304 * 0.008 ) / 3107.304 = 0.008741.

 

The AANT entry would be

 

L .008741 3107.304 A NAR 9

 

A simpler input would be just LL .008 10 .010 .02 A NAR 9, which would do the same thing.

The exponential form works as well as the quadratic forms (LUL and LLL), except if the starting system is very far from the desired target.  In that case, the exponential value may well become extremely large and the control will not work properly.  So unless the starting design is fairly close to the target, you will get better results with the quadratic form.
