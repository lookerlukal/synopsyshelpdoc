5.36 Transmission along a ray (RTRANS)        Previous Next

 

The transmission of an optical system may be calculated by several features of SYNOPSYS, itemized in Section 5.39.  The easiest way to run these options is via the menu MTR.

 

The command RTRANS, in this section, will calculate the absorption along the path of a given ray through the entire system.  It dos not consider vignetting, apodization, or coatings.  This feature requires all optical materials to be from one of the glass tables and have transmission coefficients assigned.  If this is not the case for any lens, the program will ignore that material and issue a warning message.

 

The format is the same as the RAY command:

 

RTRANS { ICOL / P } HBAR XEN YEN { SURF / 0 } GBAR

 

ICOL / P

color number, or “P” for the primary color

 

HBAR

fractional object height in the Y-direction

 

XEN, YEN

fractional entrance pupil coordinates

 

GBAR

fractional object height in the X-direction

 

If SURF is entered, the program will itemize the transmission at each element.  An example follows.

 

SYNOPSYS AI>RTRANS 3 0 0 1 SURF

SYSTEM TRANSMISSION ALONG SELECTED RAY

SURF.  PATH LENGTH  THIS PATH   CUMULATIVE

------------------------------------------

  2    14.6111      1.0000      1.0000

  6    5.99471      0.9998      0.9998

  8    5.05732      0.9986      0.9984

 10    6.13605      0.9944      0.9928

TRANSMISSION ALONG RAY:   0.992823E+00

                                                 

You can also evaluate the transmission through a single element, as described in Section 5.50.
