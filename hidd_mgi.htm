MGI Geometric Image Analysis Dialog

 

The MGI dialog provides simple access to many of the most often used geometric analysis features of SYNOPSYS.

 

Click                To

 

SPT

Make a spot diagram in one or more colors.

 

KNI

Make a knife-edge trace in either the X or Y direction.

 

RMS

Find the RMS spot size, variance, standard deviation, and the coordinates of the image centroid relative to the chief-ray point in the requested color.

 

FOCUS

Find the focus position that minimizes the RMS spot size.

 

ILLUM

Find (or plot) the relative illumination as a function of field point.

 

Select SYNOPSYS mode switches that apply to these features.
