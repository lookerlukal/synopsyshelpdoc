7.15 Forward scattering (FSC)                         Previous TOC Next

 

SYNOPSYS can model the scattering on a surface due to dust or pits, and show the ray pattern that emerges in a variety of ways.  The input is

 

 FSCATTER ISN NRYS ANGLE TSCF MODE STAT

 [ PER EL AZ SCF JSSS JSPS ]

 RAY { ICOL/P } HBAR XEN YEN GBAR

 RAY ...

 PLOT

 

This feature may also be run from the dialog MFS.

 

ISN

is the surface at which to scatter light

 

NRYS

is the number of scattered rays to model

 

ANGLE

is the semi-angle of the scattering cone

 

TSCF

is a drawing scale factor; interpretation depends on the MODE

 

MODE

Here is where you request the kind of analysis you want.

1 = display spots at the image

2 = make an oblique-perspective view of the pattern

3 = show the pattern as colored boxes

4 = show ray paths on a PERSPECTIVE drawing.

 

STAT

This selects the statistics of the scattered rays:

1 = center weighted

2 = uniform

 

PER...

If you request mode 4, this is where you enter the perspective arguments.  The PER command must be placed before any RAY commands.

 

RAY...

This defines the input ray.  You may enter as many RAY commands as you wish.

 

The program will trace the ray defined by the arguments of the RAY command up to surface ISN.  At that point it will create a pencil of NRYS rays, centered on the requested ray, and attempt to propagate those rays to the final surface.  The cone of rays will depart from the direction of the input ray following surface ISN, by a random angle, adjusted for the selected statistics, not more than ANGLE.  When all rays are accounted for, it makes a drawing.

 

If the lens is in VIG mode, all scattered rays are tested and deleted if they exceed any user-defined apertures.  In NOVIG mode, only rays that fail to trace are deleted.

 

In MODE 1, the picture resembles a footprint, where all rays that reach the final surface are shown.  The TSCF factor gives the scale of the picture, where a value of 2.0 means twice scale, and so on.

 

MODE 2 formats the pattern as an oblique-perspective view, where the local ray density is indicated by the height of the pattern.  In this case, TSCF gives the height, in inches, of the highest peak on the pattern.  The area of the final surface that is displayed is taken from the current CAO of the final surface.

 

MODE 3 creates an array of colored boxes showing the local ray density at each point.  The scale TSCF is ignored for this format, and the image scale is taken from the current CAO of the last surface.

 

MODE 4 creates a PERSPECTIVE drawing of the lens.  The path of the requested ray is shown up to surface ISN, and the scattered rays are shown emanating from that point.  The scale of the drawing is taken from the PER input in this case.

 

It is reasonable to request several thousand rays for modes 1 through 3, and more rays make a smoother picture.  In mode 4, however, it is best to request only a few hundred, since the drawing becomes quite cluttered with a larger number.

 

FSC works for nonsequential systems, but only up to the scatter point.  Thereafter, the scattered rays are traced sequentially.

 

A separate feature can model a diffuser, where every ray arriving at a given surface is scattered according to the same algorithm.  This is the USS type 10 surface.

 

Here are examples of the modes, using the sample lens from library location 1.

 

 

GET 1

CHG

VIG

CFIX

9 CAO 22

END

FSCATTER 7 10000 20 3 1 1

PER 0 0 3 1 123

RAY P 0 0 1

PLOT

 

 

Here we have made all of the apertures hard (with CFIX), and entered a CAO on surface 9 that is larger than the default (so we can see rays scattered at a larger angle).  The rays are scattered from surface 7.  We asked for 10,000 rays inside a cone of 40 degrees total angle.  The listing and plot show that of these, only 8954 arrived at the final surface.

 

The same input, but mode 2 this time, gives the picture below.  Here, it is easier to see the central concentration.  These examples use the center-weighted statistics: in this case, the cone is divided into concentric shells, and the number of rays in each shell is a constant.  Since the central shell has the smallest angular area, the ray density there is therefore higher than in the outer shells.  This models the usual case in forward scattering where narrow-angle scattering is more intense than is wide-angle.

Now we will use this feature to examine the effects of dust inside an eyepiece.

 

 

At HBAR = 0.7, shown on the drawing, the ray at surface 9 is very near the focal point.  We want to see whether a dust particle located there would appear sharply in focus to the user.

 

GET 9

CHG

END

FSCATTER 9 2000 20 100 1 1

PER 0 0 1 2 18

RAY P .7 0 0

PLOT

 

 

The scattered rays are indeed collected in a small group, but the angular size of that group is about 0.25 radians, which is about 14 degrees as seen by the eye.  So this particular dust particle will not be a serious problem.
