FORMULATION OF CONIC SECTIONS                        TOC

 

Curvature specifications of conic sections are stored by SYNOPSYS in two locations per surface.  Into these locations are placed the values of the quantities A and C in the equation

 

A Z**2 - Z + C S**2 = 0

(S**2 = X**2 + Y**2)

(1)

 

which describes all conic sections of revolution about the optical axis.  If each type of conic is considered a generalized ellipse, whose axes are as shown in Figs.  Ca through Ce, the entries in the following table show how it is described by equation (1).

The four conics may alternately be described by an equation of the form

 

Z = ( Y**2/R) / ( 1.0 + ( ( K + 1 ) Y**2/R**2 ) )**1/2 )                        (2)

 

which is equivalent to the previous descriptions if we make the substitutions

 

R = a**2 / b                        ( -a**2 / b if hyperbola)                (3)

 

K = R / b - 1 = ( a**2 / b**2 ) - 1.                                                (4)

 

Input data for these surface shapes may be input to SYNOPSYS in two ways: If the conic can be described by a radius R and a conic constant K, the data may be specified (in an RLE or CHG file) by

 

SN RD R CC K.

 

If it is more easily described by an equivalent ellipse, the data may be entered with

 

SN B b A a                          (Note: enter b before a.)

 

where b and a are the semi-major and semi-minor axes of the ellipse.  Note that the conics are defined in such a way that the geometrical length of a is always positive except for the hyperboloid, for which it is imaginary.  Since you cannot otherwise enter an imaginary number into SYNOPSYS, the program will interpret a negative value of a as really an imaginary value, since a can never be negative.  The sign of b is indicated in the figures.

A second convention exists to avoid dealing with the infinite value of b in a paraboloid; K is entered equal to -1, and the value is incremented by the program by 1.0E-7 automatically.  If the elliptical axes are to be entered, it is sufficient to enter a large number for b, say 10.E10, and to enter the value of (Rb)**1/2 for a.
