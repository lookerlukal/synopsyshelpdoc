MPE – Perspective Lens Drawings Dialog

 

The MPE dialog can set up the input for a PER drawing, an RPER drawing, a SOLID model drawing, or an RSOLID drawing.

 

While the input differs slightly between these three, all of them let you specify a selection of rays or raygrids that will be shown, and in which color.  The PER drawing shows the lens from a fixed perspective in wire-frame view; RPER lets you spin the lens around and select the view you like; SOLID shows a fixed-perspective view with hidden lines removed and several options for the color and shading of the elements, and RSOLID makes a solid model that can be rotated on the screen with the mouse.

 

If the current PAD display shows a perspective drawing on the top, you will be able to transfer the view angles to the MPE dialog by clicking on the PAD button.  This way you can quickly find the angles you want in PAD, and then make your drawing from the same angle.  If the PAD display does not show the perspective view, this button is disabled.
