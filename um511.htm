5.1.1 Surface Profile (SPROFILE)                                Next Previous TOC

 

The shape of a surface represented by power-series deformation coefficients, Zernike coefficients, ASTORICs, or surface designations NCZONE or USS may be visualized by plotting the surface as an oblique perspective or contour drawing.  The input is

 

[ CONTOUR [ SCF ] ]

[ FRINGES [ WAVL ] ]

SPROFILE ISN {0/F} SCF [ {AP/0} [ PRINT ] ]

 

where

 

ISN     is the surface number containing the aspheric coefficients

F        shows the full surface shape, including the base curve and conic constant, if any.  If zero is entered, the program shows only the aspheric contribution, not the base curve or conic contribution (except for ASTORICS, which show the total shape).

SCF   is the scale factor, in inches of deformation per inch on the plot

[ AP ]        is the clear aperture to be analyzed (default is the CAO of the surface).

PRINT produces a printed list of (X,Y,Z) coordinates across the surface.

 

Contour output is obtained as in section 8.6.1, and a fringe pattern is produced if a FRINGE command precedes the SPROFILE command.  In this case, the fringes give the pattern that would be seen in a double-pass configuration with a reference wavefront that exactly fits the base curve of the surface plus any conic constant that may be present (if the “F” option is not selected; if it is, the reference wavefront is flat.)  The default reference wavelength is .6328 μΜ.  A sample SPROFILE plot is shown below.  (This example shows the fringe pattern resulting from the cylinder error that the Monte-Carlo program has put on a surface to model the irregularity error of that surface.)

 

If you want the printed list (produced by the word PRINT in word 6), be sure to activate print capture first.  The data so produced will generate up to 2500 data points, which is too large for a single Command Window buffer.  Type

 

 PON filename

 

Before you run the SPROFILE command, and then POF C afterwards.  The data will be in the file you created.
