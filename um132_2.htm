13.2.2 Other MACro commands                                        Previous Next TOC

 

Other useful commands that can be included in a MACro are

 

 

SKIP

and

EOS

 

which cause any text between them to be skipped when the MACro is executed.

 

The command EDS (End Data Set) causes all of the text that occurs after it to be skipped.

 

You can annotate a MACro with the commands

 

; any text you like                        (not restricted to the character set)

! any text you like                        (not restricted to the character set)

 

or        TEXT

 

       one or more lines of text

 

       END

 

 

The command NEW within a MACro causes execution to pause when the MACro is run, and you are then asked for a new line of input.  That input replaces the NEW command, and execution continues.  If the MACro is running in a loop (see DO MACRO), you can therefore substitute a different line each time the loop reaches that point.
