4.1.1 DWG with SPEC (DSPEC)                        Previous Next TOC

 

The SPEC listing can be sent to a graphics plot instead of the Command Window.  A lens drawing, similar to either the DWG or PERSPECTIVE drawing, is shown at the top of the page, with the listing below.

 

The command is

 

 DSPEC [ TSCF [ JSSS JSPS [ F/0 ] [ YOS ] ] ].

 

In this input, TSCF is the scale factor, where 2.0 would be twice scale, and the surface range is given in the data JSSS and JSPS, as for the DWG command.  This feature always traces at three field points: center, upper, and lower field.  If an entered scale exceeds the rather small area available for the picture, it will be reduced automatically.  In the event that the lens contains folds that cause it to extend below the normal picture area, you may enter an optional Y-offset in YOS, in units of inches, which will cause the listing portion to be shifted down to avoid overlap.  A negative YOS moves the listing down.

 

If the optional TSCF is zero or missing, the program will calculate a scale factor that will fit in the width of the page.

 

This feature may also be run from the menus MPL and MLL.

 

If the optional “F” is entered in word 5 of the command, the listing will include the system as well as all surface data, and will add a CAP listing to the end.  Otherwise, only the surface data are printed.  An example DSPEC plot is shown below.

 

 

If the listing is not short enough to fit on a single page, the program will create additional pages for the rest of the data, and will automatically put the GAW option into effect so the additional page(s) do not overwrite the first page.

 

If the lens contains any tilts or decenters out of the meridional plane, or any prisms, the program automatically uses the perspective drawing instead of the Y-Z profile drawing.

 

Here is how we might run this feature on a folded system:

 

FET FOLDS

DSPEC 0 1 99 F –2.5.

 

Here we move the listing down by 2.5 inches to make room for the picture, and request a full spec listing.
