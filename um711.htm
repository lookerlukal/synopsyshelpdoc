7.1.1 Zoom lens drawing (ZDWG, ZPER)                                Next Previous TOC

 

Another version of the DWG program can show all of the zoom positions of a ZFILE zoom lens (see 10.7.1) on one or two pictures. EFILE edges are honored if switch 39 is ON (see 7.8).

 

The command is

 

ZDWG [ TSCF [ JSSS JSPS [ HBAR HBAR HBAR HBAR ] ] ].

 

If the lens has five or fewer zoom positions defined, the drawing allocates five sections; if over five zooms are defined, ten sections are allocated, and the individual drawings will be smaller.  If more than 10 zooms are defined, a second page will be plotted as well.  In this case, you must select the graphics option to make a new window for each picture, so that the second page does not erase the first.

If the lens has tilted or decentered elements, which will not be shown properly by ZDWG, you can use another feature instead. The format is

 

 ZPER EL AZ TSCF JSSS JSPS

 

where the arguments are the same as for the PER feature, with the restriction that only a single line of input is allowed. (If you enter the ZDWG command for a lens with tilts or decenters, the program will automatically substitute a ZPER command.)  If you prefer a Y-Z profile of the elements, without the circular rims provided by PER, be sure to turn on switch 18 before you run this command.  An example of ZPER is shown below.  ZPER will automatically include a set of default rays at three field points.
