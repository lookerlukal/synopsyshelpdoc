14.1 Thermal-Optical-Structural-Program (TOSP)                Previous Next TOC

 

SYNOPSYS can evaluate the effects of certain mechanical and thermal perturbations on an otherwise well-corrected image. The program TOSP is accurate in the limit of small changes and can perform three tasks.  A similar feature, ENM, models changes directly on the system itself.

 

1

TOSP can read a data file created by a thermal or structural analysis program, such as NASTRAN, containing a set of surface deformations or index changes due to thermal gradients at a number of nodes in a specified format.

 

2.

TOSP can create a data file describing the deformations or index changes at many nodes in accordance with the coefficients of a polynomial input by the user, in the specified format.

 

3.

TOSP can create a set of rays, subject to vignetting and apodization, and analyze the change in OPD of each ray as a result of the perturbation file described above.  This set of rays is then available to the IMAGE analysis programs which can evaluate the effects of these purturbations on the image quality.

 

The image evaluation is accurate to first order in the perturbations; the path of each ray in the set is evaluated only for the nominal (unperturbed) design, and then the OPD error is approximated using the formula

 

ΔOPD = δ - ( N'COS(θ) - N COS(ϕ) )

 

where ΔOPD is the change in OPD of the ray, δ is the surface displacement (perpendicular to itself), N and N' are the initial and subsequent indices of refraction across the surface, and θ and ϕ are the incident and refracted ray angles measured from the surface normal.  For index changes, the OPD change is just the index change times the path length of the ray in the medium.

 

Only diffraction-limited images may be analyzed with TOSP, since only the diffraction-theoretic programs DMTF, DMODEL, DPF (and DIFF), PSPRD, and VARIANCE can utilize the OPD data so generated.  The program is called by the input

 

IMAGE

TPF { ICOL / P } HBAR NRYS GBAR

TOSP.

 

The TPF entry specifies the field, color, and number of rays to be traced.

 

Assuming that a data file has been prepared properly (see below), the next input would be

 

GO

 

which will calculate the path perturbation for each ray, and store the OPD data for later analysis.

 

The program will honor all CAO'S, RAO'S, or CAI'S, if any, to determine which rays remain unvignetted, and the ray intensity is automatically apodized if the lens has a gaussian object (OBG).

 

Following this input, the stored pupil data may be inserted into the image analysis with the command

 

GPP       (Get Perturbed Pupil).

 

The desired image analysis command may then follow, and this must be in the same color and at the same field point as that specified on the TPF command.  An informative message is printed to the effect that the previously stored data are being used, and an error message is printed if the image parameters do not match.

 

One characteristic of a TOSP analysis is that the reference sphere for OPD calculations (centered at the principal ray point) does not shift as a result of the wavefront perturbations.  The diffraction analysis will evaluate the image in the usual manner, sampling the image over an area surrounding that point out to a distance of about 5.74 times the Airy disk radius.  But if the wavefront is tilted, the actual image may be centered at a different place.  An analysis such as PSPRD will therefore directly show any lateral motion of the image due to the environment.  If the wavefront is deformed to an amount greater than about two waves, the diffraction programs may not be able to sample the entire image and inaccuracy will result (and a warning message will be displayed).  However, if the wavefront is merely tilted by as much as four waves, the diffraction programs are adequate since the image remains well confined and merely moves from the center of the sample area to nearer the edge.  A plot from PSPRD will in any case illustrate the nature of the image change, and this check is recommended whenever using TOSP.
