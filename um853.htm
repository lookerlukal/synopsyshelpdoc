8.5.3 Geometric MTF (GMTF)                                 Next Previous TOC

 

Most MTF analysis should be performed with the MTF commands described in section 8.6.3, which take the effects of diffraction into account.  The GMTF command in this section finds the MTF by means of a Fourier transform of the spot diagram, ignoring the effects of diffraction, and it is only valid if the image size is large compared to the size of the diffraction pattern.  It can be run interactively from the dialog MFM.  The command-mode input is

 

[ FCO FREQ ]

GMTF { ICOL / M / P } HBAR NRYS  0 GBAR [ P ]

 

where

 

ICOL

is the color number, and the optional M specifies that a multicolor analysis is to be performed using previously entered data (see 8.2). P requests just the primary color.

 

HBAR

specifies the fractional Y object coordinate.

 

NRYS

gives the requested number of rays.  The image to be analyzed by GMTF may contain up to 20,000 rays.

 

0

Placeholder for format consistency.

 

GBAR

fractional X object coordinate

 

P

if entered, causes the MTF to be plotted.

 

The optional FCO input specifies the desired cutoff frequency for the MTF analysis.  If not entered, the program will calculate a default frequency that is a function of the maximum spot diameter.  This default frequency may vary substantially from one image to another, and the FCO input is recommended when requesting GMTFs, and especially when combining MTF's at different wavelengths.

 

The FCO specification is retained until removed or changed.  It may be removed with the command FCO 0.

 

As each ray bundle is traced, the program will print, for reference, the lateral color error between the chief ray in the requested color and that in the primary color.  For each frequency, a modulus and phase is printed for the X-MTF (sagittal) and Y-MTF (tangential).  The phase is the lateral shift of that frequency, in degrees, relative to the primary color chief ray intercept.  A positive phase indicates a positive shift.

 

The GMTF feature is a Fourier-transform type of MTF, and is also found in dialog MFM.   A multi-field GMTF feature is also available; this is the MGMTF, also found in dialog MMG.  

 

If switch 87 is on, the MTF plots will show grid lines in addition to the usual fiducials.
