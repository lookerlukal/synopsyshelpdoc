15.1.2.2 The AlParameter                                                        Previous Next TOC

 

A useful way to alter lens data, as the Do MACro loop executes, is with a special variable called the AlParameter.  This may be varied in the same way as any lens data, but it does not automatically go into the lens.  Instead, it is formatted and goes into the symbol “AIP".  If this symbol occurs in the MACro, the current loop value will be substituted. (See section 15.1.3 for a discussion of symbol substitution.)

 

For example, if the MACro contains

 

IMAGE

FOCUS 2 AIP 300

ORDINATE BACK,

 

then the Al sentence

 

Do MACro for AIP = 0 to 1.

 

will produce a plot of back focus versus the AIParameter.  In this case, the plot gives the shape of the focal surface as a function of field, with the spot size minimized at each point.  The AIP symbol in the FOCUS command replaces the HBAR entry, giving the fractional field at each data point.

 

The AIParameter may also simply cause a MACro to loop a given number of times (given by the STEPS value + 1), even if a plot is not the object.

 

Switch 45 is automatically turned on if the AIPparameter is involved in a loop, anticipating substitution within data.
