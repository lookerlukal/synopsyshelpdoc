6.3.1 Targetted Raytrace (TARGET)                                Next Previous TOC

 

SYNOPSYS can find the ray that intercepts a given surface at a specified location.  The input is

 

TARGET { ICOL / P } HBAR { X / CAO } { Y / CAO } SN GBAR [ SURF ].

 

{ X / CAO }

desired coordinates of the ray on surface SN. CAO requests the value given on the CAP listing.

 

{ Y / CAO }

same as above, in the Y-direction.

 

[ SURF ]

causes a surface-by-surface listing of the ray path.

 

TARGET ignores CAO's, but will indicate if the ray fails to trace or encounters feathered edges between surfaces not tilted or decentered.  It can be run interactively from the dialog MRR.
