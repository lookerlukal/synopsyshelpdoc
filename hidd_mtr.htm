Lens transmission calculation:

 

The dialog MTR shows several features of SYNOPSYS that can calculate the transmission of the system, each in a different way and each considering a different set of effects.  These are:

 

1. PRAY, which only works if the lens is in polarization mode, and calculates the polarization vectors at the image and optionally at each surface.  This models coatings only, finding reflection losses.

 

2. RTRANS, which traces the requested ray and calculates the amount of absorption along the path, ignoring other losses.

 

3. TRACE, which traces a grid of rays and reports the fraction of the rays entering the pupil that reach the image.  It models vignetting and apodization only.

 

4. PTRACE, which is the same as TRACE except it also models reflection losses at coatings.  The lens must be in polarization mode.

 

5. TN, which models all effects, excluding the obliquity of the beam at the image (which has no effect on the T/number).

 

6. ILLUM, which includes all effects, including obliquity.

 

7. GTRANS, which calculates the absorption of a single element.

 

More information about these features is given in Section 5.39.
