4.3 Lens File Output (LEO)                        Previous Next TOC

 

The command

 

 LEO

 

causes the lens data to be formatted as an RLE file and printed.  This file contains all of the data needed to reconstruct the lens, and is useful for documentation.  This feature is also found in the MLL dialog.
