SketchPAD – Select Display Type Dialog

 

In this dialog you can select one of several kinds of graphical displays to show on either the top or bottom of the split-screen SketchPAD display.  Once you have selected the display types, SYNOPSYS remembers your selection, and will automatically update the screen with that combination when you later reopen the SketchPAD feature by typing PAD, clicking on the button , or during optimization if you have selected the SNAPshot option.

 

Sometimes you want just a single SketchPAD display; then you can click on ‘Blank Section’, to erase the other part of the display, and use the Pan button to put the drawing where you want it and zoom in with the mouse wheel.  If the part you want to see is the top display, you can also just select the Solo top display box.  Then the top will be shown enlarged and there will be no bottom display.

 

You can define up to five different display configurations in PAD.  Click on one of the buttons , and then select the display types you want.  Whenever you click on that button again, you restore that display configuration.

 

Your selection of PAD options is saved with the command SSU (Save SetUp), and can be restored at a later time, even after exiting the program, with the command RSU (Restore SetUp).  Those commands also save all the lenses in the ACONs.  The commands SPL (Save PAD Layout) and RPL (Restore PAD Layout) save and restore just the PAD settings, but not the lenses themselves.

 

You can show a selected portion of your lens by entering the surface numbers in the data boxes for profile, perspective, or paraxial drawings.  If you later wish to show the whole lens, just click the button Draw all surfaces.  You can quickly increment or decrement the last surface number to be displayed on the upper display pane with the toolbar buttons .

 

This dialog also presents an option to link the PAD display to a command string.  If you check this box, you can enter any valid command string of up to 89 characters in the edit box, and that string will then be executed whenever the PAD display is updated.  This option lets you view and monitor properties of the lens that are not available in the above selections.  Thus, to view the multicolor MTF curve at full field at a cutoff of 100 c/mm, one would select the “Execute command link” box and then enter the string

 

 FCO 100 / MTF P 1 0 0 0 P

 

in the data box.  Then you could monitor how the MTF changes as you adjust a lens thickness with the WorkSheet sliders.  (This option is not saved with the SSU or SPL commands, and if selected, it automatically activates the option to reuse the graphics window for subsequent pictures.)  Each of the five displays can have its own command string.  For more information, click here.
