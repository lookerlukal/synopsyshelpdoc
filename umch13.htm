Chapter 13

 

13.0 INTERACTIVE FEATURES

 

Although SYNOPSYS retains nearly all of the command-mode features that for long have made it one of the most powerful optics codes in the world, recent development of fast interactive PC systems have added a host of possibilities for direct user interaction.  New features that employ these new possibilities include the rotating-perspective drawing that can be manipulated in real time on the monitor screen and give an immediate 3-D visual impression of the folding and reflections of complex layouts – along with the WorkSheet feature that lets you slide an element along the axis while watching the rays converge or diverge, or change the bending or power of an element while watching the effect on the image, lens drawing, and raypaths in real time.  All of the interactive features described in this chapter can be run from various dialog windows that contain labeled boxes into which you may enter your data.

 

Command-mode commands are entered into the Command Window, which acts like an Interactive DOS terminal in which SYNOPSYS displays its output.  You may create up to 10 instances of this window at any time, and the program input and output occur in the window that was most recently activated.  (Just click in a window to activate it.)  Online instructions are given in the help file: when the Command Window is active, press F1, or click on the menu item “Help” and then select “Index” and click on “Command Window”.  You will be directed to numerous other menu items and toolbar buttons which you may click to create new instances of the Command Window, to erase the contents, or to cycle the font size to your liking.  At least one instance of the Command Window must be on the screen at all times; if not, you will be prompted to create one.

 

Additional suggestions for utilizing the Command Window can be found in the Tutorial Manual.
