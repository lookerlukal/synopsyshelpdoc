13.7 The SYNOPSYS menu dialog tree                                Previous Next TOC

 

Most of the features of SYNOPSYS can be run interactively with a set of menu dialogs that contain data boxes with labels describing the contents.  These menus are arranged in a tree structure that starts with the dialog MMM (Main Menu Mode).  You can navigate this dialog tree by clicking on the menu item “EZ Menus”, and following the arrows until you get to the branch you want.  Or you can simply type the command of that branch, selected from the list below.

 

These menus will prepare and submit the SYNOPSYS commands required to perform the selected task.  It is sometimes helpful to be able to capture this sequence of commands in the form of a MACro, since it can then be saved as a named file and rerun without requiring that you again format all of the data boxes in the dialog.  To do so, type the command

 

 LMM        (Load Menu into MACro)

 

after the menu has executed.  (Or use the form “EM LASTMENU”)
