SpreadSheet (SPS) -- Editing Surface Data

 

The SYNOPSYS SPS spreadsheet is a convenient place to edit your lens surface data.  To edit lens system data, such as the object definition, use the command SYS, or click on the ‘System Data’ button.  Data applying to the object and the definition of the entrance pupil may be edited most conveniently with the button provided to go to the Object or Pupil Wizard, or the commands MOW or MPW.

 

Surface data can be directly edited in any of the spreadsheet boxes that do not have a cyan colored background.  Those that do indicate numbers whose value was derived from some other specification, and you must remove that specification before you will be allowed to enter a different number yourself.  This is done by clicking on the column labeled ‘Flags’, to the left of the data box you want to change.  Then you will get a dialog tree that lets you specify which of the many options you want to assign to that parameter.  The leftmost column, labeled ‘Surface Flags’, is used to set many options that apply to the surface as a whole.

 

If the lens has more than 50 surfaces, they will be displayed in groups that may be selected by clicking on buttons along the left margin of the dialog.

 

Separate pages are provided for editing clear aperture data as well as tilt and decenter data.

 

The easiest way to insert or remove surfaces is with the buttons , , and .  Those buttons will insert a surface at the current line, delete the current surface, or truncate the lens at that surface, deleting all higher-numbered surfaces.   If you click in a line below the last surface in the lens and then click the button, all surfaces between the lens and where you clicked will be added.

 

You can also change the number of surfaces in the lens with the button Add or remove surfaces.  There you may add or remove surfaces within the lens or add surfaces to the end.  You can also use either the command-mode options DELETE and INSERT, or the WorkSheet to do the same thing.

 

If you are entering a lens from scratch with the SPS editor, it is better to enter the index data first, before the radii and thicknesses are entered.  This is because most lenses will have paraxial solves here and there, and these become singular if there is no index difference at the time the solve is requested.

 

Moving around in the spreadsheet is very easy.  Just click on a number you want to edit, and type the new value.  Before the program can use the new value, it has to know that you are finished typing.  You can indicate this by pressing the Return key or by clicking in any other data, or by pressing any of the arrow keys.  In the latter case, the program will process the new data and then automatically open for editing the nearest data box found in the requested direction, if any.

 

Several data types have a flag window next to the numeric data.  Clicking there will open a dialog where you can select from a variety of options available for that data type.  You can also open those dialogs by clicking on any surface data box (to identify the surface) and then on one of the buttons .

 

A special feature applies only to glass names taken from one of the glass tables.  If you right-click on the name, you will get a list of the contents of that catalog showing a variety of properties, and you may select a different glass by double-clicking on the name or by selecting that name and clicking the OK button.

 

As is the case in all other features of SYNOPSYS, you can undo any typing you may have done by pressing Ctrl+Z while an edit box is open for editing.  To undo changes to the lens as a whole, use Ctrl+U or F3 when there is no box open for editing.  To redo a change after undoing it, use Ctrl+Y.  This feature will remember your changes for a total of 10 undo levels.

 

The SPS dialog displays a small box in which the lens profile or perspective drawing is drawn, for your convenience.  You can move this box wherever you like, and when you next open SPS it will automatically go to the same place.  In the event you want to restore the default position (when you change monitor resolution, for example) use the command DWL (Default Window Location).

 

Note that, unlike the case with other programs, the spreadsheet is not supposed to be open all the time.  Use it when appropriate, and then close it so you have ready access to the other powerful tools in SYNOPSYS.
