MOM – Standard Raygrid Selections

 

Use this dialog to create a very simple merit function based on grids of rays that will usually give a good image.

 

The dialog is largely self-explanatory; click on a “USE” box, and then select the grid pattern you want, which field points you want, and which colors to trace.

 

You also have a choice of correcting transverse aberrations or the optical path difference (OPD) of each ray – or the wavefront variance of the whole set.  The latter is valid only if you select a pattern with a rectangular grid of rays.  If you choose to optimize for high contrast, the program will weight all rays in the pupil equally, while for high resolution it will weight the central zones more heavily than the outer.

 

You can reach this dialog from MOM or from the MACro editor.  If you click on the toolbar button in the editor, the raysets that you define in this dialog will automatically be pasted into the MACro at the present cursor location.

 

Note that if you later go back to this dialog, with the previous raysets already in your MACro, the new raysets are added to whatever was there and you can easily get two sets of the same data.  So be sure to either delete the previous lines first, or select entirely new raysets later.
