10.3.1 Ray Aberrations, Settings                                                Next Previous TOC

 

The set of rays to be traced by SYNOPSYS may be entered explicitly by the user, generated automatically by the program, or both.  Automatic raysets ignore vignetting, but honor the VSET parameter (see 10.3.1.1), to model a vignetted pupil, and also the VFIELD array, if one is in effect.

 

Additional parameters may be entered to alter the lens when evaluating ray aberrations that are defined later.  (All of these are put into the AANT file.)  The first is

 

 DIOPTERS D

 

This parameter only works when the lens is in AFOCAL mode.  It changes the accommodation setting of the lens to the entered value of D.  The lens remains at that setting until the user changes it.  It is wise therefore to add a final DIOPTER D line at the end of the AANT file if you always want the system to go back to an original setting when the optimization is finished.

 

 ZOOM NB

 

will adjust the current zoom setting to zoom number NB.  This only works on ZFILE zoom lenses.

 

The input

 

 BIREF SN { ORD / EXTR }

 

will set the index selection of the material at surface SN  to the selected state.  Rays traced following this input will therefore see that index.  The material must be a BIREF before you can use this input.

 

The input

 

 POLAR { X / Y / CL / CR }

 

will set the polarization state of the system to linear-X, linear-Y, circular-X or circular-Y.  Polarization mode must be on before you can use this input.

 

These changes are made to the lens whenever the aberration array is evaluated.  The lens remains with those settings thereafter, except when you request EVAL instead of SYNOPSYS; in that case the lens is restored to the initial state afterward.  This means that on the next optimization evaluation, you may not start with the original settings.  It is wise to initialize them explicitly, at the start of the AANT file, if this is the case.
