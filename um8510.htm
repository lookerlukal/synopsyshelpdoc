8.5.10 Multi-field Geometric MTF (MGMTF)                                 Next Previous TOC

 

The geometric MTF may be obtained, and optionally plotted, at up to four field points from the dialog MMG, or up to 20 with the command-mode input

 

FCO FREQUENCY

MGMTF NRYS

 

[ HBAR HBAR HBAR HBAR HBAR ... ]

[ HGEN NHBAR ]

 

ICOL { ICOL / M / P }

[ GBAR GBAR ]

{ PLOT / GO }

 

Default fields are 0, .5, .75, and 1.0, when no HBAR entry is given.  Up to 20 fields may be given on the HBAR line, and HGEN will generate NHBAR fields evenly spaced from 0 to 1,

 

A cutoff frequency must be entered for this option, with the FCO input.
