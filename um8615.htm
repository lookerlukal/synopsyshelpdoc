8.6.15 Over Spectrum PSPRD (OSP)                        Next Previous TOC

 

Color correction in a lens is seldom perfect.  When the lens is as good as it seems to get, you might wonder if that is good enough.  The OSPSPRD feature can tell you.  This feature also can be accessed via the menu MPS.

 

The command-mode form is as follows:

 

[ PSVISUAL [ MAG GAIN ] ]

 

OSPSPRD NRYS HT [ R / L ]

COLORS NCOLS

FIELD HBAR GBAR

PLOT

 

These arguments are explained in more detail, should you require it, at the above link.
