SYS – Spreadsheet Edit System Data

 

The SYS dialog is where you can edit the data that apply to the lens system as a whole.  The options here let you specify the object distance and size, which of several pupil geometry options you want, and numerous other optional settings.  For a complete list of the command-mode system options, see section 3.1 of the User’s Manual.  To edit data that apply to a particular surface, click on the button “Surface Data”, or use the command SPS.

 

A simpler way to edit some of these data is to use the dialogs MCC or MSW for entering wavelengths, and MOW or MPW for defining the pupil and stop.
