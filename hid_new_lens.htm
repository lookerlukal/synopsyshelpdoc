New Lens command (File menu)

Use this command to delete the current lens and start the SYS editor. You can also use the SYNOPSYS command NLE, which will create a minimal system (but will not launch the editor).

Experienced users will prefer to enter a properly formatted lens file (see RLE) in the MACro editor.  This is quicker and gives a permanent record of your input.

You can open an existing lens file with the Open command, or the commands MWL or FETCH filename
