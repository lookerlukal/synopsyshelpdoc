8.6.3.3 Multifield MTF plots (MMF)                         Next Previous TOC

 

The convolution MTF may be obtained, and optionally plotted, at up to four field points from the dialog MMF, or with the command-mode input

 

       MFF                        (exact-ray method)

or        MZMTF [ GRID ]        (Interpolation method)

 

[ HBAR HBAR HBAR HBAR HBAR  ... ]

[ HGEN NHBAR ]

 

       ICOL { ICOL / M / P }

       [ GBAR GBAR ]

       { PLOT / GO }

 

Default fields are 0, .5, .75, and 1.0, when no HBAR entry is given.  The HGEN entry creates an array of NHBAR field points, evenly spaced from 0 to 1.0.  Up to 20 field points may be specified on the HBAR line or with NHBAR.

 

The MZMTF option uses the wavefront interpolation method, which will be a linear spline fit to the wavefront  These cases can take the optional [ GRID ] input (default is 80 – see 8.6.3.2).

 

If the lens is poorly corrected and you want to examine the MTF at low frequencies, you will get better accuracy with the MZMTF option..

The default cutoff frequency is the natural cutoff at the F/number of the lens and the requested wavelength (or the primary color in the case of a multicolor analysis).  This calculation (and the perfect MTF curves that are plotted for reference) is performed at the first of the field points requested.  You may enter a desired cutoff with the command

 

FCO FREQUENCY

 

prior to the MTF command.

 

This option may be accessed by the menu MMF.
