Adjust GLM boundaries

 

This dialog contains four slider bars that you can adjust to change the boundaries that will apply to glass model (GLM) variables.  When you have positioned the boundary lines the way you want, the program automatically honors those boundaries.

 

When you change the boundaries, the data box is updated with parameters that correspond to the new settings.  You can copy these data and paste them into your optimization MACro (in the PANT section) if you want them always to apply to those variables in that MACro.  Otherwise, your settings are lost when you restart the program.
