15.1.5.2 Inserting and Deleting a Given Lens With AI                        Previous Next TOC

 

Once the choices have been displayed, you may select one to insert into your lens with

 

INSERT CHOICE NB [ BEFORE / AFTER ) ( SURFACE / ELEMENT } NB

 

Example: INSERT CHOICE 6 AFTER SURFACE 11.

 

Keep in mind that these operations make major changes in the lens, and can easily produce ray failures unless used with care.

 

Other uses of the INSERT verb are of the form

 

INSERT vendor 'lensname' { BEFORE / AFTER } { ELEMENT / SURFACE } NB

 

where you give an explicit part number, taken from the catalog, and placed in single or double quotes, such as

 

INSERT EDMUND '32011' BEFORE ELEMENT 4

or

 

INSERT type with condition [and condition] ... { BEFORE / AFTER }

                                                        { ELEMENT / SURFACE } NB

 

where type is selected from SINGLET or DOUBLET, and condition may be selected from

 

FOCL

C1

R1

C2

R2

BENDING

EFL

 

For example:

 

INSERT DOUBLET WITH EFL = 50 AND BENDING = 0.25 AFTER ELEMENT 1

 

With this form you can specify a curvature or radius on surfaces 1 and 2, or give the focal length of the element you want.

 

This use of the INSERT verb performs a quick optimization to define the lens specifications, and does not draw from a vendor catalog.

 

INSERT can also make copies of an element already in a lens:

 

INSERT { ELEMENT / SURFACE } NB [ TO NB ] { BEFORE / AFTER }

{ ELEMENT / SURFACE } NB

 

For example: INSERT ELEMENT 5 BEFORE ELEMENT 2.  (Also, see MOVE, in 15.1.5.4.)

 

INSERT can insert a specified range of surfaces or elements from a lens file that has previously been saved:

 

INSERT FILE 'filename' { ELEMENT / SURFACE } NB [ TO NB ] { BEFORE / AFTER }

{ ELEMENT / SURFACE } NB

 

Example:        INSERT FILE 'RELAY.RLE' ELEMENT 1 TO 5 AFTER ELEMENT 4.

 

Al can also delete a lens element:

 

DELETE ELEMENT 3

 

The WorkSheet program can also insert and delete elements, with immediate graphical feedback.
