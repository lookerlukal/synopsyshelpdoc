13.3.3 The LE Lens editor                                                Previous Next TOC

 

The LE editor is really a dedicated MACro window in which SYNOPSYS puts a copy of the LEO file, which is an RLE-format file that describes all of the lens data.  You can start a session with LE by typing the command

 

LE

 

or by clicking on the LE Lens Edit button in PAD, or by clicking on the same button in the SideBar (although this depends on the current SideBar selection setting).  If you launched LE with the PAD button, the display is automatically updated when you run the LensEdit MACro; otherwise you have to click in the PAD window to see the changes.

 

The command

 

LEP

 

Opens the LE editor and then automatically updates the PAD display when the file is executed.
