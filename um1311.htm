13.1.1 Instant Help                                                                Previous Next TOC

 

Most of the SYNOPSYS commands that require some form of data will prompt with a display of the data format if you type the command without anything else.  Thus, if you remember that the command giving a plotted diffraction point-spread plot is “PSPRD”, you can see the format by typing

 

PSPRD

 

The program will then display

 

 FORMAT: PSPRD {ICOL/M/P} HBAR NRYS 0 GBAR [HT [L/R] ]

 

at which point you can reenter your command with the correct arguments.  Of course, commands with optional arguments, or which do not require arguments, such as DWG, do not respond this way – but you can quickly get to the help section with input of the form

 

HELP DWG

 

As you type, either in the Command Window or into a MACro editor, the program watches your keystrokes, and as soon as it looks like you are typing a valid SYNOPSYS command, it will display the format of that command in the tray, which is the thin border at the bottom of the screen.  See Section 13.20 for more information on this handy feature.
