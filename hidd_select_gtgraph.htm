Select GlassTable Graph

 

The GlassMap feature can indicate several important attributes of commercial glass types, such as stain resistance, bubble class, relative price, etc.  Simply select the data you want shown on the glass map, which will then be drawn by a vertical line whose length is a measure of that quantity relative to other glasses on the chart.

 

To use this feature, go to the SketchPAD display and click on , choose a vendor name from the list shown, and when the glass map is drawn, click on the button ‘Graph’.  You can type the command MGT to go to the glass map directly.

 

Another selection on the graph-selection dialog will display the index and effective V-number of all glasses at the current wavelengths.  This lets you see the dispersion values in spectral regions other than the CDF lines, and can help you identify materials that may be useful in color correction.  This is the selection “Plot N, V at current wavelengths”.

 

A third option, “Plot P(F,e) vs. Ve” will show a display that can be of use when designing apochromatic objectives. A description of the use of this feature will be found at www.osdoptics.com/APO_LESSON.htm..

 

Yet another option does much the same thing as the above, but is more useful when the wavelengths of the lens are not in the CDF visual range.  This option is called “Plot P(N3,N2) vs. V(N2)”.  In this case, the Y-coordinate of each glass is the quantity (N3 – N2)/(N3 – N1), and the X coordinate is (N2 – 1.0)/(N3 – N1), where N2 is the index at the first wavelength in the color order (the primary color), and N1 and N3 are the index values at the long and short wavelengths. This display is used very much as described in the previous link: in short, to correct secondary color one must select at least three materials whose positions on the graph do not lie on a straight line.

 

A fifth option displays thermal properties of the glasses, and can be used to athermalize a lens in much the same way that one corrects for chromatic aberration.  The theory behind this can be found in the article by Tamagawa, Yasuhisa, et. al., “Multilens system design with an athermal chart”, Applied Optics Vol. 33, No. 34, 8009-8013 (1 December 1994).  See below for an example.

 

Option number 6 shows a plot of the partials P* against P**.  This graph is used in designing a “superachromat”, according to the theory of Max Herzberger.  This is a lens which has a common focal point for four different wavelengths, and it is the best color correction you can obtain.  (See Applied Optics, Vol. 2, No. 6, page 553, to read the original theory.)  A description of the use of this feature will be found in Section 5.51, which also describes a command-mode feature useful for the same purpose.  The partials are defined as

 

P* = (Nf - N*)/(Nf - Nc)

 

P** = (Nf - N**)/(Nf - Nc)

 

where N* is the index at 1.014 um and N** is at .365 um.

 

The principle is simple: select three glass types that lie on a straight line on that plot and you can correct all defined colors in the lens.  The program displays a straight line on the plot, and you can change the end points as you look for a line connecting appropriate glasses.  <Ctrl><left-click> will define the lower point, and <Shift><left-click> will define the upper.  You also have to be sure that the dispersions of the selected glasses are as different as possible – to keep element powers low – and the program can draw the V-number of each glass on the plot as a short red vertical line, so you can see what is where.  To select this view, click the Abbe number radio button on the Select Graph dialog in the Special section.  A description of the use of this feature will be found at http://www.osdoptics.com/Lesson%2028.htm

(To access this feature, open the glass map with MGT, select a catalog, click Graph, and select Plot thermal properties.)

The program has added a green symbol over on the right; the coordinates are at 0 on the X-axis and 0.23e-6 on the Y-axis, which is the declared CTE of the lens housing, entered with the ALPHA property in the RLE file.  (This value applies to aluminum 6061.)  When we click on the glass SF-57, the program draws a line between that glass symbol and the green symbol, and any other glass that lies close to that line is a good candidate for athermalization.  Clicking on a different glass will draw a line through that point instead, and it is easy to find pairs that lie close to the line.  Glass type N-PSK53A appears to be a good one to match with SF-57.

 

The theory says we can athermalize at the same time that we achromatize the lens if we use those two glasses and put them into an aluminum housing.  The resulting lens is indeed almost perfectly in focus at 20 and at 100 degrees C.  The RLE file for this lens is below.

For more complicated situations, where one must use more than one housing material or widely-spaced lens groups, the THERM features are of course recommended.  Indeed, one can shadow a given design at more than one temperature and correct the image at all shadowed temperatures at the same time.  The process described above can be viewed as an attractive starting point in that case.  One can also imagine a lens housing made of more than a single material, and deduce an effective CTE from the relative proportions.  Then enter that value for ALPHA, and the thermal graph will apply to that combination.
