Command Window

 

SYNOPSYS users interact with the program in two ways: via mouse clicks on menu items or toolbar buttons, and by typing SYNOPSYS commands and AI sentences in the Command Window.

 

The Command Window maintains a buffer that can hold about 65,000 characters, and this is where your listings and printed output will appear.  When the buffer becomes full, the program will automatically delete the top ¼ and continue scrolling at the bottom, so it looks like a DOS window that scrolls forever.  This behavior is controlled by a user selection, and you can choose a different option if that is what you prefer.

 

To do this, go to Options | Command window, and be sure the option Recycle Command Window is not selected.  SYNOPSYS will then display a dialog when the buffer becomes full, which will let you

 

1. Clear the current window

2. Create a new, empty window.

 

You can also clear the current window at any time by clicking on or the menu item Window | Clear Command Window or with the SYNOPSYS command CCW (Clear command Window).

 

If the option Recycle Command Window is selected, the Command Window will manage its buffer automatically, as noted above.

 

You can create a new, empty Command Window by clicking on or selecting the menu item WINDOW | New Command Window, or by typing Ctrl+N or the SYNOPSYS command NCW.
