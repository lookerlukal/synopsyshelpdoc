12.1.3.2 BTOL Spot size quality descriptor                                Next Previous TOC

 

You may select the geometric spot size variance with the following input:

 

TOL [X/Y]SPOT VARIANCE

TOL SPF VAR VAR VAR …

DEGRADE SPOT FRACTION

DEGRADE SPF VAR VAR …

 

TOL SPOT

gives the maximum permitted value (not the change of value) of the spot variance, which is just the square of the standard deviation of the spot.  If this input is entered, the maximum value permitted by the tolerance budget will be the same for all field points unless the FWT entry has specified weightings for each point.  If this is the case the maximum value for each point is increased by 1 / WT**2, which corresponds to an increase in the RMS spot size for each field inversely proportional to the weight on that field point.  For example, if you wish the full-field point to have twice the size tolerance of the on-axis Point, you would enter a weight of .5 for that point.

 

The standard deviation is a useful measure of the spot size.  It is the same as the RMS spot size if one measures each spot from the centroid of the pattern.  To insure that this measure will not exceed a desired value, simply square that value and enter the result for the quantity VARIANCE.

 

If the image is sharp in one direction only, as in a line image for a scanner, you may tolerance the spot size in a single direction, using the mnemonics XSPOT and YSPOT.  The former will ignore the Y-dimension of a spot, and the latter the X.

 

TOL SPF...

is the same as TOL SPOT except that a desired tolerance is given for each field point, in the order described under FOV, above.

 

 

DEGRADE SPOT

 

 

specifies that the maximum permissible value of the spot variance is to be a specified fraction greater than the initial value.  This input is appropriate for cases in which you want a lens to degrade by, say, not more than 20% of its nominal value, but do not want to specify the actual tolerance yourself.  For example, if you enter a value of 0.5, the variance will be permitted to degrade to a value of 1.5 times the initial value, which corresponds to an increase in the RMS spot size of the square root of this number, or 1.2247 times the initial value.  The calculation is performed at each field point, with the point with the best initial image being assigned the tightest tolerances.

 

The resulting tolerances will be altered by the entry of FWT data, as was the case of TOL SPOT.

 

DEGRADE SPF

gives the allowed degradation to the spot size at each field point.
