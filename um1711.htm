17.1.1 Object polarization                                                        Next Previous TOC

 

To use any of the polarization features, the lens must be placed in polarization mode, using one of the following entries in the lens RLE or CHG file:

 

 

POLARIZATION OFF

POLARIZATION UNPOLARIZED

POLARIZATION LINEAR { X / Y / ANG }

POLARIZATION CIRCULAR { RIGHT / LEFT }

POLARIZATION ELLIP { RIGHT / LEFT } ( A / B ) ANG

 

To turn polarization mode off, enter

 

POLARIZATION OFF.

 

In a CHG file.  Note that UNPOLARIZED is very different from OFF: the former is treated by tracing two rays, of orthogonal polarization, and adding the results incoherently, while the latter simply ignores polarization and coatings.  The default is OFF.

 

The quantity ( A / B ) entered for elliptical polarization refers to the ratio of the semi-minor axis (A) and semi-major axis (B) of the polarization ellipse, as shown below, on the left.

 

 

The sketch on the right shows a ray as it intercepts a surface.  The surface is in general curved, and the normal vector is shown drawn to the left.  The plane containing the ray and the normal vector is the plane of incidence.  The current E-vector(s) of the ray may be resolved into two components; one is in this plane, and is called the P-vector, the other is normal to it (and is in the plane of the surface) and is called the S-vector.  The effect of the surface coating on these two vectors is a function of the obliquity angle.

 

The polarization mode may be changed during optimization, if you need to evaluate the same system in more than one state.  See Section 10.31.
