15.3.6 Optimizing the selected match                                        Previous Next TOC

 

If you enter Y, XSYS creates an optimization file in accordance with the targets and weights specified for the matching process, and submits the file to the optimizing program.  This is the only form of optimization in SYNOPSYS where you have no direct control over the input.  The merit function is generated by the program, and its purpose is not to produce a final design, but merely to correct those features of the lens that can easily be corrected so you may better judge that particular match.  The number of iterations is entered with the PASS input (default = 10).

 

When the requested number of iterations has been performed XSYS prompts for the next decision.  Note that the optimization process will be accompanied by SNAPshots every few passes, if SNAP has been requested with a pass count (as SNAP 2 (default is every pass)), and that if you click on the button during the optimization, it will terminate at the end of the current iteration.

 

The choices at that point, which are prompted by XSYS, are:

 

K

to keep this lens and exit from XSYS

 

<CR>

to discard the lens and try the next match

 

L

to see the match list again

 

NB

to try that match

 

STORE NB

to save this lens and make another choice.

 

If you exit from XSYS following the quick optimization, the default optimization input will be found in the MACro XSYSFILE.MAC.  In the event that the optimization was going well, but some property of the lens was not under control, you can then revise the merit function and resubmit it for additional optimization.

 

If you have exited from XSYS for some reason, and then decide to go back and try another match, it is still possible, provided that library location 10 has not been written to.  If switch 42 is ON, many other features will write to location 10, and you may not be able to reenter.  Otherwise, simply enter XSYS and then type REENTER.  If location 10 has been written to, XSYS will be unable to properly construct the combination lenses, and must be rerun from the beginning.
