SideBar

A toolbar, displayed along the left side of the main SYNOPSYS window, provides quick mouse access to many commands and features of SYNOPSYS.

 

  Click        To

Open a dialog where you can determine what clicking on these buttons does. You will have a choice: The button can perform any action that you have previously defined by means of the AI symbol-substitution feature. Or it can open a dialog where you have a choice of SYNOPSYS features to choose from.

 

The AI symbol-substitution feature works like this: suppose you want the toolbar button labeled “I03” in the dialog to produce a SPEC listing.  You could define the symbol by typing

IO3: SPEC

 

at the command prompt.  Then clicking on that button would send the SPEC command to SYNOPSYS and produce your listing.  The other option goes to the MLL menu where you can select from several lens listing formats, including SPEC, PRT, etc.

 

An easy way to program all of the AI symbols that you need is to click on this button, which starts the MACro editor with a copy of the file ICONS.MAC. Use this file when you change definitions if you want the new ones to come back the next time you start SYNOPSYS.  Put the command EM ICONS in your SYSTART.MAC file in order to automatically run the icon MACro when you start up.  You can also put the definitions in your own CUSTOM.MAC file.

 

An extra set of buttons is shown at the bottom of the Sidebar that you can program to do anything you want.  These are initially undefined, but suppose you want to run a MACro named MYMACRO when you click the button “U1”.  Just define the symbol

 

U1: EM MYMACRO

 

either in the Command Window or in your Custom MACro.  Then whenever you click this button, your MACro will be executed.
