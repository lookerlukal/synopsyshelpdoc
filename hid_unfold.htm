WorkSheet: Unfold

 

After clicking this button , the program will delete the fold mirror, and its attached dummy surface, where you click next.  These two surfaces must both be flat, the first a reflector with zero thickness, and the second a dummy (with no index change).  Both must have identical relative tilts.  If you click anywhere else, this tool will display an error box.

 

You can also unfold a system in command mode.  See UNFOLD.
