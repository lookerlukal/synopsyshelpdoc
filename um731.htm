7.3.1 Short-form input                                        Next Previous TOC

 

(See 7.3.2 for the long form of this feature). Plotted transverse intercept plots are obtained with the command

 

 { RPT / RFT } SCF NRYS { ICOL / P / M } [ ICOL … ]

 

Plotted OPD fans use the mnemonics

 

 { RPO / RFO } SCF NRYS { ICOL / P / M } [ ICOL … ]

 

The mnemonics RPT and RPO give the symmetric-lens version, while RFT and RFO the full-SFAN version.  The arguments of this input are explained below.

 

SCF

is the image dimension at full scale on the plot

 

NRYS

is the number of rays per SFAN (maximum 100).

 

ICOL …

are color numbers to be analyzed. If “M” is entered for the first color, all of the defined wavelengths will be traced, giving a multicolor analysis.  “P” specifies the primary color.

 

This short form of input will give fans plots for the entered colors at field points 0., 0.5, 0.75, and 1.0.  A SCF of zero on any of these commands will give a default scale factor, determined by sampling the rays at several zones.

 

Rays that encounter clear aperture violations, edge feathering, or trace failure will be deleted from the plot and will be identified in the printout unless you have input the optional NOVIG in the RLE file.  In the latter case, only ray failures will be indicated.  The vignetting message may suppressed (but not the vignetting check) by placing switch 33 ON.

 

If switch 59 is on, pupil reduction is shown when the pupil is reduced in size by any of the pupil options, even though you may have selected an option that normally shows the reduced pupil covering the full axis (such as the VFIELD and WAP options).  In this case, the abscissa of each plot gives the full pupil size, rather than the vignetted size, and the TFAN extends over only a portion of the whole axis if the pupil has been reduced.  This is a useful way to document the extent of vignetting in your lens.

 

Fans plots will be shown in color if switch 35 is ON.  The displayed colors come from a built-in color table, and will not necessarily be the same color as the wavelengths analyzed.  To display the curves in approximately the visual colors, turn on switch 85 as well as 35 before requesting the plot.  In this case, the curves are shown as solid lines rather than with a dot pattern, since the latter is then not necessary for identifying the curves.
