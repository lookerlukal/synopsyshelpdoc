12.1.3.1 BTOL Wavefront quality descriptor                                Next Previous TOC

 

You may select the wavefront variance with the following input:

 

TOL WAVE VARIANCE

TOL WAF VAR VAR VAR …

DEGRADE WAVE FRACTION

DEGRADE WAF VAR VAR …

 

TOL WAVE

gives the maximum permissible value of the wavefront variance.  This input may be used if the lens is essentially diffraction limited, and it provides a way to control the wavefront.  If this input is entered, the maximum value permitted by the tolerance budget will be the same for all field points unless the FWT entry has specified weightings for each point.  If this is the case the maximum value for each point is increased by 1 / WT**2, which corresponds to an increase in the wavefront standard deviation for each field inversely proportional to the weight on that field point.

 

Additional comments regarding the wavefront calculation and use of these quality descriptors are given in section 12.2, which should be read by all users.

 

TOL WAF...

is the same as TOL WAV except that a desired tolerance is given for each field point, in the order described under FOV, above.

 

DEGRADE WAVE

 

specifies the fractional change permitted in the wavefront variance.  The default descriptor is DEGRADE WAVE .1, (which gives very tight tolerances).

 

 

DEGRADE WAF

gives the allowed degradation to the wavefront variance at each field point.
