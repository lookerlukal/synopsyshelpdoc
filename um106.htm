10.6 Examples of Lens Design Input                                Next Previous TOC

 

 

1. Assume the above lens has been entered, and no pickups or solves are in effect.  We wish to produce an F/2 lens corrected on axis and at full field.  We do not require the lens to be at the paraxial focus.  A suitable input file is shown below.

 

PANT P

VLIST RAD 1 2 3 4 5 6

VLIST TH 2 4 6

VY 1 GBC                (The glasses will be pinned against

VY 3 GBF                the glass chart boundary.)

VY 5 GBC

END

 

AANT P

M 2 1 A FNUM

GSR 0 5 3 P                (Here we use program-generated rays)

GNR 0 2 3 P .75

GNR 0 1 3 P 1

GSR 0 5 3 1                (Correcting colors 1 and 3 in this way will

GNR 0 2 3 1 .75        help to minimize spherochromatism.)

GNR 0 1 3 1 1

GSR 0 5 3 3

GNR 0 2 3 3 .75

GNR 0 1 3 3 1

END

 

SYNOPSYS 10                (10 passes, initial damping = 1, default modes.)

 

2. We wish to correct this Dall-Kirkham telescope at the on-axis image.  The obscuration should be .33 times the aperture diameter, and the image must be 2.0 inches behind the primary.  A paraxial focus solve (YMT) was used on surface 2.  The final focal length should be 8.0.  The input would be as shown below.

 

 

PANT P

VY 1 ASPH

VY 2 RAD

VY 1 TH

END

 

AANT P

M 0 1 A 2 YA 0 0 1(here we target only two rays, on axis, since
M 0 1 A 2 YA 0 0 .5one cannot control coma in a Dall-Kirkham.)

M 8 .01 A FOCL

M .33 1 A 2 YA 0 0 1 0 2        (ray height on surface 2

DIV YMP1        divided by semi-aperture.)

M 2 1 A BACK

A TH 1        (sum of TH 1 and 2)

END

OFF 1        (if default options are not wanted.)

 

SYNOPSYS 20        (20 passes)
