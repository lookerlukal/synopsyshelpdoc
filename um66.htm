6.6 Third-order Analysis (THIRD, NTHIRD)                Next Previous TOC

 

The commands

 

{ THIRD / NTHIRD } [ ELEMENT / PLOT SCF / CPLOT SCF ]  [CHROM [ ICOL1 ICOL2 ] ]

 

or

 

{ THIRD / NTHIRD } [ SN / SURFACE ] [ FULL ] [ CHROM [ ICOL1 ICOL2 ] ]

 

or

 

{ THIRD / NTHIRD }[ JSSS JSPS / P / SENS ] [ ELEMENT / FULL ][ CHROM [ ICOL1 ICOL2 ] ]

 

will produce a listing of the third-order transverse aberration coefficients.  The optional [ P ] or [ SURFACE ] give the surface-by-surface contributions, with the total aberration given for the last surface number, and PLOT SCF a plot of the cumulative aberration at each surface. CPLOT provides an alternate format, shown below. You may enter a scale factor of zero for autoscaling of either plot. The ELEMENT option prints the coefficients by element, and the FULL option will print the aspheric or GRIN contributions individually, if present.  If two color numbers are entered with the mnemonic CHROM, the chromatic difference in third-order aberrations will be printed (note that this is one of the few places where the letter “P” may not be substituted for a color number).

 

If SENS is requested, the program evaluates certain combinations of aberrations, some showing the sensitivity of the aberrations to tilts or decenters.  These data provide a way to reduce the tolerance sensitivity of the system.  See Section 10.13 for more information on this useful feature.

 

THIRD may be run interactively from the dialog MFT.  The output is in lens units, the same as for transverse aberrations.

 

If JSSS and JSPS are input, the aberrations are computed only for that range of surfaces.

 

The program calculates the 3rd-order aberrations with an algebraic method that works for all simple shapes and some of the more complex ones -- but is not applicable for all shapes. Surfaces defined as HOEs, Forbes polynomials, and the like.do not lend themselves to this calculation, so the program instead evaluates the aberrations at the final surface by tracing a set of rays and fitting the intercepts to a power series, the terms of which correspond closely to the Seidel aberrations.  This approach itself has limitations however: it does not return the aberrations of only a section of the lens, so the requested surfaces, if any, must then include the entire lens.  Also, it does not return the pupil aberrations, which are therefore unavailable with those shapes.

 

The coefficients are scaled by the F/number of the system at the last surface, unless the command NTHIRD was entered; in this case the aberrations are normalized to an F/number of 1.0 in AIR.  A discussion of the method used to calculate the coefficients is given in App. D.

 

The quantities output are defined as follows:

 

SA3

 

is the third-order spherical aberration

CO3

 

tangential coma

TI3

 

tangential astigmatism

SI3

 

satittal astigmatism

PETZ

 

Petzval curvature

DIS(FR)

 

distortion/GIHT (distortion / 1.0 with NTHIRD)

PAC

 

primary axial color

PLC

 

primary lateral color

SAC

 

secondary axial color

SLC

secondary lateral color

Third-order pupil aberration can be computed with the commands

 

{ THP / NTP } [ JSSS JSPS ] [ ELEMENT ] [ CHROM [ ICOL1 ICOL2 ] ]

 

or

 

{ THP / NTP } { JSSS JSPS / SN / SUR } [ FULL ] [ CHROM [ ICOL1 ICOL2 ] ]

 

This form requires that JSSS and JSPS, if entered, must be valid surfaces within the lens not including the image surface.

 

The third order pupil aberrations are defined as follows:

 

SAP

third-order spherical aberration of the pupil

 

COP

tangential coma of the pupil

 

TIP

tangential astigmatism of the pupil

 

SIP

sagittal astigmatism of the pupil

 

PZP

Petzval curvature of the pupil

 

DIP(FR)

pupil distortion / GIHT (distortion/1.0 with NTP)

 

 

If two color numbers are entered with the mnemonic CHROM, the chromatic difference in third-order aberrations will be printed.  The output from the THP command is shown below.
