3.5 Examples of RLE Data                                Previous Next TOC

 

The following examples are given to clarify the use of the RLE options.

 

Example 1. Simple paraboloidal mirror

 

 

RLE

First entry in file

ID PARABOLA EXAMPLE

identification line

WAVL 2 2 2

Assume that monochromatic operation at 2 microns is needed.

OBB 0 .5 8

infinite object, .5 degree semi-field angle, 16 inch aperture (8 in. radius)

1 RD -80 CC -1 YMT 0 AIR

focal length 40 inches, concave paraboloid

1 REFLECTOR

 

2

Image plane 40 inches from the mirror.  Default curvature is zero.

END

 

 

Example 2: Doublet lens

 

 

RLE

ID DOUBLET

WAVL CDF

Lens is used for visual wavelengths.  (This line could be left out since CDF is the default wavelength specification.)

OBA 10 1.0 3 0

Object 10 inches to left, object height 1.0 inches, entering beam radius 3 inches.

1 RD 27.673 TH 1.5 GLM 1.6205 60.32

First element similar to glass type SK16.  Note that secondary  color will be approximated, since the glass model is used.

2 RAD -5.4515 TH .2 AIR

airspaced doublet (AIR is default, could be left out here.)

 

3 RD -5.176 TH .375 GTB S

 

BASF12

Glass type BASF12 will be retrieved from the Schott catalog, and indices computed for the wavelengths given on the WAVL line.

4 UMC -.05 YMT

We want an F/10 exiting beam, with the next surface at the paraxial focus.  The target of the focus solve is zero by default (no value was entered here), as is AIR.

5 RD -20

The image will be examined on a curved surface.

END

 

 

Since we wanted an F/10 lens, the required value of the UMC solve is 1/(2*FNUM) = -.05.  The minus sign is needed because the ray angle is negative (see section 2.5.4).

 

Example 3: Symmetrical triplet

 

 

RLE

ID TRIPLET EXAMPLE

WAVL CDF

OBB 0 1 1

 

APS 3

The stop is on surface 3; paraxial pupil is good enough.

1 RD 6.25 TH .3 N13 61727 62041 62757

 

Glass type SK16, showing how to input indices exactly.

2 RD -25 TH .723 AIR

3 RD -14 TH .1 N1 1.66495 N2 1.66998 N3 1.68204

Glass type BASFI2, indices input exactly.

4 PCV -3 PTH 2

Surface 4 will have reverse of curvature of surface 3, thickness of surface 2.

5 PCV -2 PTH 1 PIN 1

last element picks up values from first element

6 PCV -1 YMT 0

 

BTH .002

We want the image defocused .002 inches to the right of the paraxial focus.

7

Default values will be used for surface number 7.  Since this is the highest-numbered surface, it is assumed to be the image plane.

3 RD -15

Suppose we changed our mind before the END line was input.  A new value will simply replace whatever was entered before.  Solves and pickups will use this value since they are not computed yet.

END

the last line of the RLE file.
