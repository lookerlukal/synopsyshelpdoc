PAD Link Help:

 

The command link option in PAD will execute a command that can update a graphics window in which you can monitor behavior that is not otherwise available in PAD.  Before you use this option, however, it is wise to do the following:

 

1.  Choose a command that executes quickly.  If you want to use the WorkSheet sliders, you want quick response.  Remember, this command will be executed every time the PAD window is updated.

 

2.  Your command must fit on a single line of no more than 100 characters.  You can do a multiline command if you put all of the data on a single line and use the slash (/) character to show where the lines would otherwise end.  For example, if the command sequence contains

 

FCO 100                                                                        

MMTF                                                                          

ICOL M                                                                        

HBAR 0 .5 .75 1                                                                                                                                        

PLOT  

 

you can compress it to a single line that reads

 

 FCO 100/MMTF/ICOL M/HBAR 0 .5 .75 1/PLOT.

 

You can also create a MACro with any number of lines, and then run it as the alternate MACro.  This will avoid overwriting your main MACro, which you are probably using for something else.  Then your single-line command is simply

 

 GADS

 

3.  Type this command line into a MACro window and test it.  If it makes a picture, adjust the size of the Graphics Window and the PAD window so they don’t overlap.

 

4.  Select QUIET mode, by clicking on the toolbar button Quiet.

 

5.  Select and copy the line from your MACro window.

 

6.  In the edit box to the right of the checkbox for the command link, paste the line you copied.

 

7.  Click the checkbox so that it is selected.

 

8.  Close the dialog.  The PAD display will update itself, and the command will be executed.

 

If you optimize the lens with SNAPSHOT selected, you can now monitor the behavior that you requested in the linked command.

 

Note that the link command string and checkbox status are not saved with the SSU or SPL commands.

 

Each of the five displays in each of the six configurations can have its own command string defined.
