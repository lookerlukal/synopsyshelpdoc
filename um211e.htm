2.11 Project name                                                        Previous TOC

 

The command

 

PROJECT name

 

Will create a time stamp in the data file PROJECT.LOG giving the starting time for this project name, and also prints the elapsed time since the previous PROJECT command was entered into the Command Window.  This file may be printed offline for accounting purposes.  For example, to start work on a new lens with the code name J35-001, you would enter the command

 

PROJECT J35-001.

 

The command PROJECT CLEAR will clear the contents of the log file.  (Spelled exactly, with five characters.)  Do not use that word for a project name.  (The file can grow indefinitely, if it is not cleared from time to time.)
