8.6.3.5 Through-focus diffraction MTF (TFMTF)        Next Previous TOC

 

The convolution MTF may be plotted at a given spatial frequency as a function of focus position at up to four field points and 10 wavelengths, from the dialog MTM.  Command-mode input is

 

TFMTF FREQ [FREQ FREQ FREQ ]

 

[ HBAR HBAR [ HBAR HBAR HBAR ... ] ]

[ HGEN NHBAR ]

[ GBAR GBAR ]

 

ICOL { ICOL / M / P }

 

INCREMENT DF

 

{ PLOT / GO }

 

Defaults are HBAR = (0., .5, .75, 1.0), and GBAR = 0, in the primary color.  “M” requests a multicolor analysis, and “P” just the primary color.

 

Up to 20 field points may be requested on the HBAR line; and HGEN, if entered, will generate a set of NHBAR field points evenly spaced from 0 to 1.  The latter requires the command-mode input.

 

The program can also plot the MTF at a single field point at up to four spatial frequencies.  If more than a single frequency is requested in the command line, then all will be analyzed, and only the first HBAR will be used.  The command-mode form is required in this case.

 

Since this program uses exact raytracing rather than the interpolation options available with some of the other MTF options, it should not be requested at a spatial frequency lower than about 1/40 of the diffraction cutoff.
