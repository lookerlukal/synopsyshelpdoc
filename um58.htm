5.8 Scale                                                Next Previous TOC

 

The dimensions of a lens may be scaled up or down with the input

 

 

If NB is entered, the lens dimensions are multiplied by NB.  If FOCL NB is entered, the lens is scaled so that the resulting paraxial focal length is NB, and the other paraxial properties work in a similar manner.

 

You can scale the lens so that a selected group from JSSS to JSPS assumes the entered focal length.  (It is wise to first do a FIRST JSSS JSPS analysis to check the sign of the focal length of the group.)  If PART is entered, only that group is altered; if not, the entire lens is scaled by the required amount.  The PART option does not scale the object coordinates, even if surface number zero is in the group.  It will scale global coordinates of the group, but keep in mind that these dimensions are always measured from surface one, not the first in the group. ZFILE zoom data are not altered by this option.  The TH distance before the group is not altered, while that following it is.  The SECTION option scales the surfaces in the range that includes JSSS through JSPS by NB.

 

M, MM, CM, and INCH will scale the lens as required to change to those units.  A PXT is automatically performed on the new lens.  This feature may also be run interactively with the MUT dialog.

 

Note that “SCALE” is also part of the AI vocabulary, and will be interpreted as a noun (and not a command) if the input line contains a verb.
