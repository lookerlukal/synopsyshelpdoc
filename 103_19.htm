10.3.19 Automatic Minimum Diameter/Thickness (ADM)                        Next Previous TOC

 

Whereas the ADT monitor can prevent lenses from becoming too thin to be practical, the ADM monitor does the opposite: it can prevent small lenses from becoming too thick.  Even if all other monitors are satisfied, a small lens may wind up with a large thickness, but still within the ACC monitor limit.  Such lenses are difficult to manufacture, rather like polishing the shape of the eraser on the end of a pencil.  The ADM monitor is then called for.

 

 The AANT file input

 

 ADM [ TARGET WEIGHT [ WINDOW ] ]

 

will monitor this ratio for all elements whose thickness is a variable, and compare the value to TARGET.  If the value is larger, there is no penalty.  The penalty goes as the square of the violation if the error is less than WINDOW, and linearly thereafter.

 

To ensure that the diameters are all no less than twice the thickness, for example, the input would be

 

 ADM 2.0 1 1

 

The program selects the larger of the current CAO values on either side of the element.  The target value is always a positive number.  This monitor does not apply to DOEs, HOE, or gratings.

 

This is one of several kinds of monitors that operate continuously during optimization.
