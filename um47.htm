4.7 Specifications, with Element Power (SPC)        Previous Next TOC

 

The command

 

SPC [ JSSS JSPS ]

 

gives a table with surface curvatures, rather than radii, and the element power and clear apertures as well.  For cemented elements, the power is given in the as-used configuration (immersed in the index of the element to which it is cemented), and for the element in AIR as well.  If surface numbers are entered, only that portion of the lens is listed.  An example SPC listing is given below.  This feature is also found in the MLL dialog.
