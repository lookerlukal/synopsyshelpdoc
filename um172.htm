17.2 Modeling Surface Coatings                                        Next Previous TOC

 

Each surface in a lens may be assigned a coating, except for dummy surfaces and glass-glass interfaces.  Eleven standard coatings are available, and you may define up to ten custom coatings.  These are assigned to individual surfaces with the input

 

SN COATING name

 

where name is one of

 

NONE

QMD

QMH

HEA

AL

ALSIO

AU

AGSIO

LOSSY

LO2

PERF

COSINE

EN1

EN2

5050

 

(see the next section for a definition of these coatings) and

 

SN COATING NB

 

where NB refers to a stored custom coating (see Section 17.2.2).

 

You may also assign any of the above coatings automatically to all non-dummy surfaces with a CHG file of the form

 

 CHG

 COAT name ALL

or

 COAT NB ALL

 END.

 

(If this is done in an RLE file, the program will not yet know how many surfaces will be entered or coated.  So use a CHG file instead.)  The system will be put in LINEAR Y polarization mode, if polarization is currently OFF, after this input is processed.
