15.1.3 Symbol Substitution                                                Previous Next TOC

 

AI has the ability to redefine character strings.  A sequence of one to three characters followed by a colon ":" may be equated to a string, such as

 

AA: Change G 3 on surface number 6 to

 

Then you may enter

 

AA .0012

 

and the value is changed.  Or

 

BB: The thickness of element 6

CC: 1.2345

Print BB

BB = CC

 

In the above example, the result is as if you had entered

 

Print the thickness of element 6

The thickness of element 6 = 1.2343

 

SYNOPSYS commands may similarly be redefined:

 

DD: RAY P 0 0 1 SURF

 

Entering DD will then give the surface-by-surface path of the ray.  The substitution is recursive: if the new sentence contains an equated string, it will again be substituted:

 

EE: Plot FF for 1 radius equal GG

FF: focal length

GG: 1 to HH

HH: 2.3

 

Typing EE then gives a plot of focal length versus 1 RD.  (Of course one could simply type “PLOT FOCL FOR 1 RAD = 1 TO 2.3”, but here we are showing a neat feature.)  If FF, GG or HH are later redefined, the command EE uses the new definitions.  Note that if switch 45 is on, all characters of nearly all input lines are checked for symbols; if off, only the first three characters of AI sentences or command-level SYNOPSYS commands are checked.  Thus in the input file

 

OFF 45

CHG

3 RAD 2.345 TH 6.789

END

 

only the “CHG” portion could use symbol substitution, because the rest of the input is not at command level.

 

Note that in the previous example, the symbols "HH" and “EE” were redefined by Al.  If you later wanted to invoke the MACro editor by typing EE, the command would not work, and the ray angle tangent HH would no longer be calculated by a reference to HH.  To remedy this, you may either exit from AI with the command INT, or delete the symbols with

 

Delete symbol EE.

Delete symbol HH.

 

A list of current symbols is produced by Show symbols, and any symbol that is redefined takes the latest string equated to the name.  Up to 100 symbols may be defined, and may be included in your CUSTOM.MAC.

 

Any printing character except the "$” and the “!" may be defined as a symbol:

 

@: What is the global Z coordinate of surface 6?

 

and multiple lines may be implied by the "/" or ";" characters.

 

#: GET 5/EM STARS/print the back focus/SA3?

 

Typing the character "#" will then

 

1.

Get lens number 5 from the library,

 

2.

Execute the MACro that has been saved with the name "STARS",

 

3.

Print the back focus,

 

4.

Evaluate and print the third-order spherical aberration.

 

 

When symbols are included in an AI sentence or SYNOPSYS command line, they must not run into another word.  This prevents unintended substitution in cases like

 

SS: SHOW SYMBOLS

CHG

1 GID

GLASS

END

 

Here, the "SS" in GLASS is safe from substitution because it is part of another word (it runs into the letters “GLA”).  The exception to this rule is when a minus sign appears before a symbol:

 

XYZ: 22.3

CHANGE 4 TH TO -XYZ

 

You can temporarily defeat symbol substitution with the command SYDEFEAT and enable it again with SYACTIVATE.  Those may be handy if your MACro has mnemonics that collide with some of your symbols.

 

The program automatically recognizes a number of symbols, as explained in Section 10.3.  Thus, the characters FP1 will be replaced by the surface number that is one higher than the surface currently assigned the FLAG attribute.
