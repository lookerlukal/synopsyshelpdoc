2.9  Modes of Operation (ON, OFF)                                Previous Next TOC

 

Some of the optional features of SYNOPSYS are either called or bypassed depending on the status of a set of 100 mode-control switches.  These may be reset from their default configuration by means of a command of the form

 

 ON NB NB NB NB

or        OFF NB NB NB NB...

 

where NB is the number of a switch that is to be turned on or off.  Up to 29 switch numbers may be input at a time with ON and OFF commands.  This input may be entered at nearly any time including within data files.  The dialog MSS shows the current switch settings along with an explanation of the effect.

 

A list is given in Appendix B summarizing the effects (such as extra output) caused by placing each switch on.  When the program starts, the switch configuration is as follows:

 

 2 4 5 7 8 27 29 32 39 45 54 65 82 95 97 are set ON, all others are OFF.

 

When the MACro SYSTART.MAC is run, an additional set of switches are turned on:

 

 10 28 35 41 42 62 64

 

If you alter a switch setting, the new settings are in effect throughout the run, or until changed again.

 

To make a listing of the current settings of the mode control switches, enter the command

 

 SSW        (show switches),

 

or select the dialog MSS to see and manipulate the current switch settings. Section 10.5 discusses how these switches affect the optimization program.

 

If you want a different switch setting to go into effect every time you start SYNOPSYS, put those settings into a MACro named CUSTOM.MAC in the directory in which you want to work.  That MACro is run automatically, if it exists, whenever the programs starts.
