Options menu commands

The Options menu lets you select from the following:

Graphics | Reuse Graphics Window

Causes new plotted output to be displayed in the same window as the previous.

Graphics | Make New Window for Each Picture

Each picture is displayed in a new window.

Graphics | White Background

New pictures are shown on a white background..

Graphics | Black Background

New pictures are shown in color on a black background.  You can alter the shade with the menu item Options | Graphics | Define Background Color.

Graphics | Set Pen Width

Displays a dialog where you can select the width of the pen.

Graphics | Set Device Scale and Offset

Displays a dialog where you can select a scale factor and offset to compensate for printer differences for hardcopy plots, alter the font size for printouts, and tweak the PAD position.

Graphics | Kill All Graphics Windows

Closes all graphics windows currently open.

Command Window | Prompt When Full

Prompts for user action when the Command Window buffer is full.

Command Window | Recycle Text Window

Automatically clears top ¼ of text buffer when full.

Set ToolbarAction

Displays dialog controlling the toolbar at the left of the main window.

Set Printer Offset, Margins, Font

Displays a dialog where you can select a scale factor and offset to compensate for printer differences for hardcopy plots, and alter the font size for printouts.
