RPER – Rotating Perspective Drawing

 

When an RPER drawing has been created with the RPER command, it is displayed in a special graphics window where you can use either the mouse or the keyboard to rotate and pan the display.

 

To rotate and pan the RPER display with the mouse, click and drag the mouse in the RPER picture itself.  Dragging the left mouse button up and down rotates the picture about a horizontal axis; dragging right and left rotates about a vertical axis.  Dragging in any other direction rotates in both axes at the same time, by an amount that depends on the azimuth of motion.  Dragging the right button pans the display.  To enlarge the image, use <Shift>+Right-click.  To make it smaller, use Ctrl>+<Shift>+Right-click.

 

To rotate and pan with the keyboard, first you must click in the RPER display to give that window the keyboard input.  Then use the keyboard as follows:

 

<arrow keys>

To rotate in alpha and beta

<Left Ctrl>+<arrow keys>

To rotate at 1/10 speed in alpha and beta

<Left Shift>+<left, right arrow keys>

To rotate in gamma

<Right Shift>+<left, right arrow keys>

To rotate in gamma at 1/10 speed

 

 

<Right-Ctrl>+<arrow keys>

to pan

 

To use the arrow keys in the Command Window after you have done the above (to roll back to a previous command), first click in that window to give it the keyboard focus once again.

 

The RPER function (as well as the PER and SOLID commands) may be conveniently accessed in the menu MPE.

 

 

RPER Toolbar

 

 

Most of the above toolbar buttons serve the same function as in the normal graphics windows.  Three other buttons have been added for stereo views of the lens drawing.  The button shows two views, side by side, with a 5-degree rotation angle between them. These make a stereogram pair that can be viewed in 3-D by anyone with a stereo viewer or with the skill to fuse the two pictures (eyes diverged to infinity, each eye seeing its own view).  The button changes the display to a red-blue anaglyph picture that can be viewed in 3-D with the help of colored glasses.  Hardcopy plots may be made in both of these modes.  The red-blue anaglyph effect is not as pronounced on paper as on the screen, however, because the colored glasses reduce the contrast.  Click the button again to turn these features off.

 

The default separation between the stereogram views is 2.0 inches, but you can change this value by clicking on the button , which cycles from 2 to 4 inches, or in command mode with the command

 

SSEP separation,

 

where the desired separation is given in inches.

 

The DXF button saves the picture at the current view angle as a .DXF file, which may be imported into many kinds of CAD programs.  The IGES button converts the picture into an IGES file with view angles (0,0).

 

A stereo display may also be shown in PAD, with the added convenience of showing the 3-D display continuously as you move sliders in the WorkSheet.
