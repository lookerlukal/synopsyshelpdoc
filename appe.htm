SYNOPSYS User’s Manual

Appendix E

 

EXAMPLE SYNOPSYS FILES                                        TOC

 

This Appendix contains example files that you can copy and then run on SYNOPSYS.  Most of the analysis MACros require a suitable lens file in order to work properly.  The first section contains several lenses that you can copy and then use when you run the other examples.  These files will save themselves on disk with names “1.RLE”, “2.RLE”, etc., and the later examples will FETCH them.  If they are not there, those examples will not work – but you can always edit the examples so that they fetch or get a different lens.

 

To run any of these files:

 

1.  Select the text lines you want by dragging the mouse over them and then pressing Ctrl+C.  This puts a copy of those lines onto the Windows clipboard.

 

2.  In SYNOPSYS, create a new, empty MACro window by clicking on button .  (Or you can erase one of your current MACro windows.)

 

3.  Click in the window to select it, if it is not already selected, and then press Ctrl+V.  This copies the clipboard contents into the MACro window.

 

4.  Click on the Run Macro button .
