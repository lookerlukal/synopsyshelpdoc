9.1.1 Composite image model                                Next Previous TOC

 

Several features of SYNOPSYS can create an image model that is more complex than the single-field monochromatic or multicolor image described above.  The simplest way to model an extended source and analyze the effects of aberrations is with the Image Tools (command MIT).  This will show the appearance of the imaged source and can analyze the image in several ways, such as with a slit or knife-edge trace.  The Extended Image Analysis feature (EIA) will consider an array of object points and calculate a unique ray pattern for each one, and provide a geometric analysis.

 

Another way to combine images is with the ADD AXIS option in GMODEL.  This is discussed in the next section.
