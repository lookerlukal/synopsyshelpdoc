HIDD_MPS

MPS PSPRD over Spectrum Menu

 
The diffraction point-spread can be plotted in several ways, all of them accessible from the menu MDI.  The MPS menu shows the PSPRD in a variety of forms at several wavelengths covering the range defined in the lens.  This dialog submits the OSPSPRD command with all of the necessary arguments.  It shows the point-spread function at several wavelengths at a single field point.

 

The arguments available on the dialog are as follows:

 

Number of rays

Enter a number from 600 to 20000.  A large number will avoid aliasing but will run more slowly.

 

Height

This entry controls the height of the individual plots when drawn in oblique-perspective form.  If you enter a number other than 0, all of the plots will be scaled to that height.  If you enter 0, the program will scale each picture so that the relative intensity of each image controls the height.  A perfect image in this case has a height of about one inch.

 

Straight/Right/Left

This selection controls the view angle of the surface plots.

 

Show as surface/visual

These radio buttons control how each picture is displayed.  The surface selection draws a pattern of lines, while the visual selection shows what the image would look like to the eye.  Infrared images are shown in red and UV images in violet -- since the eye cannot see them in their real color.

 

Magnify, Gain

These data allow you to zoom in on the visual form of the display.  Magnify controls the zoom, and Gain adjusts the intensity of the picture.  If the images are not all nearly perfect, you should enter a Gain larger than 1.0 or the image may be too faint to see clearly.  This program uses the wide-area version of the PSPRD calculation in order to show images that are not perfectly corrected and extend over more than the usual sample area, which is about 11 times the size of the Airy disk.

 

WAVELENGTHS

Here you specify how many wavelengths to display; the number must be from 2 to 20.  The first plot is always the image in the color designated the long wavelength in the CORDER data, and the last is at the short wavelength, with intermediate images showing intermediate points.  This analysis is performed at a single field point, entered with the input.

 

FIELD

The analysis will be performed at the field point given.

The program will not run if any of the materials are defined in a way that the index cannot be recalculated at different wavelengths.  All paraxial curvature or thickness solves will be temporarily deleted before the analysis, to insure that the lens specifications do not change with wavelength.
