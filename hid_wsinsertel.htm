WorkSheet: Insert Element

 

The simplest way to insert a lens element is with the Worksheet dialog.  First, open the SketchPAD display with the command PAD or by clicking .  Make a checkpoint.  Then open the Worksheet dialog by clicking .  Now click on toolbar button .  The cursor changes to look like the button.  Now click on the optical axis where you want the element inserted.  The program inserts a dummy element with flat surfaces, which you may edit directly in the WorkSheet display.  If you change your mind, click on the Undo button , which will restore the latest checkpoint.

 

You can also insert an element with the command INSERT or using an AI sentence.

 

Note that if you insert an element in object space, the object distance will change.  Some of the pupil options specify a fixed marginal ray height on surface 1 (YMP1) – and this value will then apply to the new surface 1, which is closer to the object than before.  This will change the NA in object space, which you may not intend.  The best way to avoid this problem is to fix the NA, which can be done easily with the Object Wizard (MOW) or Pupil Wizard (MPW), before inserting the element.
